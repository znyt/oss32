　　金刚三藏望着香案，头也不抬说道：“三位，有什么话要说吗？”
<br/>　　林三生道：“小生无话，是陪他们而来。”
<br/>　　四宝跟金刚三藏闲聊两句，金刚三藏突然问起他师父怎么样了，两下一说，才知道他问的是鸠摩罗什，心中不禁佩服他火眼金睛，能一眼看穿自己前世今生。
<br/>　　“有关佛法，令师想必有所传授，我这便不多舌了，你大巧似拙，将来自有造化，是福源广深之人。”
<br/>　　这种话在人间经常被听到，但金刚三藏是什么身份，绝不会说什么恭维话，这么说必有缘故，四宝听了就很高兴，道了谢，不再说什么了。
<br/>　　金刚三藏看着叶少阳，道：“乃有祖先之风。”
<br/>　　叶少阳道：“大师，我祖先叶法善是个什么样的人？”
<br/>　　“我与你祖先同为国师，当时三道一法，他三人喜欢捉弄于我，不过也都是莫逆之交，一晃数千年不见……今日见你，方感慨逝者如斯。”
<br/>　　金刚三藏感叹起来。
<br/>　　“大师，我祖先当初去太阴山，这件事……您是否知道一些底细？”
<br/>　　“底细谈不上，那时我还未证道，得知他要去太阴山，也极力阻拦，但叶真人壮怀激烈，一去不回，自此再没见过。”
<br/>　　接着又一边回忆，一边给叶少阳讲述了些有关叶法善的往事，末了感慨道：“大唐初年，是法术界的盛世，道门有叶法善、张果、罗公远三杰，佛门有玄奘、鉴真二位禅师，远播佛法，当时之盛况，是你们今日所无法想象……不过俱是过往了。”
<br/>　　感慨了一番，金刚三藏望着叶少阳说道：“小叶天师，汝祖叶真人前往太阴山之前，曾我与言，若此行失败，必将精血送于轮回，末法时代，叶家必出道神，天才纵横，甚而青出于蓝，想必便是你了，而今天劫降临，你便是这人间砥柱中流，任重而道远啊。”
<br/>　　叶少阳深吸一口气，道：“大师你过奖了，我从没想过当什么英雄，做什么都是率性而为，我连个茅山都不想管，更不要说整个法术界，谁想当老大就让他去当，你之前指点慕寒，让他不争，这一点我倒是可以做到，哈哈。”
<br/>　　金刚三藏也跟着笑了笑，道：“你能不争，是你的福气，但人各有缘，你的机缘未必在不争二字上。”
<br/>　　说完拿出一片镜子出来，道：“你们都照一照吧。”
<br/>　　四宝第一个接过去，见镜框中间镶嵌着一块好像白玉一样的东西，对着自己照了照，照不见任何东西。
<br/>　　“哈一口气。”金刚三藏提醒道。
<br/>　　四宝照做，哈了一口气在镜片上，白玉似乎融化了，形成了一团气体，接着汇聚成了两个字……
<br/>　　四宝呆呆地看了一会，抬头看着金刚三藏，刚要开口，金刚三藏道：“这面镜子只能渡你本心，我却无法解说。”
<br/>　　四宝只好点头道谢，将镜子递给林三生，照过之后又递给叶少阳，三个人都照了，互相看了看，金刚三藏起身道：“诸位，我们走吧，我也要去海眼，看看那边如何了。”
<br/>　　大家一起出门，路上，叶少阳问金刚三藏为什么会一直镇守在这里，金刚三藏笑而不语，四宝想起一件事，好奇问道：“大师，你曾经是大唐国师，听说杨贵妃也拜你为师，那杨贵妃……是不是真的很漂亮？”
<br/>　　叶少阳笑道：“肯定比范凉凉扮演的漂亮。”
<br/>　　四宝一时没反应过来，“怎么就凉了？”
<br/>　　“你不看新闻的吗？凉透了。”
<br/>　　金刚三藏道：“在我眼中，这个范……不对，这个杨贵妃，跟你差不多。”
<br/>　　“跟我？”
<br/>　　“我是僧人，僧人眼中，一切皮囊俱无差别，哪有什么美丑之分。”
<br/>　　四宝摸着脑门，对这个回答显然不满意。
<br/>　　一行人走向高台，远远地就看到高台上站满了人，仔细一看惊呆了，人群里居然有崔府君！
<br/>　　崔府君坐在一个方凳上，面前摆着一张桌子，正在翻看什么，边上橙子正在伺候他翻书，两个功曹站在豫州鼎前，不断将一个又一个人影拉出来，送到崔府君的桌案前接受质询，然后被投入海眼的水井中。
<br/>　　这时候正好有一群人从高台上走下来，叶少阳看了一眼，其中有徐文长，跟在一个高大威猛的官员身边，这官员一身蟒袍，气度不凡，叶少阳看了一会，吃惊地认出，居然是转轮王！
<br/>　　一行人伺候着转轮王走到高台下的空地上，将桌椅板凳摆好，俨然一副要在这里办公的意思。
<br/>　　徐文长就站在转轮王身边，看到叶少阳一行人，对他们招了招手，叶少阳走过去。
<br/>　　转轮王赞许地看了叶少阳一眼，“小天师，多亏了你。”
<br/>　　叶少阳道：“你们这是做什么？”
<br/>　　“办公啊，”徐文长解释，“不是说了，要将这海眼当做轮回井来用，接引人间生魂。”
<br/>　　他抓住一个功曹，问道：“阎罗王来了没？”
<br/>　　那功曹解释阎罗王正在收拾公务用具，立刻就来。
<br/>　　“小天师，这边积压了好几天的公务，亟待处理，我这边先忙了，徐师爷你陪小天师说话。”
<br/>　　徐文长把叶少阳一干人拉到边上去，边走边道：“三司府衙都要搬过来，在这边临时办公，保证三界轮回正常运转，小天师，诸位，大家功不可没，我代表阴司拜谢诸位了。”说完躬身行礼。
<br/>　　叶少阳也就受了，低头叹道：“可惜秋莹死了。”
<br/>　　这时候捉鬼联盟的人陆续过来了，在叶少阳身后默默站成一排。
<br/>　　徐文长安慰道：“赤月罗刹和木落真人也死了，秋莹也算死得其所。”
<br/>　　叶少阳道：“他们死一百个，也抵不上我的秋莹。”
<br/>　　徐文长有点尴尬地点点头，道：“秋莹死得其所，为阴司立下大功，我已报知圣帝，等积压公务处理完，我亲自找府君，让他在生死簿上为秋莹加上这一功，为她来生做些机缘。”
<br/>　　听到这个，大伙心中多少有些安慰。
<br/>　　叶少阳抬头看着高台上忙忙碌碌的人们，道：“海眼可以代替轮回井是吧，那以后生魂就从这里出来了？”
<br/>(本章完)
