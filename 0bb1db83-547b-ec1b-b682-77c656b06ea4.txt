　　“这件事情不许对任何人说，包括你的那个死党陈勾！不然老师不会放过你！！”
<br/>　　苏云揪了下林峰耳朵威胁道。
<br/>　　“知道了老师，这种事情我怎么可能和别人说，狗子也没资格知道老师穿的是紫色内内。”
<br/>　　林峰立即认真的说道。
<br/>　　“你来这里是干什么？不会真的有什么偷窥癖好吧？”
<br/>　　苏云不想再谈论紫色内内这个事情了，立即岔开话题，狐疑的看着林峰。
<br/>　　而林峰的目光就落在了之前苏云用过的那个卫生间，此时此刻有个空投在那里。
<br/>　　很显然正如系统所说，除了林峰，其余谁都看不见。
<br/>　　“没什么，我这不是巡逻嘛，我担心有丧尸躲在厕所里。”
<br/>　　林峰道。
<br/>　　“原来这样啊。”苏云顿时相信了。
<br/>　　“你先回去吧老师。我再在这里逛一逛。”
<br/>　　林峰支走苏云，不然让对方看到自己在厕所里捡东西，岂不是会被真的当成变态了。
<br/>　　“那你自己小心。”
<br/>　　苏云点了点头离开了厕所。
<br/>　　林峰松了口气，“好在自己激灵，先看看这一次有什么吧。”
<br/>　　林峰立即打开空投。
<br/>　　咔擦。
<br/>　　箱子打开了顿时林峰眼睛冒光。
<br/>　　“这是……飞镖？”
<br/>　　林峰落在了一堆飞镖上。足足有二十多个。
<br/>　　“好东西。”
<br/>　　林峰把这些玩意给丢入物品栏之中，这个飞镖虽然数量有二十多个，但是只占据一个背包格子。
<br/>　　除了飞镖之外，还有一本书。
<br/>　　《绝世飞镖高手传承》暗器大陆出版社荣誉出品。
<br/>　　暗器大陆出版社？这尼玛什么鬼出版社。
<br/>　　林峰无语了，打开这东西一看，里面居然是白的，什么都没有。
<br/>　　“这尼玛地下盗版印刷厂吧。”
<br/>　　叮！
<br/>　　恭喜宿主领悟绝世飞镖！
<br/>　　林峰愣了，旋即狂喜，原来是这么个情况。
<br/>　　再看手中的那本《绝世飞镖高手传承》已经消失了。
<br/>　　除了飞镖和飞镖高手传承之外，还有两枚丹药。
<br/>　　“这是什么玩意。”
<br/>　　林峰拿起来闻了闻。
<br/>　　叮！
<br/>　　爆气丹：可以增加力量速度，持续时间一分钟！
<br/>　　叮！
<br/>　　金刚不坏丹：可以增加防御力，持续时间一分钟！
<br/>　　“好东西！具体可以增加多少？”
<br/>　　林峰好奇道。
<br/>　　系统：换算成如今地球的实力，可以增加一个阶段。
<br/>　　“也就是说，我现在若是用了，那就是二阶的黑铁战士实力了。”
<br/>　　林峰收起这两个丹药，关键时刻再用。
<br/>　　林峰看箱子里再也没有东西了，直接就回去了。
<br/>　　这一次，他要再去一次第六层。
<br/>　　依然只带陈勾和孙畅。
<br/>　　毕竟整个班级内就他们战斗力最强，不惧怕丧尸。
<br/>　　而苏云老师虽然也是一阶了，但是对方穿着打扮不适合剧烈运动，虽然苏云老师似乎还是个跆拳道黑带的高手。
<br/>　　“林峰，这一次你有什么计划安排？”
<br/>　　孙畅道。
<br/>　　“没有计划安排，正面刚，你们盾牌准备好了吧？”
<br/>　　林峰道。
<br/>　　“嗯。”
<br/>　　陈勾和孙畅今天多了一个武器，木头盾牌，这东西是他们从课桌上卸下来的，长方形的盾牌，后面做个手可以握着的把，毕竟在学校内，剪刀和强力胶水还是可以找到的。
<br/>　　“我打头阵，你们掩护侧翼，没问题吧？”
<br/>　　林峰刷的一下拿出了武士刀。
<br/>　　“握草，武士刀？哪里来的？淘宝上买的？不对啊，现在淘宝也不送货了吧？”
<br/>　　陈勾意外至极。
<br/>　　孙畅也是被吓了一跳。
<br/>　　“在办公室里找到的，估计是哪个老师收来的吧。”
<br/>　　林峰撒了个谎。
<br/>　　“原来如此。我去，估计是哪个高三的混子学生带到学校来装逼的。”陈勾等人顿时相信了。高三那群不读书的，什么事情都干得出来。
<br/>　　刷。
<br/>　　林峰拔出武士刀，旋即深吸口气，三人就是冲了下去。
<br/>　　林峰拿着武士刀敲了敲墙壁顿时发出了清脆的声音。
<br/>　　“大傻逼，看这里！”
<br/>　　吼吼。
<br/>　　楼道内的丧尸立即被他吸引了，足足七八个。
<br/>　　不过林峰三人已经见过血了，除了那脸上的烂肉有些恶心之外，没有多大恐惧。
<br/>　　“把盾牌立起来，挡住丧尸后，我再刺！”
<br/>　　林峰立即道。
<br/>　　他们选择以静制动，背靠死角。
<br/>　　一来是这个楼道上有教室，教室内肯定有丧尸，若是前进的话搞不好就会被忽然出现的丧尸给扑倒在地，被丧尸扑到了，基本上就是死路一条了，根本没有机会站起来，所以这个方法是最为稳妥的。
<br/>　　七八个丧尸全都冲了过来。
<br/>　　陈勾和孙畅大喝一声纷纷举着盾牌顶了过去。
<br/>　　砰！
<br/>　　两人被丧尸巨大的力道给撞退了三四步不过依然抗住了。
<br/>　　那丧尸在那里张牙舞爪的就是想要过来。
<br/>　　林峰抓紧机会立即就是一刀切过。
<br/>　　刷刷刷。
<br/>　　顿时三个丧尸的脑袋一半被割掉了，脑浆狂喷，倒在了地上抽搐。
<br/>　　刷。
<br/>　　又是一刀，剩下的四个丧尸全部被削掉了头颅。
<br/>　　“牛逼啊这把刀，太锋利了吧。”
<br/>　　陈勾等人震惊道。
<br/>　　“别废话了，又有丧尸来了！这一次最起码二十个！”
<br/>　　孙畅提醒道。
<br/>　　“还是刚才的办法！”
<br/>　　林峰道。
<br/>　　二十个丧尸全部张牙舞爪，陈勾和孙畅手心里都是汗水。
<br/>　　砰砰砰
<br/>　　一个个丧尸巨大的力道就好像是个小车子一样，好在陈勾和孙畅都是一阶的黑铁战士，若是之前的话，估计就被撞飞了。

<br/>　　林峰眼疾手快立即连忙斩杀那些丧尸帮助他们减轻压力。
<br/>　　顿时七八只丧尸倒在了刀下。
<br/>　　“下面有一个丧尸爬过来了！”
<br/>　　孙畅立即大声道。
<br/>　　林峰低头一看就看到个丧尸像个壁虎一样爬了过来。
<br/>　　“还有地上爬的？”
<br/>　　林峰立即一刀刺了下去，把那丧尸钉在地上，然后一脚踩爆了他的脑袋。
<br/>　　刷刷刷。
<br/>　　几道刀芒砍杀过去，二十多只丧尸全部死翘翘。
<br/>　　而陈勾和孙畅都已经累得不行了。
<br/>　　“太累了妈的，这群丧尸力气太大了。”
<br/>　　“可不是，刚才我感觉自己都要差点顶不住了，足足二十个啊！”
<br/>　　不过他们此时更多的还是兴奋，看着倒在他们身前不远处的一堆尸体，他们何时能够想到自己可以有机会杀这么多。
