        “是兄弟，但毕竟同父异母，他是嫡长子，我只不过是个庶出！其实，我连庶出的都不算。”傅昼景苦笑。
<br/>
<br/>        童筱颖抿了抿唇，拉住傅昼景的手，安慰道：“昼景，这事会不会是个误会？你们毕竟是亲兄弟，就算是同父异母，那也是手足情深，血浓于水，大哥不至于要这么做！”
<br/>
<br/>        “季思妍是季氏集团董事长的女儿，她还有个哥哥叫季运恒，是季氏集团未来的继承人。筱颖，你想想，季思妍家世这么好，而且我已经有助理赵斌了，大哥却仍旧把季思妍派到我身边来，是何用意？”傅昼景条条是道地说道。
<br/>
<br/>        童筱颖却听得目瞪口呆。
<br/>
<br/>        她真的万万没想到，季思妍居然是那个喜欢在“夜惑”泡妞，人称“季少”的季运恒的妹妹！
<br/>
<br/>        提到“季运恒”时，傅昼景一直在盯着童筱颖，就想看看童筱颖是什么反应。
<br/>
<br/>        那日他在“夜惑”里和同事一起开庆功宴，在清吧的雅座里亲耳听到坐在隔壁雅座里的季运恒高谈阔论，说“小谢”的第一次给了他。
<br/>
<br/>        不仅如此，“小谢”还和什么“连少”、“十爷”也上过床。
<br/>
<br/>        小谢是谁？
<br/>
<br/>        那日，他就坐在雅座里亲眼看到了小谢，也亲耳听了小谢的歌声。
<br/>
<br/>        小谢不是别人，就是他的筱颖，哪怕她戴着面具，他也认得出来。
<br/>
<br/>        童筱颖的眼睛里闪一丝异样，让傅昼景笃定了自己的想法。
<br/>
<br/>        筱颖果然认识季运恒，筱颖果然是小谢。
<br/>
<br/>        他真的没想到，一个巨额负债，可以毁掉一个家庭。
<br/>
<br/>        就因为他负债，筱颖失去了贞操，而他也成了渣男中的一员。
<br/>
<br/>        明明，他不想变成这个样子的……
<br/>
<br/>        可为什么现实与自己所设想的未来越走越远？
<br/>
<br/>        当时，在他负债的那天，他就应该果断地选择和筱颖离婚的。
<br/>
<br/>        至少，现在筱颖不会为了他去“夜惑”里卖。
<br/>
<br/>        至少，现在筱颖还是清白之身。
<br/>
<br/>        至少，现在他不会在意外得知筱颖失身后而让自己在感情上堕落牵手季思妍。
<br/>
<br/>        至少，现在他孑然一身，心胸坦荡。
<br/>
<br/>        至少……
<br/>
<br/>        傅昼景心里逐渐迷茫起来，为什么当初会心软？
<br/>
<br/>        心软到，现在，他一点都不想对筱颖放手了。
<br/>
<br/>        “我不懂，也不明白。”童筱颖缓缓垂下眼帘，微微摇了摇头。
<br/>
<br/>        傅昼景回过神来，只好耐着性子和童筱颖解释：“傅氏集团，是爷爷的。我爸爸当年去参军了，一生精忠报国，一辈子给了部队，从未继承过傅氏集团的一丝一毫。后来，大哥也选择去当兵，但是，爷爷从未想过要把傅氏集团传到我手中。而是一直想等我大哥提前退役，接手傅氏集团！但是，在大哥之前，爷爷退休后，傅氏集团一直是我堂叔冷岩峰在打理，爷爷为了打压我堂叔，才让我进傅氏集团工作。现在，爷爷又用大哥来打压我和堂叔。堂叔现在已经知难而退，溃不成军。堂叔告诉我，大哥在他身边安插了眼线，他的一举一动，大哥了如指掌。而现在的我，也一模一样，被大哥的眼线监视了！”
<br/>
<br/>        童筱颖听得似懂非懂，但她可以肯定一件事，那就是，大哥傅夜沉似乎真的知道昼景的动静。
<br/>
<br/>        因为，她记得那次大哥要她留在总裁办里做饭的时候，她找借口说要回家做饭，但大哥傅夜沉却告诉她昼景有应酬不会回家。
<br/>
<br/>        当时，她还以为昼景的应酬是大哥间接造成的，毕竟大哥是昼景的上司。
<br/>
<br/>        所以，大哥知道昼景有应酬理所当然。
<br/>
<br/>        但是，现在再仔细回想，她发现或许真如昼景所说的一样。
<br/>
<br/>        为了证实自己的猜测，童筱颖看着傅昼景，忽然一本正经地问道：“昼景，就是这次你出差的那天我给你送U盘，后来我又去了大哥的总裁办，那天中午，你说你有应酬不能回来吃午饭。那么，你应酬的事，有告诉过大哥吗？”
<br/>
<br/>        “你送U盘的那天！”傅昼景身心一怔，他岂会不记得那天的事，因为那天，他和季思妍的父亲季董事长签了合同，中午才请的季董事长吃饭。
<br/>
<br/>        这件事情，是他的私事，大哥当然不知道。
<br/>
<br/>        “不记得了吗？”童筱颖见傅昼景不吭声，讷讷地反问。
<br/>
<br/>        傅昼景突然拉住童筱颖的手，忐忑不安地质问：“那天大哥和你说了什么？”
<br/>
<br/>        “大哥说你有应酬，不会回家吃饭，所以让我留在总裁办里给他做饭吃。”童筱颖如实回答。
<br/>
<br/>        傅昼景向来心思缜密又敏感，童筱颖这么一说，他瞬间便明白是怎么一回事了。
