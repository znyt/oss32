　　第706章
<br/>　　“陛下。”
<br/>　　大殿之前不知何时站满了人，人人披甲持矛，腰间佩刀。这些人便是属于李氏皇族的鱼龙卫，当年由明帝一手调教出来，对李氏忠心耿耿。
<br/>　　林少羽破长安时，他们损失了一批，后来又有人填充进来，免不得就有人往其中安插亲信。
<br/>　　目前林少羽面前这一千余人，就是他手中握着的最后力量。
<br/>　　殊死一搏，全靠他们。
<br/>　　这些人中，以田七为首，他一身黑甲，手捧过来一杆长枪，道：“陛下，让属下出去杀贼，陛下还须保重龙体。”
<br/>　　“废什么话。”李乐天气不打一处来，道：“将他的脑袋割下来，去相府。”
<br/>　　田七隐隐觉得，今天的天子和往常不太相同。或者，往常那个温文如玉，令人如沐春风的天子不过是他的表现，李家男儿骨子里都有一股亡命徒的劲儿。
<br/>　　李乐天走在队伍最前方，佩剑持刀，一身黑龙袍。今天他是一军之帅，也同样是先锋，身为帝国天子，他本不须承担如此风险。可当他走在队伍最前方，一股莫名的力量在身后鱼龙卫体内点燃。
<br/>　　他们愿意为李乐天去死，何况李乐天已将生死置之度外。
<br/>　　未出宫门，他们已经被拦住。面前也同样是鱼龙卫，数量要超出李乐天这面一倍左右。
<br/>　　为首是一名校尉，他也是相府安插进来的，七拐八拐算起来，是崔云间的子侄辈。
<br/>　　鱼龙卫多的是这样的人，一人得道鸡犬升天，而那些升天的鸡犬，也会反哺崔府。
<br/>　　目前，他根本没明白是怎么会事，此刻拦在李乐天面前，道：“陛下，为何要出宫？”
<br/>　　李乐天向前跨出一步，道：“朕为何出宫，要与你商量么？”
<br/>　　“这个……不敢，属下也是担忧陛下安全。”
<br/>　　李乐天冷哼一声，道：“崔正密谋造反，斩立决，凡持戈相击者，视为同谋，杀无赦。”
<br/>　　崔正下意识退后一步，第一个想法竟然是：天子知道我的名字。他执掌鱼龙卫，经常在李乐天面前晃，但却没有说过一句话。在他想来，天子根本不知道自己这号小鱼小虾。
<br/>　　原来他是知道的。
<br/>　　李乐天为先，已经掩着手下死士冲杀过来。长枪所至，一枪穿了崔正的脑袋。
<br/>　　于此同时，李乐天陷入包围中，他持枪而立，口中冷哼一声道：“大胆，你们也想造反么。”
<br/>　　这群人面面相觑，渐渐有人放下兵器，匍匐在地。
<br/>　　李乐天长枪一甩：“走。”
<br/>　　打的便是时间差，以迅雷不及掩耳之势，将一柄尖刀捅进相府的心窝。当然，这件事胜算不高，不过，李乐天既然敢踏出这一步，已经将生死置之度外。
<br/>　　当李乐天踏出宫门时，城中已经有兵丁集结。这是百里胜从凉州边境带来的八百死士。
<br/>　　虽然只有八百，但都是从生死战场中厮杀出来的。互相之间早已生同生，死同死，八百人抱团在一起，便爆发出可怕的力量。
<br/>　　他们集结完毕，便与城中巡逻的城防军发生战斗。百里胜手持双鞭，口中爆喝。
<br/>　　“弟兄们，随我杀了这场夜长安，天亮之后封王败相，也与死去的弟兄讨还公道。”
<br/>　　双方在长街之上绽开血战，动静越闹越大，周围的百姓已经被惊动。但他们都保持着事不关己高高挂起的态度，究竟发生什么事，他们并不清楚。至于明天是谁执掌长安城，他们也并不十分关心。
<br/>　　反正不管是谁，坐上那个椅子的都是王八蛋。
<br/>　　却谁也不知，在道路两旁的屋顶上，还趴着两个人。
<br/>　　一人身材矮胖，正是凉州的白元飞。另外一人满嘴络腮胡子，是程大雷的亲随刘发财。
<br/>　　为了关注长安城的变局，白元飞便被派到这里，刘发财也跟着过来凑凑热闹。
<br/>　　“啧，真的打起来了。老白，你说这一局谁会赢？”刘发财问道。
<br/>　　“嘿，你问我我问谁去，你真以为我是徐军师，有未卜先知的本事。”白元飞道。
<br/>　　刘发财撇撇嘴：“城中还有咱们一百多弟兄，你说咱们要帮谁？”
<br/>　　白元飞摇摇头：“咱们谁也不帮，好好看戏，等事情结束后，第一时间将消息禀报给大当家。”
<br/>　　李乐天已经杀到相府门前，路上几乎没有遇到任何阻拦。
<br/>　　但这个时候，相府已经准备完毕。相府之中多有豢养的家丁护院，更不用说那些精心培养的死士。
<br/>　　如今，相府大门紧闭，李乐天还未靠近，便迎来一阵箭雨。
<br/>　　手中都没有趁手的攻城兵器，如此，李乐天竟被相府挡在大门外。第一轮遭遇，鱼龙卫已经折损十几员。
<br/>　　李乐天拧起眉头，忽然喝了一声：“马孟起何在？”
<br/>　　“喂。”
<br/>　　一个声音忽然响起，随着声音一匹快马奔来，到李乐天面前时，他方才勒马停下。
<br/>　　“你这个人不地道啊，我刚跟你，你便让我与你卖命。”
<br/>　　李乐天对他无礼的话语不以为意，伸手一指大门：“你不是要一场大富贵么，破了那扇门，就有你得大富贵。”
<br/>　　少年郎看了看面前紧闭的大门，啐了口吐沫，道：“罢了罢了，人死鸟朝天，不死万万年，今天便是今天了。”
<br/>　　他忽然纵马向大门闯去，手中的大枪抡开，挡掉箭雨。在冲到门前时，忽地从马背上跃起，一脚点在门口的石狮子上，再一跃，拔地数丈，竟然落到大门中。
<br/>　　双方人马都倒抽了口凉气：天下还有如此高手。
<br/>　　李乐天大手一挥：“冲。”
<br/>　　用手中的兵器掩杀，靠近院墙，大门被一撞而开。院子内，马孟起正杀得痛快，不管男女老幼，他逢着便杀，早已是鲜血淋漓。
<br/>　　李乐天带人掩杀进来，与马孟起汇合。这少年郎哇呀呀大叫：“痛快，痛快，再吃我一枪。”
<br/>　　“莫要恋战，往里闯。”
<br/>　　所谓擒贼先擒王，李乐天的目的便是斩杀崔相。那个老人一死，便是树倒猢狲散，自己今天要做的事，便有三分数了。
<br/>　　这些人并非鱼龙卫，李乐天一喝便散，他们对崔相忠心耿耿，只要将他们杀尽杀绝方才罢休。
<br/>　　双方交战，都是死伤惨重。
<br/>(本章完)
