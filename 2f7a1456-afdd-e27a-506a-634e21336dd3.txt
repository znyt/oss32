”<br/>　　<br/>　　    金达说：“那会不会给省领导造成不认真负责的印象啊？”<br/>　　<br/>　　    傅华笑了，说：“像这种不具名的举报信，查与不查都是可以的。”<br/>　　<br/>　　    金达笑笑说：“你这招可是有点损啊。坐等总不是一个办法，我还是把这封信批给公安局把，不管结果会如何，先让他们查着再说吧。”<br/>　　<br/>　　    晚上，秦屯用一部公用电话将郑胜约了出来，两人去了一家很偏僻的茶馆，躲在了包间了。<br/>　　<br/>　　    秦屯把举报信拿给了郑胜看，郑胜看完之后，神色凝重了起来，这封信的内容基本上把他当初谋害伍弈的情形描述了出来，肯定是知情人才会写出这样的信。<br/>　　<br/>　　    秦屯看着郑胜脸上的表情，他想探究一下信上所说的事情是不是真的，郑胜阴沉的表情说明，举报信不是空穴来风。<br/>　　<br/>　　    秦屯心沉了下去，如果牵涉到了杀人，事情就大了，他说：“郑总，你跟我说句实话，事情是不是你做的？”<br/>　　<br/>　　    郑胜这时还在脑海里思索会是谁写的举报信呢，听秦屯这么问，他当然不能承认是自己做的，便说：“秦副书记，你还不了解我吗？我怎么会做出这样的事情呢？”<br/>　　<br/>　　    秦屯冷笑了一声，说：“我就是了解你，才觉得你能做出这样的事情。郑胜啊，我们也一起办了不少事情了，这件事情你可要好好对待，我可不想跟你去坐牢。”<br/>　　<br/>　　    郑胜看了看秦屯，他知道这个伙伴已经开始害怕了，便冷冷的说道：“秦副书记，我也是一条汉子，你放心吧，不论到什么地步，我都是不会牵连你的。出了事我会一肩扛起的。”<br/>　　<br/>　　    秦屯看了看郑胜，他心里对郑胜这个承诺是不敢完全相信的，可是，他也拿郑胜没有别的办法，只好说：“希望你能说到做到。我当初帮你，也是为你好，可不想到了最后被你牵涉进去吃牢饭。”<br/>　　<br/>　　    郑胜笑了，说：“秦副书记，你是为了自己捞钱好不好？说得这么高尚，真是好笑。不过我郑胜这个人是有恩报恩有仇报仇的，你既然帮了我我就不会害你的。”<br/>　　<br/>　　    秦屯干笑了一下，说：“那就好，那就好。”<br/>　　<br/>　　    郑胜说：“不过还有一件事情你需要帮我一下，这一次举报信写得这么详细，公安局应该会启动侦查的，我知道你在公安当中有不少关系，这一次事态很严重，你一定要动用你的关系，让他们把侦查的进展多透露一些出来。”<br/>　　<br/>　　    秦屯也知道这一次事关自己的生死，必须全力运作才有可能转危为安，也就不推辞，说：“行，我会安排的。”<br/>　　<br/>　　    郑胜叹了口气，说：“真是他妈倒霉，我刚觉得事业又有起色了，这嘛事又跳出来，看来我真是需要找人看看了，是不是我踩到了哪个厉鬼的尾巴了。”<br/>　　<br/>　　    求神问鬼往往是人走到了一个失去了主张才会做的事情，郑胜这个时候想要找人看看，说明他已经乱了方寸，不知道该怎么办了，秦屯心中越发担心，看了看郑胜，说：“这个时候你可一定要稳住啊，千万不要自乱阵脚。事情现在还没到山穷水尽的地步，我估计公安暂时也不能拿你怎么样。我研究过了很多案例，好多人最终被定罪，不是因为别人，是因为自己啦把自己交代出来才被定罪的。”<br/>　　<br/>　　    郑胜笑了，说：“你倒是有准备，是为了自己可能进去才研究的吧？”<br/>　　<br/>　　    秦屯干笑了一下，说：“有备无患吧。”<br/>　　<br/>　　    郑胜说：“好了，这个道理我早就知道了，坦白从宽，牢底坐穿，抗拒从严，回家过年，我知道该怎么做的。”<br/>　　<br/>　　    秦屯苦笑了一下，说：“那就好，那就好。”<br/>　　<br/>　　    秦屯说完，看了看郑胜，两人面面相觑，心情都很沉重，都不知道下面该说些什么，房间内一时没了声音。<br/>　　<br/>　　    17、几乎在同时，穆广在关莲的家里，把举报信给了关莲，说：“你看看这个。”<br/>　　<br/>　　    关莲接过信来看了看，她并不清楚这件事情，伍弈出事的时候，他还与郑胜的圈子没有什么关联。<br/>　　<br/>　　    关莲看了看穆广，穆广一脸的阴沉，便小心的说：“哥，你是在担心郑胜出事吗？”<br/>　　<br/>　　    穆广点了点头，说：“你知不知道这件事情是不是郑胜做的？”<br/>　　<br/>　　    关莲摇了摇头，说：“我只是海盛置业的一名职员而已，根本就不能参与到机密，我也不清楚究竟是不是郑胜做的。”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
