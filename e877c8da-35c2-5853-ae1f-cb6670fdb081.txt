　　前往佟千雪所住小区附近地铁站的地铁线有一段是在地面之上，当地铁从隧道中穿出，光亮透过车窗照进车厢的时候，让人有一种拨开黑暗重见光明的感觉。
<br/>　　外面的天气丝毫没有要放晴的意思，天空中乌云很厚，厚重的乌云仿佛随时都会将这片天给压垮一般。
<br/>　　地铁车厢的移动电视上，正在播报着杭山市未来几天的天气。因为受到台风的影响，杭山市未来几天依旧会是以大风暴雨天气为主，主持人提醒市民外出途中注意安全。
<br/>　　坐在慕容飞身边几个上了年纪的大妈看完电视上的天气预报，顿时皱起眉头，对这糟糕的天气发表着自己的不满。
<br/>　　地铁在地面上又行驶了一站路，豆大的雨珠便是倾盆而下，砸落在地铁车厢车窗上发出让人心颤的“噼啪”声，没过几秒钟地铁外的街道马路和楼房就因为这场暴雨而变得扭曲模糊起来。
<br/>　　夏季的暴雨来势汹汹，但持续的时间并不会很长，等到慕容飞到站的时候，外面的暴雨已经是变成了小雨，雨势不大，但是雨点很迷，如果就这么走在路上，很快就会被淋成落汤鸡。
<br/>　　地铁站内，湿漉漉的地面上全是黑漆漆的鞋印，为了防止乘客在追赶地铁的过程中滑到，地铁站的工作人员特意在地铁站内放上黄色的“小心滑倒”的警示标志，并且在一些地方铺上了地毯。
<br/>　　不过这警示标志对于很多乘客来说似乎并没有什么作用，通过安检的乘客从安检机上拿起背包，手里拽着一把还在滴答着雨水的雨伞，听到广播内地铁到站的声音后，拔腿就朝着楼梯的方向跑去，追赶已经到站了的地铁。
<br/>　　慕容飞抬起手腕，看了眼手表上的时间，此时正值上班高峰，这就不难理解这些乘客即便是脚底下打滑，也不愿意放慢脚步了。
<br/>　　这种天气，地铁站中一定会有卖雨伞的小贩，把一块防水布铺在地上，上面花花绿绿的摆满了折伞，二十块钱一把，概不还价，专为那些出门忘记带伞而被大雨堵在地铁站出不去的人准备。
<br/>　　一般来说，这种雨伞也就是一次性使用，那细的也就比牙签粗上那么一点的伞骨，实在让人不敢恭维它的使用寿命，如果是在大风加上暴雨的天气，这种雨伞撑出去基本就是在一阵迎面大风中直接报销的结果。
<br/>　　地铁站出口处，小贩手里拿着好几把包装还算过得去的折伞，嘴里念叨着“二十一把，二十一把”向走出地铁站，而手中没有雨伞的人进行兜售。
<br/>　　“天堂伞看一下了，二十一把。”
<br/>　　小贩将一把蓝色的折伞递到慕容飞面前，慕容飞低头看了眼并没有去接，出租车候车点距离地铁站并不算远，以慕容飞的奔跑速度，慕容飞非常有信心在自己身上被雨淋湿之前冲到出租车候车点。
<br/>　　可是慕容飞刚踏出地铁站，又立刻退了回来，因为他看到出租车候车点处拍着长队，队伍已经超出了候车点雨棚的范围，就算自己奔跑的速度再快，能在雨水淋湿自己之前冲到出租车候车点也无济于事。
<br/>　　这种季节是出租车生意的旺季，与其在公交车站等半天等一辆车厢里挤满了人，闷热无比，黏糊糊湿嗒嗒，空调形同虚设的公交车，还不如多花一点钱打一辆出租车，不仅可以直接送到自己想去的目的地，而且空调还给力。
<br/>　　看到慕容飞退回来，卖伞的小贩又走了上来，把伞递给慕容飞，脸上带着得意的笑容。
<br/>　　慕容飞摸出钱包，取出一张二十元纸币将小贩手中那把出厂价可能连两块钱都不到的劣质雨伞给换了过来。
<br/>　　打着雨伞走到出租车候车点，慕容飞发现旁边垃圾箱中扔满了被风吹翻，全都是来自地铁站出口处那个小贩手中的雨伞。慕容飞暗骂一句“奸商”，排在队伍最后面，等着出租车。
<br/>　　等车的队伍虽然长，但是出租车数量也多，毕竟没有人愿意错过这么一个赚钱的好时机，这样的生意高峰可不是天天都有的。
<br/>　　慕容飞拉开副驾驶车门钻了进去，报了地址，司机按下挡风玻璃后面的“空车”牌子，打着转向灯汇入了车流中……
<br/>　　……
<br/>　　某高档住宅小区。
<br/>　　一身居家服的佟千雪手里抱着一个毛绒玩具，坐在飘窗窗台上，看着外面阴沉沉的天空，还有从云层中不断飘落下来的雨丝。
<br/>　　窗户上，雨珠顺着玻璃往下滑落，留下一道道痕迹。
<br/>　　怀孕的前三个月是最危险的时候，稍有不慎就有可能流产，加上佟千雪在检查的时候就发现有流产迹象，所以这段时间她向医院请了假在家休息，慕容飞的妈妈和自己妈妈会一天隔一天的来这里给她做饭，照顾她。
<br/>　　这段时间，佟千雪也没有什么食欲，基本属于吃什么吐什么，闻到厨房里的味道，就算是喝口水都会吐出来。佟千雪这样的反应让黄虹很高兴，坊间都有孕吐越强烈生男孩子可能性越大的说法。
<br/>　　尽管外面下着雨，气温也不算高，但是这闷热的天气始终让人身上有种黏糊糊的感觉很不舒服，恨不得一整天泡在浴缸里面才好。就算是房间里开着空调，可佟千雪额头上还是多了一层细汗，把头发都黏住了。
<br/>　　佟千雪轻轻抚摸着自己还没隆起的小腹，靠在窗台墙壁上，呢喃道：“坏家伙，你现在在地球什么地方呢？是炎热干燥的中东，闷热潮湿的南美洲，还是冰天雪地的极地？”
<br/>　　“叮咚！”，“叮咚！”
<br/>　　就在这个时候门铃响了起来。
<br/>　　“来了！”
<br/>　　佟千雪对着门口的方向应了一声，抱着毛绒玩具，踢踏着拖鞋去开门。
<br/>　　一般这个时候都是黄虹或者是自己妈妈买了菜过来给自己烧饭，而且这个小区的管理很到位，即便是送快递也是不允许进入住宅楼内的，所以佟千雪没有任何防备的就打开了门……
