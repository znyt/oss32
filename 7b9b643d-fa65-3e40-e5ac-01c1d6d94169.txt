　　而另外一边，烈日宗死伤惨重，南日山浑身血迹，狼狈不堪。
<br/>　　“不！这不可能！你到底是什么人？”
<br/>　　南日山难以置信的大吼了一声，终于开始害怕了。
<br/>　　叶尘摇了摇头，淡淡道：
<br/>　　“为什么你们总是问这么无聊的问题？死人，是不需要知道答案的！”
<br/>　　说完这话，叶尘脚下一步迈出，整个人犹如瞬移一般，已经跨越数千米的距离，来到南日山的跟前。
<br/>　　南日山顿时瞳孔一缩，慌忙跪倒在地，双手举起，道：
<br/>　　“别！别杀我！我愿意将我烈日宗上千年的财富，拿出一半出来，奉献给前辈，恳请前辈饶我一命，可好？”
<br/>　　叶尘两眼微微一眯，冷冷道：
<br/>　　“听上去似乎不错！不过，等我灭了你们烈日宗，一切自然都是我的！”
<br/>　　南日山顿时浑身一颤，继而脸色露出一抹怒色，
<br/>　　“前辈虽然修为惊人，可是这话未免太狂妄了些吧？我烈日宗有百万弟子，今日若是在烈日城，前辈也未必能赢！”
<br/>　　“更何况，我烈日宗隶属于赤阳府，你今日若当着众人的面杀了我，赤阳府是绝对不会放过你的！！”
<br/>　　南日山一说出‘赤阳府’三个字，周围众人顿时又是一片躁动。
<br/>　　“赤阳府！那可是天玄星域的三大势力之一！”
<br/>　　“天玄星域乃是四级星域，远远不是我们天狼星域能够相比的！”
<br/>　　“难怪烈日宗能够称霸天狼星域上千年，原来背后的靠山竟是赤阳府！”
<br/>　　“据说那赤阳府，有好几位化神天君坐镇，势力覆盖周围数十个星域，绝对是巨无霸级的存在！”
<br/>　　“就算这少年，真的具备化神境的实力，也不敢轻易招惹赤阳府吧？”
<br/>　　……
<br/>　　听到众人的议论声，南日山顿时越发得意，脸上露出一抹傲然之色，接着又道：
<br/>　　“我也不怕实话告诉你，我舅舅乃是赤阳府的赤羽天君，你若是敢杀我，他老人家和整个赤阳府，都肯定不会放过你！”
<br/>　　南日山这话一出口，众人顿时再次一片哗然，就连莫长林和流光派众长老等人的脸上，也纷纷露出一抹深深的忌惮之色。
<br/>　　他们虽然之前就知道，烈日宗和赤阳府的关系不浅，却并不清楚，竟然有着这层关系。
<br/>　　难怪烈日宗能称霸天狼星域上千年，果然绝非偶然啊！
<br/>　　“说够了吗？”
<br/>　　一直沉默不语的叶尘，冷冷瞥了一眼南日山，淡淡开口道。
<br/>　　不知道为何，南日山莫名的一阵发颤，感觉自己仿佛被一头猛兽盯上了一般，哆里哆嗦道：
<br/>　　“你、你——你当真敢杀我？”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“我叶狂仙此生，从不受人威胁！”
<br/>　　说完这话，叶尘直接一掌拍落下来。
<br/>　　“不！！”
<br/>　　南日山显然没有想到，在自己报出赤阳府和舅舅赤羽天君的名号之后，叶尘竟然还敢对自己动手！
<br/>　　南日山转身拔腿就跑，可惜他的实力，原本就远远不及叶尘，如今重伤之下，自然更加不堪一击。
<br/>　　轰！
<br/>　　南日山连惨叫都没有来得及发出，就被叶尘那浑厚无比的掌力，活生生打爆，化成了一团血雾。
<br/>　　嘶~~
<br/>　　周围众人看到这一幕，顿时纷纷倒吸了一口凉气。
<br/>　　那可是烈日宗的宗主，天狼星域至高无上的第一人，而且又有赤阳府和一位化神天君做靠山。
<br/>　　任谁也没有想到，叶尘竟然丝毫不留情，说杀就杀！
<br/>　　难道他一人挑了烈日宗不算，还想以一己之力，对抗诺大的赤阳府不成？
<br/>　　疯了！
<br/>　　简直就是疯了！
<br/>　　事实上，叶尘诛杀南日山，虽然看似有些冲动，其实也并非没有考虑过后果。
<br/>　　他先杀了南黎川，而后又斩了烈日宗数十名高手，双方的仇恨，已经是不死不休的局面！
<br/>　　就算他今日饶了南日山，此人也肯定不会善罢甘休，倒不如直接灭掉，以免后患无穷。
<br/>　　更何况，畏畏缩缩，可不是他狂帝的性格！
<br/>　　而此时，
<br/>　　另外一边，
<br/>　　看到叶尘一掌打爆了南日山，楚星河早就吓得尿都快出来，慌忙扭头便跑，生怕叶尘想起之前的事情。
<br/>　　然而，
<br/>　　他才刚刚跑出没几步，一道人影在空中闪过，
<br/>　　下一刻，
<br/>　　叶尘已经如同鬼魅一般，拦在了他的面前。
<br/>　　“噗通！”
<br/>　　楚星河顿时吓得浑身一哆嗦，直接双腿一软，便跪倒在了地上，脸色惨白无比，哆里哆嗦的哀求起来，
<br/>　　“叶、叶前辈……饶命！饶命啊！”
<br/>　　叶尘连看都没有看他一眼，直接屈指一弹，
<br/>　　“忘恩负义之辈！该死！”
<br/>　　嗖！
<br/>　　一道紫色剑气在空中划过，
<br/>　　楚星河瞬间便被剑气，直接劈成了两半！
<br/>　　“啊！！”
<br/>　　周围流光派众人，顿时纷纷吓得连连倒退不止，就连云灵儿也不由得捂着小嘴，脸色有些发白。
<br/>　　虽然她一向不喜欢楚星河的为人，可毕竟也是她的师兄，如今落得如此下场，也不免有些心伤。
<br/>　　而叶尘，随手杀了楚星河之后，似乎并没有罢手的意思，转而又将目光，转移到流光派掌门莫长林的身上。
<br/>　　莫长林顿时吓得面如土色，当即也“噗通”一声，就跪倒在了地上，
<br/>　　“叶前辈恕罪！之前晚辈误信楚星河的一面之词，竟然误会了前辈，实在罪该万死，方才听灵儿师侄说，叶前辈和玄离师兄有旧，恳请看在玄离师兄的面子上，饶过晚辈这一次！”
<br/>　　莫长林说完这话，又连忙望向一旁的云灵儿，哀求道：
<br/>　　“灵儿师侄，师叔平日里待你不薄，你可要替我向叶前辈解释一下啊！”
<br/>　　不得不说，这莫长林到底是老狐狸，瞬间便找到了自救之法。
<br/>　　云灵儿听到这话，果然大为不忍，连忙上前一步，跪倒在叶尘跟前，道：
<br/>　　“叶前辈！掌门师叔也是一时糊涂，轻信了楚星河的鬼话，求您，就饶他一命吧！”
<br/>　　望着跪在地上，眼泪直打转的云灵儿，叶尘终究还是心软了几分，
<br/>　　“罢了！看在灵儿姑娘的面子上，今日之事，我可以不追究……”
<br/>　　莫长林顿时大喜，然而还没等他高兴起来，叶尘话锋忽然一转，
<br/>　　“不过，死罪可免，活罪难逃！断你一臂，算是一个小小的警戒！”
