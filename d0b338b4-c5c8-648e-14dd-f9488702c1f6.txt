　　三儿已经带着两个人把卡宴里的东西搬进了酒吧。
<br/>　　一个马仔手中抱着箱子，没注意到脚下的台阶，脚下踩空一个踉跄，差点摔倒。
<br/>　　三儿把自己手里的箱子放下，对着那个毛手毛脚的马仔一通臭骂，“你他妈没长眼睛啊！？把东西摔了，你他妈有九条命都不够赔的！”
<br/>　　“对、对不起，我、我下次注意。”马仔唯唯诺诺的道着歉，一脸惶恐。
<br/>　　“还想要有下次！？”
<br/>　　三儿一巴掌扇在马仔脑袋上，骂道：“还觉得不够丢人现眼？滚！傻比！”
<br/>　　马仔把箱子放在吧台上，赶紧离开了这个是非之地。
<br/>　　李霄力坐在吧台前，点燃了一根雪茄对着三儿招了招手，向他问道：“三儿，最近那女人那边什么情况？”
<br/>　　“哪个女人？”三儿满脸茫然。
<br/>　　“就是布克尔特让我们盯着的那个女人，叫佟、佟什么来着？”李霄力一时间想不起了佟千雪的名字。
<br/>　　“哦，您说的是佟千雪啊。”三儿立刻反应过来，说道：“就是市儿童医院那个医生。”
<br/>　　“对，就是那女人。”李霄力对三儿问道：“最近佟千雪那边什么情况？”
<br/>　　“也没什么特别情况，我花钱把他们小区的一个买通了，一直盯着呢。”三儿回答道：“她好像怀孕了，这段时间一直在家里养胎呢。”
<br/>　　“你还真别说，那女人长得真不错。”李霄力摸着自己的下巴，眼睛中闪烁着欲望，说道：“这女人要是能玩一晚上，那滋味肯定不错，而且还是个医生，制服诱惑，啧啧啧……”
<br/>　　“力哥，那我找机会把人给你绑了？”三儿试探着对李霄力问道。
<br/>　　“你他妈是不是蠢？！”
<br/>　　李霄力对着三儿的脑袋就是一巴掌，瞪着他骂道：“那女人是我们能动的吗？她男人是部队里的，级别还不低，连布克尔特都不敢轻举妄动，你有几条命敢去动她！？就你手下那几把土枪和化隆造能干的过别人的机枪和自动步枪？惹了她，我们就该被连锅端了！”
<br/>　　“力哥，你都不把普提查放在眼里，还打算跟布克尔特合作啊？”三儿对李霄力问道。
<br/>　　“普提查是普提查，布克尔特是布克尔特。”李霄力抽了口雪茄，说道：“只要盯着那个女人，就能从布克尔特那里拿钱，这白送钱的买卖干嘛不做？谁会嫌钱多？今天晚上，让你买通的那个保安去‘碧海潮生’玩玩，拉拢拉拢关系，给人点甜头，人家才会更卖力的干活。”
<br/>　　“明白了力哥，我回头就去联系他。”三儿回答道。
<br/>　　李霄力从吧台前站起身来，从口袋中掏出那瓶装着三颗“水晶珠”的透明小瓶扔给三儿，说道：“让他试试。”
<br/>　　“明白，我会安排好的。”三儿收好瓶子说道。
<br/>　　“好好做，我不会亏待你。”李霄力带着警告的口吻，对三儿说道：“但是如果做不好，什么样的下场，你应该很清楚！”
<br/>　　那个被三儿买通的小区保安就是慕容飞要进小区时，质疑慕容飞证件是假冒的年轻保安。
<br/>　　年轻保安姓钱，单名一个“鑫”字。名字里面加起来有四个“金”，可以看得出他家里人对他有多大的期望，但是钱鑫这小子不争气，高中没考上，上了一个中专。本来家里人还想着钱鑫能读中专学一个手艺，以后也方便找工作，现在这个本科生大规模泛滥的社会，拥有一项专业技能，要远比那些拿着学位证书和四六级英语合格成绩单的本科生更容易找工作；可是钱鑫这小子不争气，读书的时候屡次违反校规校纪，书还没读完，就被学校给开除了。
<br/>　　被学校开除后，钱鑫就在社会上到处混，也没有工作，彻底成了一个“啃老族”。家里人找了很多关系，才给他找到了这份小区保安的工作。
<br/>　　这个小区的保安工作制度为标准的“三班倒”，今天是钱鑫的第二个白班，就在钱鑫考虑着下班后去哪里放松一下的时候，他放在桌子上的手机响了起来，钱鑫看到手机上的来电显示，连忙接听了电话，“三爷。”
<br/>　　“小钱啊，这段时间怎么看不到你来玩儿啊？”电话那头的三儿对钱鑫问道。
<br/>　　“这不是手头紧嘛。”钱鑫讪笑道：“我一个月就这么点工资，一个月玩儿一次都过得紧巴巴了，这经常来我不得喝西北风啊。”
<br/>　　“我昨天刚升了职，这次算我请你的。”三儿十分豪爽的对钱鑫说道：“今天晚上，‘碧海潮生’一切消费都算你三哥的。”
<br/>　　“真的！？”
<br/>　　钱鑫眼睛就跟探照灯一样闪亮，“碧海潮生”夜总会可是杭山市出了名的销金窟，一晚上的平均消费都是以“万”作为单位来计算，以钱鑫的收入水平肯定是消费不起的，所以听到三儿在电话中肯定的语气，钱鑫心中的兴奋可想而知。
<br/>　　“我什么时候骗过你？”三儿笑着回答道：“我已经打过招呼了，晚上你去‘碧海潮生’随便玩，全部免费。我还帮你找了个漂亮妞儿，保准让你满意。”
<br/>　　“谢谢三爷，谢谢三爷。”钱鑫忙不迭跟三儿道谢，如果此时三儿站在钱鑫面前，钱鑫说不准就会直接给三儿跪下叫爹了。
<br/>　　钱鑫挂断电话后，赶紧从椅子上站起来，手忙脚乱的跑到自己的更衣柜前，打开更衣柜从里面翻出一瓶通过海外购买来的号称“三十分钟见效，用完就能让女人叫破喉咙”的“神油”。
<br/>　　“妈的，终于能用到这东西了。”
<br/>　　钱鑫赶紧把“神油”藏进衣柜口袋中，幻想着晚上能把三儿给他安排的美女狠狠征服。
<br/>　　下午下班后，钱鑫先是回了趟家，睡了一觉养足精神，然后洗了个澡，还喷了一点冒牌阿迪达斯香水，然后骑上电瓶车直奔“碧海潮生”夜总会而去。
<br/>　　钱鑫心中是一阵阵激动，恨不得屁股下的电瓶车多出两个翅膀，直接飞到“碧海潮生”夜总会去。
