　　后山绝地，田不易满脸凝重看着眼前景象，周围草木茂盛，唯独中间圆形区域寸草不生。
<br/>　　“好一个凶险之地。”田不易脸色大变。
<br/>　　苏重四处打量，感受着丝丝缕缕飘散在空中的凶戾气。他竟意外发现，这里极其适合修炼庚金剑气。这会儿功夫，道种制造剑气的速度就大大提升！
<br/>　　“可惜，根源已被张小凡血炼，不然效果会更好。”
<br/>　　田不易四处查探，最终也没看出什么头绪。不过最后倒是确定不是人为，纯属意外造成，只能草草回山。
<br/>　　张小凡气血衰败，田不易认为是绝地影响，给他放了几天假，让张小凡好好休息。其他几位师兄弟，被师娘苏茹扔进黑竹阵法里修炼。
<br/>　　照顾张小凡的任务就落在了苏重头上。
<br/>　　“二蛋，我没事，你快去修炼吧。”张小凡怕耽误苏重修炼，不断劝阻。此刻他躺在床上，满脸苍白，说起话来都有气无力。
<br/>　　苏重深深看了张小凡一眼，把别再身后的黑色短棍递给对方：“血炼最是损耗气血，同样凶险万分，这次你能活着回来纯属侥幸。”
<br/>　　张小凡眼神躲闪，看着黑色短棍讷讷不言。
<br/>　　“小凡，这东西经过你血练，他就是你的。不过两件兵器全都充满邪意，平日里看不出来，时间长了必会影响心智。如果要用，必须想办法抵挡侵蚀。“
<br/>　　张小凡沉默不语，他想到自己差点儿被两件法宝吸干的事。不过对于什么侵蚀心神的说法，他还是半信半疑。不过一件兵器，还能改变人的想法？
<br/>　　苏重一眼就看出张小凡不信，长叹一口气悠悠道：“那个老和尚算的上是得道高僧了吧。”
<br/>　　“什么老和尚？”张小凡那顿时紧张起来。
<br/>　　“就是教你功法的那个老和尚。”苏重瞥了对方一眼直言道。
<br/>　　“你知道？”张小凡惊的差点儿从床上蹦起来。
<br/>　　“他可不止教了你功夫。”苏重不理对方：“你只是个普通山村小儿，如何才能让你进入青云，以达成他的目的呢？青云是天下第一大派收徒严格，怎么才能让你进入青云？”苏重盯着张小凡郑重道。
<br/>　　张小凡顿时呆愣当场！
<br/>　　“小凡啊，该想想了。我去给你做饭，你好好休息。”看着陷入迷茫的张小凡，苏重起身离开。
<br/>　　等苏重端着饭菜回到张小凡房间的时候，张小凡双眼通红。
<br/>　　“是他对不对，是他杀了全村人对不对！”张小凡满脸狰狞。
<br/>　　“是他。”苏重点头。
<br/>　　张小凡一下从床上窜起来就往外跑。
<br/>　　“他是天音寺四大神僧，你确定青云会为了你我去找他们的麻烦？”苏重淡淡道。
<br/>　　张小凡立刻呆滞当场。青云是不可能为了一个徒弟就和天音寺开战的。而且普智已经死了，还怎么讨说法。
<br/>　　往阴暗的方面想一想，青云甚至会对此事缄口不言，然后再想办法从张小凡口中套出大梵般若秘诀。这可是和太极玄清道齐名的功法密决。
<br/>　　张小凡一下瘫坐在地上。
<br/>　　苏重把噬魂递给张小凡：“那天在后山，这东西一直攥在你手里，我没告诉师傅，拿着吧！”
<br/>　　张小凡看到黑色短棍，眼神一下狰狞，抓过黑色短棍就狠狠扔出门外。
<br/>　　这上面有嗜血珠，那是普智给他的。他本来一直当宝贝留着，现在却得知普智是自己的大仇人。张小凡一点儿都不想碰普智给他的东西。
<br/>　　“小凡啊，想要讨个说法，别人是靠不住的。只能靠自己！这根短棍是不可多得的法宝，且经过血炼，和你心意相通威力无穷。到底要不要用，你自己看着办吧。”苏重起身离开。
<br/>　　原本轨迹，张小凡一直被蒙在鼓励，苏重觉得还是让他知道的好。最后如何选择，还看张小凡自己。
<br/>　　知道真相，张小凡最近经常发呆。也不用苏重怎么照顾，他只要每天把饭菜送过去就行。
<br/>　　青云门灵气浓郁，山上有不少灵草。苏重结合藏书阁医术，改制了一些补充气血的药膳。除了精神不振，张小凡身体很快即恢复过来。
<br/>　　清晨，苏重起床后就在竹屋内修炼道种诀。有了道种后凝聚符文速度大大提升。以前需好久才能凝聚一枚，现在只要一刻钟。
<br/>　　但新的问题出现，他吸收戾气的速度快，周围的戾气却不够他吸收！
<br/>　　“诛仙剑阵用的就是戾气，我这个消耗天机印相比起来，笼罩七脉搏主峰的天机印才是真的大手笔。想要和这么庞大的诛仙剑阵抢食，难如登天！”苏重苦笑：“怎么才能更快的凝聚剑气符呢？”
<br/>　　融合庚金剑气后的道种，威力全都基于剑气符数量。他凝聚的剑气符越多，他的持续攻击时间越长。也就有更多时间让道种吸收戾气。一定程度实现消耗与恢复的平衡，从而达到速回气效果。
<br/>　　苏重正在苦思，张小凡却敲门走了进来。
<br/>　　“二蛋，我已经好了，你不用再给我单独送饭。”张小凡那一手握着黑色噬魂短棍，肩膀上蹲着一只灰色小猴子。稚嫩的脸庞平静深沉，竟然有种突然成熟起来的感觉。
<br/>　　“你想明白就好。这根棍子很厉害，但他会侵蚀身心。老和尚当初发疯，就有这颗嗜血珠的原因。想要用它们，就要做好准备。你从给老和尚哪里得到的功法不能停，它不能解决问题，但能延迟发作时间！”苏重道。
<br/>　　“我知道了。”张小凡盯着黑色棍子看了半晌，然后再次握紧噬魂短棍：“二蛋，我去找师兄们修炼去了。”
<br/>　　“去吧，先把聚元功练好。然后就可以两种法诀同时修炼，不要着急，前期虽然慢点儿，后来却会快速追赶上。”
<br/>　　“我知道了！”
<br/>　　看着张小凡远去背影，苏重不知道最后结局如何。即便仍然没有改变，张小凡也不至于像原著中那么被动的接受一切。
<br/>　　“对了，我怎么忘了那个地方！”苏重看着张小凡才想起来，后山还有一块凶险绝地。
<br/>　　等他来到后山，看到寸草不生的圆形区域，感受着弥漫空气的丝丝缕缕凶戾之气，顿时满意的笑了。
<br/>　　“虽然没了源头，但剩下的这些戾气，也足够我前进一小步了！”
<br/>　　一旦沉浸修炼，苏重就不知时间流逝。等他回过神来的时候，那篇圆形区域内，竟然已经长满黑竹。虽然矮小细嫩，但却是已经看不见原本荒凉模样。就连中间那个小水洼也消失不见。
<br/>　　“这里已经不适合修炼了。”苏重颇为遗憾，他还想多修炼一段时间。内视丹田，360枚剑气符在丹田沉浮：“有这收获也不错！”
<br/>　　经过这短时间没日没夜吸收炼化，他终于凝聚出了360枚剑气符。
<br/>　　庚金剑气以360枚符篆为一个阶段，修炼到这里就已经算是一个小高手，可以作为战力应用。剩下的就是不断积累剑气符，积累的越多，爆发力持久作战能力越强。
<br/>　　“相比于简单积累剑气符，还是剑气符阵更有趣啊。”苏重眼神闪烁。
<br/>　　剑气符阵是《庚金剑气》内提出的一个概念，用剑气符做为根基构建阵法。释放剑气时威力更强。
<br/>　　虽然只有一个简单模型，甚至只能算是构思，却给了苏重不少灵感。
<br/>　　“剑气符阵大有可为！”
<br/>　　这段时间又是黑竹阵法又是天机印，他对阵法二字特别敏感。青云最厉害的诛仙剑阵同样是阵法。如果一个人就能拥有一座诛仙剑阵，那该多厉害？
