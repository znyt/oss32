　　地球，静海市。
<br/>　　偏西北方向，一条昏暗的小巷中，一名白衣青年半靠在墙上，嘴角挂着血痕，腰间冒着黑色火焰。
<br/>　　此时，慕清霄闭目养神，脸色苍白，嘴角挂着一丝苦涩。
<br/>　　他用巽芳的信息，换取到了自己想到的东西，果真如他所料，交易完成后，长琴太子第一时间对他下手。
<br/>　　回想起刚才的那一掌，依旧心有余悸，长琴太子不愧是半仙级别的存在，一记翻天印横渡虚空，恐怖如斯。
<br/>　　若非刚才跑的快，在晚上一秒，在那一掌下，绝对是魂飞魄散的下场。
<br/>　　饶是如此，也在恐怖的冲击力下，落得个半残的下场，腰间更是直接被轰出一个血洞。
<br/>　　好在他的恢复能力强大，否则，后果不堪设想。
<br/>　　此时，腰间血洞上，九霄琉璃焱跳烁，血液停止流淌，甚至，以肉眼可见的速度恢复起来。
<br/>　　压下心中的情绪，慕清霄将储存空间中的烛龙之鳞取了出来，微微一笑，道：“呵呵，不亏。”
<br/>　　古剑奇谭位面终于结束，天墉城有紫胤真人在，百里屠苏估计不会有危险。
<br/>　　慕清霄心中也无任何愧疚感，他在天墉城的十年里，炼制丹药无数，最起码帮弟子辈实力提升了一个台阶。
<br/>　　将古剑奇谭的信息抛之脑后，慕清霄注意力都落在仙器上，只见他双手结印，咬破舌尖，一滴金色血液滴落在烛龙之鳞上。
<br/>　　顷刻间，原本黯淡无光的烛龙之鳞，其上光泽流转，就像是活了一样。
<br/>　　设下灵魂印记后，慕清霄将烛龙之鳞收进储存空间，神识就向四周扩散而出。
<br/>　　很快，以他为中心，四周千米的范围都尽收眼底，望着现代化的街道汽车，来往人群......
<br/>　　收回神识，慕清霄剑眉皱了皱，现在竟然才九月份。
<br/>　　也就是说，他在古剑奇谭位面待了整整十年，地球才过去七个月，不应该是两年吗？
<br/>　　“系统，为何地球才过去七个月？”
<br/>　　“叮，每个星域位面都有属于自己的时间法则，暂且不是宿主能够接触的。”
<br/>　　时间法则？
<br/>　　闻言，慕清霄神情怔了怔，沉默片刻，等伤势恢复的差不多，便换上修身裤与衬衫，离开小巷。
<br/>　　...
<br/>　　...
<br/>　　站在金帝别苑外，望着周围的情况，慕清霄微微一笑。
<br/>　　此时，金帝别苑相比七个月前，已经有了天翻地覆的变化。
<br/>　　销售大楼依旧矗立在最前方，只是周围的别墅与凉亭已经被夷为平地。
<br/>　　外围也被栏杆重重拦了起来，只留下一扇进出大门，其余地方都有禁止进入的警告标识牌。
<br/>　　由于道路上的树木没有处理，也有高楼的遮挡，旁人根本看不见最中间的别墅。
<br/>　　跨进大门，轻车熟路的走在道路上，其中除了一些树木，四周已经空荡荡一片，只有最后面一栋别墅留了下来。
<br/>　　包裹别墅的高墙也已经拆除，跃过销售大厦，别墅清晰的映入眼中。
<br/>　　空间微微一阵波动，慕清霄的身影就出现在门口的游泳池旁，回头看了眼身后空荡荡的一片，眼中闪过一抹思索。
<br/>　　收回目光，推门进入别墅中，一股熟悉的幽香就钻进鼻孔中。
<br/>　　深吸了两口，慕清霄感觉整颗心都平静了下来，也唯有家才能给他这种独特的感觉。
<br/>　　来到客厅，液晶电视正播放着电视剧，茶几上放着水果与零食，却不见人影。
<br/>　　“难道都出去了？”
<br/>　　可刚坐在沙发上，却听见浴室中传来一阵水花拍地的声响。
<br/>　　嘴角牵起一丝邪异的笑容，慕清霄轻手轻脚的走了过去，刚推开浴室大门，一具完美无暇的玉体就映入眼中。
<br/>　　雪白细腻的酥颈，顺着水流往下看，是两座圆润饱满的圣女峰，峰顶的嫣红令人口干舌燥。
<br/>　　平坦没有一丝赘肉的小腹，再往下便是最神秘的幽谷。
<br/>　　似乎发觉有些不对劲，雏田转过身来，看到站在门口愣神的慕清霄时，脸蛋瞬间涌上一片红晕。
<br/>　　“你......你回来了。”
<br/>　　出乎意料的是，雏田除了有些羞涩，并没叫出声来，莲藕般的玉臂抱在胸前，挡住诱人的春光。
<br/>　　“嗯，回来了。”
<br/>　　说罢，慕清霄走上前去，伴随着一阵惊呼声，就将诱人的玉体抱在怀中。
<br/>　　要知道，在古剑奇谭位面，他可是守身如玉，近十一年没有碰女人，刚回来就看到这一幕，哪里受得了。
<br/>　　直接将雏田抱在怀中，一手把玩着胸前的圆润，一手搂着她的腰肢，狠狠的吻住那诱人的红唇。
<br/>　　“唔唔！”
<br/>　　伴随着一阵娇吟声，雏田知道逃不出他的魔爪，莲藕般的玉臂搂住他的颈部，羞涩的回应起来。
<br/>　　感受到佳人的情意，慕清霄吻的更加猛烈，撬开唇齿，瞬间就捕捉到一条润滑小舌，彼此交缠......
<br/>　　两人相拥，足足缠绵亲吻了十多分钟，直到雏田气喘吁吁，慕清霄才一路向下，开始侵占玉体别处。
<br/>　　感受到胸口奇妙的感觉，雏田美眸迷离，粉唇轻启，羞涩道：“别，别在这里，回房间里。”
<br/>　　闻言，慕清霄一把将她拦腰抱起，快速向屋内冲去，几分钟后，其中就传来一声痛呼。
<br/>　　半响过后，屋内就响起低吟浅唱之声。
<br/>　　不知何时，别墅大门打开，两道倩影走了进来，突然，两人似乎听到某卧室中传来怪声，悠长低吟。
<br/>　　陆雪琪与阿碧面面相觑，顿时眼中闪过一抹羞涩。
<br/>　　经历过男女之事的她们，自然清楚知道这是什么声音，相视一眼，蹑手蹑脚，偷偷的向卧室门口走去。
<br/>　　“色鬼，真的回来了。”
<br/>　　偷偷将卧室们打开一道缝隙，望着其中的一幕，两女脸颊瞬间涌上一片绯红，心跳如雷，腿都有些发软。
<br/>　　甚至，眼眸中闪过一抹春色，但最后却将房门关上，坐在沙发上，心不在焉的看起了电视。
<br/>　　随着时间的推移，夕阳西下，霞光满天，坐在客厅中，两女依旧能听到卧室中的婉转低吟。
<br/>　　终于，伴随着一声高呼，声音才逐渐停歇。
<br/>　　坐在客厅沙发上的陆雪琪与阿碧，早已经身体发软，心里乱糟糟的。
