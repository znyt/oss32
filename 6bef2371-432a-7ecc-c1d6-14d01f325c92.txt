　　第396章
<br/>　　李行哉就立在宋伯康面前，披着毯子，下身只有一件犊鼻裤，不停吸溜着鼻涕，看上去不伦不类，毫无帝国皇子的风度。
<br/>　　宋伯康愣了好大一会儿，才试探的问道：“殿下，怎么失火了？”
<br/>　　“嘿，老宋，你可有点不地道！”
<br/>　　李行哉手重重砸在宋伯康肩膀上，砸得宋伯康一头雾水，可李行哉紧跟着的话，却令他浑身冷汗。
<br/>　　“你派来的那两个歌姬要杀我啊。”
<br/>　　“呃……这不大可能吧？”
<br/>　　提着两颗人头的福德勒在李行哉背后出现，阿珠阿翠，一对千娇百媚的解语花，如今只剩下两颗血淋淋的脑袋。人们吓了一跳，说也奇怪，大家想起她们活着时有多美艳，现在就能给大家带来多大的恐怖。
<br/>　　“怎么不可能，你可是不知道，这两个女人可是蛇蝎心肠，我特别宠爱他们，但没想到她们要放火烧死我，幸亏福德勒护驾及时，杀了二人，我才幸免于难。可是吓得我不轻。”李行哉如此说着，但脸上丝毫没有害怕的意思，他同时还冲周围挥着手，一脸洋洋得意。
<br/>　　宋伯康难以置信的打量着这位帝国皇子，现在他的样子，像个无赖、混混、乞丐……但就是没有半点帝国王子的风度。
<br/>　　就算六皇子曾经是个废物，但风花雪月，玩世不恭，却也有些风流味道，怎么现在自甘堕落到这个样子。
<br/>　　宋伯康愣了愣，忽然想起李行哉在蛤蟆寨待了很长时间。想想程大雷的为人，似乎一切都解释得通了。
<br/>　　他心思急转，思考着这件事代表着什么，也就是李行哉这么做的用意。很快，他就平静下来，道：“给殿下重新安排住处，严加保护，绝不能让这样的事情再次发生。”
<br/>　　目光有意无意的扫过福德勒，用冷冰冰的语气道：“让他和殿下分开住。”
<br/>　　李行哉一怔，脸上的笑容瞬间消失。
<br/>　　宋伯康却是笑了笑，道：“天色不早，殿下又受了惊吓，还是早些休息吧。”
<br/>　　城主府的大殿上灯火通明，陆洛冷哼一声，道：“这火必然是他自己放的，竟然还敢杀了我们的人。”
<br/>　　“大人，要不要给他点颜色看看，让他知道，这里不是长安，他还真当自己是帝国皇子么。”
<br/>　　送宋伯康脸上却带着笑意，道：“好了，现在他能做的也不过是这些了，我不知道这有什么好生气的。”
<br/>　　陆洛一楞，道：“大人，他可是在挑衅我们。”
<br/>　　“挑衅，我看谈不上吧，这无非是撒泼的手段而已。”宋伯康道：“就像一个在婆家受了气的小媳妇，能做的也不过是摔摔碗，板起脸，其他的什么都做不了，不用放在心上。”
<br/>　　本来，宋伯康对李行哉还有些忌惮，士别三日当刮目相待，曾经是废物，可不代表永远都会是废物。但现在李行哉只能用这种幼稚的手段，来表达自己的情绪。可这幼稚的手段，又有什么用。他杀了两名歌姬，宋伯康能继续送过去监视他的歌姬，他烧毁一座院子，宋伯康就有三座院子，四座院子给他烧，哪怕他将整座城主府烧成灰烬，宋伯康也毫不在乎。
<br/>　　终究，他不是值得令宋伯康忌惮的存在。
<br/>　　……
<br/>　　琴川关，将军府。
<br/>　　相比长安城的紧张，凉州城的暗潮汹涌，琴川关可以称得上是花团锦簇。虽然是在肃杀的冬天，却依旧能感受到此间的欣欣向荣。
<br/>　　这是一个晴天，虽然寒风呼啸，但仍旧是艳阳高照。书房内烧了暖炉，很快便热了起来，程大雷窝在塌上，听阿喜汇报最近的情报。
<br/>　　长安城的情况仍旧不妙，也说不好程大雷支持朝廷还是支持正义教，也或许那个都不支持。现在两方面仍在对峙，互相之间打得仗越来越少，可外围诸侯和江湖草莽之间的战斗却是每天都在发生。
<br/>　　今天黑胡子杀了王牧之，明天马朝安灭了孙五六，名字乱七八糟，具体谁属于哪个势力，程大雷自己也理不清楚。
<br/>　　今天程大雷收到一封信，是高飞虎从江南送来的。高飞虎在江南的山寨，算是蛤蟆寨的分店，送信过来的意思是看程大雷要不要出兵，高飞虎本身是想要趟一趟这次浑水的。
<br/>　　“大当家，咱们要不要去长安？”阿喜问道。
<br/>　　“山寨里的弟兄是不是都想去？”
<br/>　　阿喜有些尴尬的点点头。
<br/>　　程大雷无可奈何的叹了口气，虽然琴川关建了学堂，也着力培养山寨自己的人才，可江山易改本性难移，这些人骨子里还是一群江湖草莽而已。容易热血冲头，二杆子精神大爆发。现在长安城是一团乱麻，局势瞬息万变，说不得蛤蟆寨也能扫荡诸侯，驱逐群贼，占了龙庭，然后自己也弄个丞相将军当当。
<br/>　　高飞虎打的就是这心思，山寨弟兄也大多是这心思。
<br/>　　“回信告诉高飞虎，打死也不需出兵，自己的事情还管不过来，瞎凑什么热闹，给我老实待着。”
<br/>　　阿喜点头答应，示意自己明白。看他的意思，是也有几分蠢蠢欲动的，只不过程大雷没有出兵的念头，他也之后老实窝着。
<br/>　　“凉州城那边是什么动静，时间过去这么久了，还没有出兵么？”程大雷问。
<br/>　　阿喜点点头，道：“大当家，长安城的情况，可能有些不对？”
<br/>　　“喔，怎么不对？”
<br/>　　李行哉推测得不错，凉州城的确有蛤蟆寨的暗线。事实上不仅凉州城，周围的几座城池，程大雷都埋着眼睛呢，他这个人，对情报工作一向都是很重视的。
<br/>　　“据手下打探到的消息，最近宋伯康似乎没有出兵的意思，城里半点风声没有透出来。据说殿下最近花天酒地，夜夜笙歌，还有最近城主府内着了一场火，具体原因尚未查明。”
<br/>　　程大雷这就有些想不明白了，既然不打算出兵，宋伯康接走李行哉做什么。以李行哉的心情，此刻当是迫不及待赶到长安。
<br/>　　当然，程大雷也不希望李行哉在长安和林少羽碰着面，双方谁杀了谁，程大雷也不愿意看到。只是现在的情况，真透着一点奇怪了。
<br/>　　“派人去查查。”
<br/>
