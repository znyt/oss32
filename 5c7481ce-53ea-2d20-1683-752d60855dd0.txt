　　人间与恶灵空间之间的壁垒，十分平滑，不像跟青冥界或者鬼域那样，轻易就能打开虚空裂缝，两边的生灵想要到达对方的世界，十分难以做到。
<br/>　　因此，古往今来，也没几个生灵穿越到恶灵空间去的（灵气比人间还稀薄，毫无吸引力，这也是一个原因），为了防止有生灵反穿过来，人间法术界一直修筑着两个空间的壁垒，不断加固，就是避免这种情况。
<br/>　　自己面前这个叶少阳，居然是从恶灵空间里来的！
<br/>　　“你……你是怎么出来的！”
<br/>　　“靠我自己，当然出不来。”叶少阳走到她面前，歪头打量着她，说道：“告诉你无妨，星月奴在当代，一共收了六大关门弟子，有五个都被道风杀了，最后一个相传十分神秘，没人见过，你可知道，他是谁？”
<br/>　　这意思已经很明显了，芮冷玉吃惊地望着他，居然……是他？
<br/>　　这个真是永远都想不到。
<br/>　　叶少阳深吸一口气，转身望着洞穴入口方向，伸手感知了一下，脸上露出笑意：“打起来了。”
<br/>　　停顿了一会，他悠悠说道：“是我师父进入恶灵空间，找到了我，把我领到这个世界，我闭关修炼数年，我，就是她最得意的弟子，也是她一枚蛰伏的棋子，而今，是我用行动来报恩的时候了。”
<br/>　　“你师父？就算你是恶灵空间的，你师父不也是青云子吗？你背叛了师门？”
<br/>　　叶少阳笑起来。
<br/>　　“什么师门，如果你在那世界里呆过，你就会知道，一切感情都是假的……我曾经也坚信过道义，为了她，我对抗全世界，但是到最后，我却被她骗得一无所有……在我濒临死亡的时候，我师父救了我，带我到了这个世界，我现在没什么可以坚持的，除了我自己。”
<br/>　　他定定的望着芮冷玉，咬牙切齿的模样，让芮冷玉十分不适。一团邪恶的气息，在他额头萦绕不去。
<br/>　　“你，入魔了。”芮冷玉淡淡说道。
<br/>　　“魔？哈哈，什么神魔，什么好人坏人，不过是主宰们手中的一颗棋子，只有跳出棋盘，才能掌控自己的命运，而这过程中，没什么是不可以牺牲的，冷玉……你可知道，当初欺骗我的人，就是你！是你害的我失去了所有！是你让我从人间道神变成了人人喊打的怪物！”
<br/>　　芮冷玉震惊了，半晌摇了摇头，“那不是我。”
<br/>　　“对，那不是你，所以，我也没打算伤害你。”
<br/>　　芮冷玉不跟她纠缠这个，进而问道：“你把我们引来这里做什么？”
<br/>　　“跟你没关系，你是自己跟来的。我是要引道风过来，然后……杀掉他。”
<br/>　　“他是你师兄啊！”芮冷玉愣了一下之后冲他吼了一声。
<br/>　　叶少阳缓缓摇头，“在这个世界，我没什么亲人，而且……他非得死不可，他若不死，我的计划就没办法进行下去。”
<br/>　　叶少阳说着，又朝外面看了一眼，从这里看不到洞穴外头，但可以感受到一阵阵的气息波动，显示着外面的战斗正在激烈进行中。
<br/>　　芮冷玉虽然被定了魂，魂魄无法移动，自然也就无法操控肉身，但她的感官还是有的，她也感受到了外面的异动，说道：“别白费心机了，你们杀不了道风的。”
<br/>　　“没有人是杀不死的。道风，固然很厉害，但你不要忘了，这里是轩辕山！这里最不缺的就是强者，随便一个强者出去，都可以横扫人间，道风，呵呵，就算他浑身是铁，又能打成几根钉？”
<br/>　　芮冷玉目光闪烁，思索片刻，想要叫他，但只说出一个“少”字，便闭上嘴，然后自语道：“你不是叶少阳。”
<br/>　　“很快，我就是真正的叶少阳了。”叶少阳看着她，阴恻恻地说道。
<br/>　　这种表情，更加让芮冷玉对他有了一种疏离感。
<br/>　　她认识的叶少阳，绝不是这个样子：平时那个逗比的叶少阳就不用说了，就算叶少阳戾气爆发、陷入癫狂，也绝对不会是这种阴狠的样子。
<br/>　　他更像是一个暗黑版的叶少阳。
<br/>　　芮冷玉冰雪聪明，听了叶少阳这句话，仔细一想，猛然明白了，倒吸一口气，道：“你要取代叶少阳！”
<br/>　　“你一直都是聪明的，本来，我是不想告诉你的，想一切成功了之后，再以叶少阳的身份去见你，见你们所有人，没人会怀疑的，但你既然跟来了，那便告诉你也无妨，反正，我们还有很多时间。”他又看了外面一眼，微笑着说道。
<br/>　　“这个世界的叶少阳，被困在了古墓里，他，永远都不可能出来，我完全可以取代他，神不知鬼不觉……道风是唯一的障碍，他跟叶少阳一起生活了十年，只有他能发现其中的真假……所以，他必须死，不过，我们本来的撒算是先除掉古墓里的叶少阳，再来对付道风，但是等不及了。
<br/>　　道风日前已经拿到山海印，我们担心他会破坏整个计划，于是我才假意骗道风来这里，哈哈，他果真上当了！”
<br/>　　芮冷玉越听心越冷，喃喃道：“是啊，如今想来，这是多么拙劣的手段……你根本连故事都没有编好。”如今仔细想来，叶少阳这个提议，简直漏洞百出，他是怎么从古墓出来的？是谁告诉他星月奴的肉身在这里的？
<br/>　　他为什么如此相信那个人，冒这么大的风险来到轩辕山？
<br/>　　叶少阳也听明白了她的意思，笑道：“这一切当然是漏洞百出，但因为是我说的，所以，一切不合理也都合理了。”
<br/>　　因为他是叶少阳，而听的人是道风。包括芮冷玉自己，还有杨宫梓，他们对于叶少阳，都有绝对的信任，不会怀疑他说的任何一句话，更不会怀疑他的动机。
<br/>　　所以叶少阳一说，他们都相信了，然后来了这里……
<br/>　　“所以，你很引以为荣吗？”芮冷玉感到悲哀之余，反唇相讥。
<br/>　　“不，我也不想他死……”叶少阳眼神暗淡了一瞬间，尔后又现出了决然之色，“但是，我要活下去。况且我与你们不是一个世界的人，这么做也不算是背叛。”
<br/>(本章完)
