<br/>  青崖口，位于新奉县南郊的一座山上，此山并不大，但却险峻，而就是在这里有着一众凶悍山匪，他们就是以此为名……<br/>　　<br/>　　  青崖口山匪，有三大当家，号为石家三兄弟，老大石坚，老二石勇，老三石奉。<br/>　　<br/>　　  不过石奉现在已经被王康砍了脑袋。<br/>　　<br/>　　  在一名山匪的带领下，徐允jinru了山洞，他在离开县衙将家眷安顿好，就联系好人，上了山！<br/>　　<br/>　　  正如王康说的那样，他跟青崖口山匪有着很深的联系，甚至他能在此前坐稳新奉县县令一职，背后就有青崖口山匪的扶持。<br/>　　<br/>　　  这次山匪袭击伯爵府车队，就是他在董易武的授意下，把情报泄露给青崖口山匪的。<br/>　　<br/>　　  他来这的目的，自然也是为了对付王康，阻碍富阳伯爵府接收新奉县，他已经在王康的雷霆手段之下，办砸了！<br/>　　<br/>　　  而现在，就要继续下一个计划，开始阻挠骚扰，让他们无法在新奉县待下去。<br/>　　<br/>　　  “徐大人，还请跟紧我。”山匪一言，将他的思绪带回。<br/>　　<br/>　　  徐允这才是回神，这是他第一次来青崖口的老巢，之前都是隐蔽联系。<br/>　　<br/>　　  此刻，在领路人的引领下，他们已经进了山洞，洞内纵深一片漆黑，两人已经举了火把。<br/>　　<br/>　　  徐允这才看见，他们正走进一条狭长小道，都说山匪巢穴狡兔三窟，就算进来也难以真正找到，这话果然是不假。<br/>　　<br/>　　  就这样的青石小径在洞里有着很多条，弯弯绕绕，而且不是熟知，根本分辩不出差距。<br/>　　<br/>　　  难怪张将军几次清剿，都是治标不治本，打不过人家可以跑，明知道就在这一片，你还找不到人家。<br/>　　<br/>　　  这般过了半个时辰，路途逐渐宽畅，也能看到了光亮，不多时，面前的视眼也是开阔……<br/>　　<br/>　　  徐允知道，青崖口山匪真正的老巢到了！<br/>　　<br/>　　  这是一个大洞，依旧还在山中，按徐允的推算，应该是山腹之内，虽是如此，但并不感觉憋闷，应该是有通风口。<br/>　　<br/>　　  山洞很大，四周插满着火把燃烧，如是白昼，洞内有着不少人，而在最上方则是三柄刻成的石椅，石椅之上铺着野兽毛皮。<br/>　　<br/>　　  正中和下首都坐有一人！<br/>　　<br/>　　  见得徐允走进，下首那人立即起身相迎，他就是青崖口山匪的二当家，也是石奉的二哥，石勇！<br/>　　<br/>　　  此人名字虽有勇字，实际擅谋，是军师的角色。<br/>　　<br/>　　  他的面色并不好看，但见到徐允还是强挤出笑容道：“徐大人来了。”<br/>　　<br/>　　  “徐大人？”位于主坐的石坚站了起来，冷哼一声。<br/>　　<br/>　　  他的面相跟石奉相似，身高比石奉要矮不少，但却更加的粗壮结实，肌肉结扎满满的鼓起。<br/>　　<br/>　　  “被人家从县衙赶了出来，还称大人？算什么大人！”<br/>　　<br/>　　  石坚冷声道。<br/>　　<br/>　　  听了这话，徐允的脸色瞬间变得难看，然而还未等他说什么。<br/>　　<br/>　　  石坚愤怒道：“都是你让我去抢劫富阳伯爵府的车队，才使得我的三弟都是送命，如今尸首分离，连头都拿不出！”<br/>　　<br/>　　  他说着，握紧拳头直接砸在旁边的石桌上，石桌应声而裂，所有人大气都不敢喘。<br/>　　<br/>　　  听了此言，石勇的眼眶也是有些发红，他们三兄弟至从来此，兄弟齐心拼死拼活，才有这番作为，如今却是失去一位。<br/>　　<br/>　　  “跟我有什么关系？是你们大意，只带五十人就去打劫，真当那王康是吃素的吗！”徐允却是不惧，冷声道。<br/>　　<br/>　　  “王康！”<br/>　　<br/>　　  挺了此言，石坚目中尽是杀意，咬牙道：“杀人偿命，我定要将他碎尸万段！”<br/>　　<br/>　　  “我们也不必相吵，现在事情已经发生，吵也无济于事，”<br/>　　<br/>　　  石勇深吸了口气，稳定情绪又道：“现今我们有共同的敌人，还望能如之前，精诚合作！”<br/>　　<br/>　　  “我来此就是这个意思。”徐允点头道。<br/>　　<br/>　　  石坚冷哼一声，也不再多说，当时知道抢劫王康车队，他们也是激动无比，贪婪钱物，如今石奉身亡也却实怪不到徐允的身上。<br/>　　<br/>　　  “不知徐大人有何高见？”石勇请徐允坐下，而后问道。<br/>　　<br/>　　  “我有两计！”<br/>　　<br/>　　  徐允开口道：“其一发动新奉县城民制造混乱，让那王康自顾不暇，其二，直接杀上门，你们是山匪，无所顾忌其他！”<br/>　　<br/>　　  他顿了顿又道：“不过第二条需要暂缓，那王康刚驻进新奉县在此方面定有防备，贸然前往怕会吃亏，等些时日他有所放松，再行报复！”<br/>　　<br/>　　  “不行！”<br/>　　<br/>　　  石坚直接喝道：“一想到三弟头颅被砍下做成人头观，我就食宿不稳，报仇不隔夜，不然我心难安！”<br/>　　<br/>　　  “万万不可！”<br/>　　<br/>　　  徐允忙的站起，“如此着急忙慌，恐怕要吃大亏，那个王康可不是一般人啊！”<br/>　　<br/>　　  “不过是个富家少爷他又能厉害到哪？”石勇也是附和，在为三弟报仇的心思上，他与石坚的心是相同的。<br/>　　<br/>　　  “他们刚进驻了新奉县，还没彻底稳下，越早下手才越好！”<br/>　　<br/>　　  石勇又是道：“我已经探得那王康来此只带了兵卒三百，除去城防街道，能护卫他的也不过百人！”<br/>　　<br/>　　  “今晚，我会亲自带弟兄们，为我三弟报仇血恨！”石勇冷声道。<br/>　　<br/>　　  “新奉县是我们山匪说了算，想当领主那也得问问我们同意不同意！”<br/>　　<br/>　　  “此事不可，还需从长计议，”徐允忙着劝阻，王康的狠辣雷霆他已经见识过，心里更是不敢有半点小视。<br/>　　<br/>　　  虽是刚来，但相信已经做好安排，若是他们今晚就去袭击，这定然是自投罗网！<br/>　　<br/>　　  “徐大人不必多说！”石勇摆摆手道：“此事我们兄弟已经定下！”<br/>　　<br/>　　  “今晚，就在今晚！”石坚也是冷声道：“我要砍下那王康的头颅，以祭奠我三弟的在天之灵！”<br/>　　<br/>　　  这两人是被仇恨迷了心，蒙了眼啊！<br/>　　<br/>　　  看到这一幕，徐允摇摇头，他知道自己劝说已经无用……要想对付王康岂能是这么简单？<br/>　　<br/>　　  夜幕很快降临，新奉县城整个漆黑一片，各家各户都是闭门不出，因为新奉县山匪流串，晚上并不安全。<br/>　　<br/>　　  在此之际，在新奉县城街头一道道身穿黑衣的人影出现，他们都是青崖口的山匪，为首的正是二当家，石勇！<br/>　　<br/>　　  “目标县衙，我只有一个命令，把他们全部杀光！”<br/>　　<br/>　　  【作者题外话】：三更<br/><br/><br/>
