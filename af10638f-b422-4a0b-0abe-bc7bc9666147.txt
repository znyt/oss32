        &lt;h3&gt;第三卷第六十章  江俞轩的小心思&lt;/h3&gt;
<br/>
<br/>        陈子昂休假了。
<br/>
<br/>        公司由江俞轩打理，江俞轩还要去洛城看看的那个项目。
<br/>
<br/>        陈子昂一休假，他就觉得压力大了，心里感叹着，那个女人真是够奋斗的，同时也暗戳戳的鄙视自己：“哎，江俞轩，你说你一个江家大少，干嘛窝在这个小公司还乐此不彼呢？”
<br/>
<br/>        其实，他们现在的公司已经不小了，财务总监已经在着手启动上市的计划了，不过这都是一个过程，加上这次耗材研发成功，能够大量投放到市场的话，那将是一场划时代的改革。
<br/>
<br/>        信息化是指培养、发展以计算机为主的智能化工具为代表的新生产力，并使之造福于社会的历史过程。（智能化工具又称信息化的生产工具。它一般必须具备信息获取、信息传递、信息处理、信息再生、信息利用的功能。）与智能化工具相适应的生产力，称为信息化生产力。
<br/>
<br/>        信息化，以现代通信、网络、数据库技术为基础，对所研究对象各要素汇总至数据库，供特定人群生活、工作、学习、辅助决策等和人类息息相关的各种行为相结合的一种技术，使用该技术后，可以极大的提高各种行为的效率，为推动人类社会进步提供极大的技术支持。
<br/>
<br/>        网络化。数台计算机形成一个信息交互系统则可称之为“互联网”或“网络”。这种网络化技术可以使互联网成为一个相对独立的整体，并采用一种标准的计算机网络语言(技术上称为协议)使所有的计算机得以相互交流。
<br/>
<br/>        他们最开始还只是局限于网络化，简单的以硬件相支撑，加上软件，进行集成，公司研发的软件只能是针对某一个行业，在他们工作的过程中，会遇到各种各样的问题，但是随着时代的进步，各种衍生的电子产品和软件应运而生，信息化时代将会彻底到来。
<br/>
<br/>        而前两年某巴的个人网上贸易市场平台的建立更是将我国互联网推上了新的高度，拉开了我国与别的国家的差距，具有划时代的意义。
<br/>
<br/>        这一切都说明，时代在改变，社会在进步，陈子昂的公司只是在时代浪潮下的一粒沙，是被时代推着前进的，不变通就会被淘汰，陈子昂看到了机遇，也在做转型之旅。
<br/>
<br/>        目前公司研发的产品只是特定的单位，现在各种软件应运而生，大大小小的厂家全国几十家，还有一些国外的厂家在我国早已经打开了市场，竞争相当激烈。
<br/>
<br/>        而，他们目前手头上所提供的耗材也是一笔很大的收入，对于废旧硒鼓及墨盒的处理是一个问题，前一段时候专家呼吁要求从源头上推广和发展“绿色耗材“、“环保耗材“刻不容缓。这里的绿色耗材，专家指出应有三个衡量准则，即可回收、可循环和可重填，“耗材塑料件采用可回收材料，通过回收再造，这样既可节省资源，又能环保“。
<br/>
<br/>        当陈子昂提出来要研发的时候，他心里就有了一种想法，废品利用，将原装的硒鼓回收回来进行二次利用，当时遭到了陈子昂的拒绝，陈子昂说：“我们要研发一种全新的能够媲美原装品牌的替代品，而不是拿着回收回来的旧硒鼓进行灌装。”
<br/>
<br/>        江俞轩想问陈子昂知不知道重新研发需要耗费多少人力物力，但是看着陈子昂一往无前的样子他不再说话。
<br/>
<br/>        他只能带领大家一起努力。
<br/>
<br/>        且，陈子昂的想法是对的，现在耗材的用量是愈来愈多，如果有质美价廉且环保的产品推出市场肯定是会受到欢迎，而且有一定的市场份额。
<br/>
<br/>        每一种事物的发展到延伸肯定是有一个过程的，他说的三年之期被陈子昂要求一年。
<br/>
<br/>        疯子陈子昂是想将他们都逼死！
<br/>
<br/>        早上要开早会，布置一下这个礼拜的工作，他也要去洛城。
<br/>
<br/>        “今天简单的说几句，陈总休假了。”江俞轩见高层基本都到了就开门见山说，大家都知道江俞轩是公司代言人。
<br/>
<br/>        大家一言不发，原本大家都会各司其职，各部门配合得可以说是天衣无缝，要不然公司也不会发展得这么快。
<br/>
<br/>        “软件部门继续进行售后修复和研发，耗材部门责任重大，陈总的一年之期有点长，市场容不得我们偷懒耍滑，机会更不会等我们，我们都是公司的老员工现在更是公司不可分割的主体，大家任重道远。”
<br/>
<br/>        江俞轩看了一眼大家：“大家有没有要说的事情？我下午要飞洛城。”
<br/>
<br/>        “我们没什么要说的，我们要去工作了。”大家齐声说，尤其是耗材部门的人，他们更是惜时如金，怕别人耽误他们一点点的时间，现在江俞轩又将时间提前了，给他们无形之中又增加了压力，不过有压力才有动力嘛。
<br/>
<br/>        他们的研发是成功了，但是他们还要经过很多次的测试，他们要一炮打响，而不是回头重来，这个世界上不会给你很多回头的机会。
<br/>
<br/>        江俞轩其实比陈子昂更着急，他要结婚，结婚要休一个礼拜的婚假，他家老爷子已经催了他好多次，让他回去。
<br/>
<br/>        他想早点将产品推出去，在离开公司之前也对陈子昂有个交代。
<br/>
<br/>        父亲催促了他好多次，婚礼的事情他也顾不上管，都是母亲和张倩楠在张罗着。
<br/>
<br/>        想起昨天和张倩楠在一起的情景，他心里都有一点点内疚。
<br/>
<br/>        “俞轩，我们都要结婚了。”
<br/>
<br/>        “是啊，我们要结婚了啊。”江俞轩搂着张倩楠的肩膀。
<br/>
<br/>        “可是，你都没有好好和我谈过恋爱。”
<br/>
<br/>        “难道我们现在不是在谈恋爱吗？”
<br/>
<br/>        “你.......”张倩楠撅起了她那粉嘟嘟的小嘴巴。
<br/>
<br/>        “我们都认识多久啦？老夫老妻的了吧？再说，最近公司要研发一种新产品，忙得不行，我等过一段时间好好陪陪你。”其实他们真的好久没有见面了，在同一个城市。
<br/>
<br/>        “你都卖给你们公司了，你给你们公司当老公吧！”
<br/>
<br/>        “公司人多的很，男男女女都有，你要让我当大众，情人？老婆，你这心忒狠了点吧，我可是你将来的私人用品。”
<br/>
<br/>        “贫嘴！”张倩楠笑了。
<br/>
<br/>        “不过最近我真的很忙，明天要出差洛城，来来回回怕是要得三五天。”
<br/>
<br/>        “你去吧，注意安全。对了，陈子昂最近好吗？”
<br/>
<br/>        “她？能不好吗？能吃能睡，一天就像是没心没肺的样子。”
<br/>
<br/>        “江俞轩，你很喜欢她吧？”
<br/>
<br/>        “没有，倩楠你想啥呢？那可是你的嫂子。”
<br/>
<br/>        “我函哥也真是可怜。江俞轩，我很庆幸遇见你。”张倩楠深情的看着江俞轩。
<br/>
<br/>        “倩楠，我也很庆幸遇见你，你知书达理，贤淑聪慧，尤其是，心里有我。我们这一生一定要好好的。”
<br/>
<br/>        “嗯，我要一直和你在一起，不受相思苦，不受分离苦。”张倩楠温柔而深情的说，张函和陈子昂的事情给了他们惊醒，他们需要珍惜彼此。
<br/>
<br/>        “我会一直给你洗发描眉，相依一生。”
<br/>
<br/>        期间有一段时间陪着陈子昂到处飞，张倩楠吃了飞醋，不得已，他搬出了张函，自从听了张函和陈子昂的故事后，张倩楠再也不酸溜溜的说话了，而且还很支持他的工作。
<br/>
<br/>        有时候还关心的问陈子昂的近况。
<br/>
<br/>        人这一生一定要懂得取舍，失去的未必是适合你的，得到了一定要珍惜！
<br/>
<br/>        他看着陈子昂，陈子昂就像是一面镜子，让他不会纠结在无望的感情里，会静下心来去找适合自己的爱人，好的是，他找到了！
<br/>
<br/>        他和陈子昂注定是朋友，相互欣赏，相互扶持的朋友。
