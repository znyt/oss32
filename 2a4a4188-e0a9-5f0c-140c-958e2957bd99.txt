　　燧人氏三人想了想，觉得玄天说的不错，脸色不由变得凝重起来，道：
<br/>　　“大哥，你说的不错，我人族现在还是很弱小，我们身边还有许多危险存在，人族迫切需要提升实力。”
<br/>　　“对，大哥，之前是我等眼光狭隘了，就按你说的办吧。”
<br/>　　玄天嘴角露出一丝让人难以察觉的微笑，他说这么多，就是要让燧人氏三人意识到问题的严重性，虽然他们之前也十分努力，但他们终究是限制于自身的眼光，不能将目光看的更远，形势分析也不够到位。不过，这也不怪他们，毕竟他们好多事情都是听来的，没有经历过战火与灾难，眼光毕竟是有限的。
<br/>　　“放心，我也不是让所有人族都出去洪荒历练，我只是让金丹武宗境的武者们出去外面历练。武者到了金丹境，已经堪比洪荒中的玄仙了，他们已经有了一定的自保能力，不至于那么不堪。”
<br/>　　“更何况，武者到了金丹境，以后想要提升不是那么容易的，他们也需要历练一番，增加自己的见识或者感悟，这对于他们以后突破神通境以及之后的境界会有莫大的好处。毕竟，生死之间，不仅有大恐怖，也会便随着大机缘。”
<br/>　　“这些武者出去后，不仅能够锻炼自己的战力，增强自己的实力，还能顺便见识一下洪荒的地貌，为以后人族迁居而做准备。”
<br/>　　玄天自己为他们仔细分析道。
<br/>　　“大哥说的不错，与其以后被别的种族杀害，不如现在多流些血。我们赞同大哥的意见。”
<br/>　　燧人氏大声说道，同时他也觉得，自己等人的目光，还是不够长远啊。看来，自己等人也是需要历练的啊。
<br/>　　“让金丹期以上的武者出去历练，那样最好不过了。”有巢氏也赞同道。
<br/>　　“我听三位哥哥的。”缁衣氏说道。
<br/>　　“好，既然如此，那便回去准备吧，我们还需要规定一个期限，让他们不论如何，都要在这个期限内回来，不然，若是人族遇到了危险，这些人又回不来，那又该如何？”玄天再次说道。
<br/>　　“不错，是该有个期限，大哥觉得多少为好？”
<br/>　　“我认为五百年就足够了，五百年内，人族暂时没有太多危险，但是五百年后，可就不好说了。”先天思索了一会儿，说道。
<br/>　　其他三人也微微一下想，觉得可以，便同意说道：
<br/>　　“好，就五百年。”
<br/>　　“好，既然如此，你们就准备出去宣布吧。另外，我也需要出去走走，为我们人族日后做准备，人族这里，暂时就靠你们了。”玄天突然间好似想到了什么，这样说道。
<br/>　　“什么？大哥，你也要出去？”燧人氏三人不由得大惊道。
<br/>　　“是啊，人族以后想要发展，就必须先了解一下洪荒的形式，我也需要出去洪荒走走，顺便历练一番。”玄天平静道，这是他早就有的想法。
<br/>　　“要不，大哥，你别走。若是你发生危险了要怎么办？”缁衣氏有些担心道。
<br/>　　“不必担心，我如今的实力已经堪比洪荒准圣了，只要我小心一些，洪荒中能威胁我的人不多。”玄天对着三人说道。
<br/>　　“那……好吧，大哥，你小心点。”三人想了一下，知道劝不了，便同意了，不过还是让他小心一些。
<br/>　　“放心，我会的。”
<br/>　　玄天感受到一阵温暖，轻声道。同时，他心中暗道，人族，我会守护好的，毕竟，我的前世，也是人族啊。
<br/>　　之后，人族三祖便出去，向人族众人宣布了这个决定，虽然其中也有人不同意，但大多数人族还是识大体的，他们也意识到了人族的危险，同意出去历练。
<br/>　　就这样，陆续有人族金丹或者金丹以上的强者离开不周山，出去洪荒中历练。
<br/>　　一个月后，玄天一样默默离开人族，向着洪荒中走去，谁也不知道他去了哪里。
<br/>　　……
<br/>　　昆仑山，玉虚宫中。
<br/>　　元始坐在一旁，手持一枚黑色棋子，看着眼前的棋盘，棋盘中沟壑纵横，演化出当今洪荒各个势力的局面，其中，巫、妖两族，一个掌天，一个霸地，是最为强势以及最为强大的两方势力，当然，也有一些别的势力，如东王公所创建的紫府，西方灵山、镇元子万寿山、冥河血海等，就连人族都在其中。人族在其中虽然算不上最大的一方势力，但是也不弱小，而且形式也颇有些微妙。
<br/>　　在元始的对面，是老子，老子手持一枚白子，向着棋盘上落下，却正好截断了紫府的路，而元始的黑子却是带领着人族在众多势力夹缝之中生存，最后，他们将当今形势通过一盘棋来演化，妖族灭了紫府，攻向万寿山，巫族进攻妖族，最后巫妖大战，两败俱伤，人族崛起。
<br/>　　“大兄，你怎么看？”老子看着棋盘上大战的巫、妖两族，向元始问道。
<br/>　　元始微微一笑，却是不语，一子落下，却是巫族一个祖巫隐没，巫族生机不绝，之后，又一方实力突然冲出，来历不明。
<br/>　　“嗯？”
<br/>　　老子瞳孔一缩，有些不明白，这方势力又是何人，那里正位于西方附近，按理说西方已经没有其他势力了，这群人又是从何处冲出来的。
<br/>　　元始却是微笑不语，因为他知道那方势力是谁，他们位于西方太华山，正是他的恶尸万劫魔尊所率领的劫魔殿。
<br/>　　“你们决定了。”突然，元始开口问道。
<br/>　　“是啊，大兄，女娲师妹都已经成圣好久了，我等的时机也快到了。”老子淡漠道。
<br/>　　“可有把握？”元始问道。
<br/>　　“通过上次天劫大道法则显化，我对自己的法则参悟也深刻了许多，有一定的把握，或许可以了。”
<br/>　　“或许可以？”元始微微皱眉，“这是要冒险啊！”
<br/>　　“人生本就是许多冒险，如若不然，岂非很无趣？”老子毫不在意，微微一笑道。
<br/>　　“是啊，人生本就很无趣。也罢，既然你们决定了，那我在助你们一臂之力。”元始淡淡道。
<br/>　　“有大兄的帮助，自然是把握更大了。”老子道。
<br/>　　“既然如此，我再为你们讲解一番成为最后这一步的关键。”元始淡淡道，同时也将闭关中的通天叫了过来。
<br/>　　“大兄！”不久，通天便来到玉虚宫中，向着元始行礼道。
<br/>　　“嗯。”
<br/>　　元始微微颔首，看着通天道：“不错，三弟看来是此次闭关进步很大啊，一身剑道已经凝练到了极致，万法唯剑。很好，看来距离成圣也不远了。”
<br/>　　“大兄过誉了。”通天见此，傲然一笑道。
<br/>　　“吾一般可不会夸人啊。”元始淡淡说了一句，接着问道：“决定好了？”
<br/>　　“嗯。”通天脸色凝重，点头道。
<br/>　　“好，既然如此，那吾便助你们一臂之力。”元始点头，好似早有预料一般。
<br/>　　“多谢大兄！”
<br/>　　“无妨！你我兄弟，何须多谢？”
