　　时光匆匆，转瞬间百年过去了。
<br/>　　自从上次劫魔殿参与三族大战，便让洪荒众人见识到了劫魔殿的厉害之处，劫魔那种完全不畏死，不惧死，杀不死的怪物，使得劫魔殿一跃成为洪荒之中的顶级势力。这些年，劫魔殿一直在以太华山为中心，不断向外扩张。
<br/>　　劫魔殿向外张开了他那狰狞的獠牙，开阔疆土，劫杀群雄，不到百年时间，便将太华山方圆数十亿万里的地区占据了，完全算是一个盛大的魔朝。
<br/>　　劫魔殿展现出来的实力与手段也极为惊人，准圣高手便有数位之多，大罗高手更是有上万。更何况，在上面还有一位最为神秘的万劫魔尊。他们的手段也极为残忍，嗜杀成性，所过之处，生命不留，全部转化为劫魔。
<br/>　　在洪荒之中，他们一时之间成为人人喊打的大魔头。不过，因为被劫魔殿所展现出来的实力，慑服了众多势力，最重要的是，劫魔殿完全没有威胁到他们的利益，所以，却是没有多少人敢来招惹他们。
<br/>　　但是，劫魔殿的这番作为，却是使得西方灵山西方教极度不满。西方是西方教的大本营，他们一直将西方当做是他们的地盘，如今却有人在他们的地盘之上撒野，他们能不生气才怪。
<br/>　　攘外必须安内，所以，西方两位圣人，召集门下弟子，打着除魔卫道的口号，准备前来攻打劫魔殿，将这个西方大地之上的魔道巨擘毁灭。
<br/>　　当然，这只是两位圣人的说辞，真正的原因，恐怕除了两位圣人之外，也只有万劫魔尊自己知道了。
<br/>　　万劫魔尊的传承来自远古时期的魔祖，而西方教的神通，也有一些魔祖的痕迹在内，要说两者之间没有关系，谁也不会相信。
<br/>　　如今西方教还没有发展起来，门下弟子也不多，修为高，资质好的弟子，更是不多了，所以，攻打劫魔殿的主力，还是在两位圣人手上。
<br/>　　这天，接引与准提两位圣人带着门下弟子来到劫魔殿大本营太华山，劫魔殿的人好似早知如此，全部退回到太华山中。
<br/>　　“终于有人要对劫魔殿出手了，还是圣人。嘿嘿！”
<br/>　　圣人一动，瞬间就吸引了天地间无数强者的目光，人们将目光投向西方大地，顿时就知道了圣人要对劫魔殿动手了。
<br/>　　“早该如此了，劫魔殿凶狠毒辣，滥杀无辜，早就该出去了。”有人如此说道。
<br/>　　“劫魔殿要倒霉了，谁让他们如此高调的，嘿嘿！”有人幸灾乐祸道。
<br/>　　“是啊，圣人出手，劫魔殿这次可是在劫难逃了。”
<br/>　　“又一个顶级势力要被灭除了，洪荒真是多纷争啊！”
<br/>　　“谁能永远屹立不倒，如同巫妖二族那般强大的实力，都被近乎灭族了，劫魔殿又算得了什么？”
<br/>　　“再强的势力，再圣人眼中，都是蝼蚁，覆手可灭。”
<br/>　　“……”
<br/>　　众多洪荒大神通者们，神念横空，相互交织，议论纷纷。对于圣人出手灭一个势力，那是毫无悬念的事情，每个人都觉得，劫魔殿被灭，那是在所应当的事了，要怪就只能怪，劫魔殿没有圣人强者坐镇。
<br/>　　甚至，就连天地间其他圣人也都在关注着这一战，他们也想看看，到底是何种原因，能够让圣人拉下脸皮来出手。
<br/>　　“尔等魔头，嗜杀成性，只知杀戮，不修善恶，造成无量因果。今日贫道便要替天行道，除了尔等这些魔头。哼！”
<br/>　　只听准提圣人一声大喝，紧接着，一道巨大的金掌，遮天蔽日般从天而降，拍向劫魔殿，要将里面的劫魔与宫殿一起毁了。
<br/>　　“哼！我劫魔殿的地盘，什么时候轮到你西方教插手了。给我滚回去！”
<br/>　　就在此时，只听从劫魔殿深处传出一阵嚣张的声音，紧接着，一道漆黑的光柱，从劫魔殿深处传出，冲散了准提圣人的遮天金掌。
<br/>　　“嘶！”
<br/>　　天地间各路强者顿时纷纷倒吸一口气，圣人的一掌竟然被灭了，竟然被灭了？要知道，那可是圣人的一掌啊，竟然如此轻易的便被灭了，说出去谁相信啊，可事实就是如此，由不得他们不相信。
<br/>　　“哼！”
<br/>　　准提圣人的面色有些不好看，自己的一击竟然被人轻易破除了，饶是以他圣人的脸皮，也有些吃不住。
<br/>　　他再次凝聚出惊天一掌，这是他却是动了真力，使出圣人的威严，拍出这一掌。之前他以为对付一个还没有成圣的人物，不许要自己动真力，如今看来，自己还是小觑了他们。
<br/>　　“滚！”
<br/>　　眼看那巨掌就要再次拍下来，众人又听到一阵大喝，紧接着，一道巨大的黑色刀芒，划破虚空，将准提的金掌划成两半，散落开来。
<br/>　　众人：“……”
<br/>　　你怎么这么吊？你妈知道吗？
<br/>　　紧接着，在众人惊骇的眼神中，虚空出现一道身穿黑色道袍的身影，他黑发乱舞，宛若魔神，眼神妖红诡异，闪烁着魔性的力量，他的全身上下更是有一种破坏的魔力，能够影响人的心神，看了他一样，就仿佛会产生幻觉，陷入杀戮之中。
<br/>　　他就仿佛是毁灭之源。
<br/>　　“你是谁？”
<br/>　　准提看着突然出现在他面前的神秘人，有些凝重的问道，因为他能够从此人身上感受到一种超越准圣的力量，也就是说，这人不是准圣，但不是准圣，那又会是什么呢？难道是圣人？这怎么可能？天地间怎么可能还有其他圣人？
<br/>　　“你都攻打到本尊的地盘上了，你才来问本尊是谁？真是笑话！”黑衣人冷笑道。
<br/>　　“你是劫魔殿之主？”
<br/>　　“劫魔殿之主？也可以这么说。但本尊更喜欢别人叫我万劫魔尊！”黑衣人冷冷道。
<br/>　　“万劫魔尊？没听说过。”
<br/>　　“那是你见识浅薄。万劫不灭邪圣仙，万道森罗魔至尊。哈哈！”万劫魔尊嚣张大笑道。
<br/>　　被别人鄙视了。洪荒众人脸色不由得发黑，准提圣人更是脸色难看之极，阴沉的看着眼前这个自称是万劫魔尊的人物，缓缓道：
<br/>　　“劫魔殿乱杀无辜，行事霸道，造成无量恶果，乃魔头的行为，阁下还请跟随我回去念经思过，早日洗脱罪孽。”
<br/>　　“笑话，我本是魔中至尊，不杀人，那本尊要干什么？真是好笑！”万劫魔尊面对圣人毫不畏惧，冷笑道。
<br/>　　“看来阁下是执迷不悟了，那就休怪贫道手下不留情。”准提面色不变，紧紧盯着万劫魔尊，圣人的心境，岂是那么容易破的。
<br/>　　“要动手便动手，说得那么好听，真是虚伪，本尊都替你恶心了。”万劫魔尊毫不犹豫的便反击道。
<br/>　　洪荒中人：“……”
<br/>　　“哈哈哈！说得好，我最看不起的就是这样的人了，正是痛快！”昆仑山，通天教主哈哈大笑，觉得万劫魔尊颇有意思，很服他胃口。
<br/>　　他的弟子们：“……”
<br/>　　师傅，你真的是这样想的吗？
<br/>　　“轰！”
<br/>　　这次准提明显是动怒了，他却是不再说话了，而是直接动手了。
