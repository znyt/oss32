　　各堂长老闻言，皆是震惊不已，乌蒙灵谷虽然比不得天墉城，却也不可小觑，现在居然被灭族了。
<br/>　　长老们心里都有些不安，议论纷纷，开始猜测究竟是何方势力灭了乌蒙灵谷。
<br/>　　慕清霄则站在一旁，沉默不语，整个天墉城，除了他自己，谁也不知道答案。
<br/>　　若非他知晓来龙去脉，否则也不可能知晓，策划整件事情的幕后之人，本是天界的长琴太子。
<br/>　　“行了，执剑堂重地，可不是让你们互相争议的地方。”
<br/>　　涵素真人有些不耐烦的挥了挥手，示意他们安静下来，现在紫胤真人有伤在身，封印焚寂恐怕是不能出手。
<br/>　　“执剑长老，封印焚寂之事，便交给本座与各位长老，你伤势颇为严重，尽快去闭关疗伤吧。”
<br/>　　“如此也好。”
<br/>　　紫胤真人总算是放下心来，焚寂煞气虽恐怖，但天墉城除了剑术强悍，其次就是封印术了。
<br/>　　涵素真人与各堂长老的实力都不低，联手封印焚寂并非难事。
<br/>　　此刻，他被焚寂煞气所侵蚀，伤势颇为严重，若非他实力强大，能够压制住体内煞气，后果不堪设想。
<br/>　　饶是如此，也让他有些吃不消，必须尽快闭关，驱除体内的焚寂煞气，以免伤势恶化。
<br/>　　将事情商量好，紫胤真人便向剑塔深处走去。
<br/>　　“锵！”
<br/>　　待紫胤真人离去后，涵素真人复杂的看了一眼焚寂，调动灵气，将煞气肆意的焚寂拔起，准备开始封印。
<br/>　　其实他心里清楚，乌蒙灵谷之所以被灭，恐怕与焚寂脱不了干系。
<br/>　　现在，紫胤真人将此剑带回天墉城，简直就是给天墉城带回来一个大麻烦，往后的日子，绝对不会安宁。
<br/>　　当然，他作为天墉城的掌门，自然不怕宵小之辈，也是有剑仙紫胤真人坐镇，他才同意将此剑封印在天墉城。
<br/>　　“紫胤真人有伤在身，不宜出手，由本座与各位一齐封印此剑，各位长老没意见吧？”
<br/>　　慕清霄自然是没意见，涵究真人也没意见，至于还虚真人与凝虚真人则犹豫了一下，也满口答应。
<br/>　　毕竟，掌门与执剑长老决定好的事情，他们有资格反驳，但反驳有没有效果就不知道了。
<br/>　　到时候将两人都给得罪了，反倒是得不偿失。
<br/>　　...
<br/>　　...
<br/>　　剑塔中央，焚寂剑悬浮在焚炼炉中。
<br/>　　慕清霄，涵素真人，涵究真人，凝虚真人，还虚真人分别站在五个方向，形成阴阳五行之阵。
<br/>　　只见包括慕清霄在内，五人盘坐在四周，将焚寂包围在中央，双手快速结印，一道道封印法决形成。
<br/>　　金木水火土，阴阳五行封印，天墉城最高级的一类封印，需要五位属性强大的修士，分别施法。
<br/>　　拥有九霄琉璃焱的慕清霄自然就是火属性，至于其他四人则是金木水土四种属性。
<br/>　　“嗡嗡——”
<br/>　　在封印形成的瞬间，焚寂剑似是不甘被封印，赤红的剑身剧烈颤抖起来。
<br/>　　煞气冲天，尖锐的剑鸣响彻整个剑塔，似乎想要全力冲破封印。
<br/>　　“金木水火土，封。”
<br/>　　随着双手结印，五种颜色的封印形成，最后在中央融合成黑白两色，焚寂剑为阵眼，犹如八卦。
<br/>　　就算焚寂剑在强，现在都没有宿体，根本无法发挥出恐怖的力量。
<br/>　　慕清霄与各堂长老想要封印，也并非难事，除了涵素真人，其余长老都是练虚期强者，而且都是高阶练虚。
<br/>　　练虚期强者，放在修真界，基本都已经能担任一些二流门派的掌门，实属不弱。
<br/>　　随着五人源源不断的将灵气输入阵眼，阴阳五行，封印成！
<br/>　　虽然各堂长老都消耗了不少灵气，焚寂剑也总算封印成功。
<br/>　　在将焚寂封印成功后，除了慕清霄与涵素真人，其余长老脸色都微白，额头上都浮现一层细汗。
<br/>　　显然，此举对他们的消耗，也颇为严重，恐怕没有两三天的时间，是没法完全恢复了。
<br/>　　而涵素真人也是因为修为高深，自然也就轻松一些。
<br/>　　至于慕清霄，完全跟没事人一样，封印焚寂消耗的灵气，对他来说，完全就是九牛一毛，根本不算事。
<br/>　　因为心法的缘故，他体内的阴阳灵气，压缩在压缩，自形成阴阳湖泊后，浓郁的灵气就源源不断。
<br/>　　输出的时候，也在吸取天地间的灵气。
<br/>　　站起身来，涵素真人见慕清霄与先前没有一丝变化，眼中满是赞赏，其余长老看向他的眼神，就像见了鬼似得。
<br/>　　“各位，都没事吧？”涵素真人客套的看了各堂长老一眼。
<br/>　　“多谢掌门真人体恤，我等无事，只是将此剑留在天墉城，恐怕是祸非福啊。”
<br/>　　各长老哪一个不是人老成精，自然明白其中的道理，乌蒙灵谷被灭，焚寂剑出现在天墉城，两者没有联系打死他们都不信。
<br/>　　闻言，涵素真人道：“既然执剑长老将此剑带回，自然是与天墉城分不开关系，事已至此，也只能这样了。”
<br/>　　“可是......”
<br/>　　“行了，此事休要再提，你们都去吧。”
<br/>　　待各堂长老离去，涵素真人看向慕清霄，道：“凝丹长老，你对此事有何看法？”
<br/>　　慕清霄微微一愣，没想到涵素真人会问他这种问题，思索片刻，便开口道：“船到桥头自然直。”
<br/>　　“凝丹长老倒是洒脱。”
<br/>　　涵素真人微微一笑，真的很难想象，面前的青年只有二十多岁，旋即目光看向剑塔深处，欲言又止。
<br/>　　慕清霄道：“掌门真人不必多虑，那孩子是执剑长老带回来的，相信执剑长老自会将此事负责到底。”
<br/>　　“哎，也罢......”
<br/>　　深深的叹了口气，涵素真人便转身离去。
<br/>　　慕清霄则站在剑塔中，望着眼前焚寂，最后瞥了眼剑塔深处，也是转身离去。
<br/>　　焚寂与韩云溪都已经出现在天墉城，剧情开始以正常的轨迹发展，他无需多虑，只需要慢慢等待。
<br/>　　等待一个最佳的时机，相信，再过几年的时间，长琴太子也会来天墉城吧。
<br/>　　离开剑塔后，慕清霄就前往凝丹堂，取了一些灵药后，便返回自己的宫殿，开始闭关修行。
