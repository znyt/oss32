　　李承乾和唐善识两个小年轻怀着忐忑的心，从船长完出来，在一路问候声中下到了甲板上，又在一群老货的拍拍打打下来到了李二的身边。
<br/>　　此时的两人已经被拍的浑身发麻，险些站立不稳。
<br/>　　“太子、善识，这铁甲舰首航成功，你们两个居功至伟，来来来，把这酒喝了，算是朕给你们庆功！”
<br/>　　‘哐、哐’两杯酒被老头子摆到了桌上，上面满满一层的煤灰。
<br/>　　李承乾看着那杯子几乎都傻了，这东西特么能还能喝么？老头子这是闹哪样？
<br/>　　唐善识也搞不清皇帝陛下是要闹哪样，不过他和李承乾不一样，老李让他喝那就只能喝。
<br/>　　于是小唐尴尬的端起了杯子，苦笑的仰头将杯中酒倒里了嘴里。
<br/>　　好在只是一些煤灰，并不是什么毒药，喝了也就喝了，平时在炼钢坊‘喝’的并不比这杯子里的少。
<br/>　　不过，随着小唐这一仰头，却发现了其中的门道。
<br/>　　只见头顶的天空上，无数煤灰正顺着两上大大的烟囱争先恐后的向外涌着，而那些脱离的烟囱束缚的煤灰正自空中滚滚而下。
<br/>　　我了个去的，难怪皇帝会给他们两个喝这东西，敢情……。
<br/>　　小唐已经不敢再想下去了，消消捅了李承乾一下，给他递了个眼神，示意他快点。
<br/>　　李承乾不知道小唐在搞什么鬼，但也知道这酒不喝不行，于是便也喝了下去。
<br/>　　……
<br/>　　一群老头子们被安排到了船舱里面去了，李承乾和小唐两个无奈的蹲在甲板上，尴尬的笑着。
<br/>　　他们两个几乎算到了这次首航能够发生的一切意外，唯独没有想到那两个破烟囱会给惹来这么大的麻烦。
<br/>　　“高明，这可咋整？陛下会不会真的生气了？”隔了半晌，小唐忧心忡忡的问道。
<br/>　　“不行，老头子从来不记仇，刚刚不是罚过咱俩了么，这事儿就算是过去了。”李承乾耸肩说道。
<br/>　　而在他心里却在想，过去个屁，下次指不定老头子啥时候想起来，还得旧事重提。
<br/>　　再说了，老头子罚过了，老娘那边还没动静呢，估计晚上回去之后自己也不可能有什么好果子吃。
<br/>　　想到这里，李承乾就觉得有些郁闷，早知道就不让老头子和老娘一起过来了。
<br/>　　不过话虽然如此，但唐善识还是相信了，如释重负的笑了笑，然后对李承乾问道：“对了，高明，这船还接着弄不？还是就搞这一艘就算了？”
<br/>　　“必须接着弄，不过要再等一段时间。”李承乾想了想说道。
<br/>　　唐善识若有所思的看了看李承乾，啧了一声说道：“高明，你还指望着那个什么电焊机呢？我跟你讲，这个太不靠谱了，那东西时好时坏根本就稳定，焊出来的东西根本就不结实。”
<br/>　　“那就想办法让它稳定一些，办法总是人想出来的不是！”李承乾横了小唐一眼：“再说那东西有多重要你也不是不知道，只要有了那个东西，很多以前没办法弄的东西，以后我们都可以弄出来。”
<br/>　　“我也知道那东西重要，可是现在除了你多少懂点这个，咱整个大唐还有谁能整明白这东西，你说的那个什么电容器，什么稳压，连我都听不懂，你觉得这事儿靠谱么？”
<br/>　　唐善识现在是真拿李承乾这家伙没招了，一天到脑也不知道他脑子里在想些什么，总是有些奇奇怪怪的东西被写出来送到研究署。
<br/>　　可是这话说回来了，谁也不可能一口吃个胖子吧，没有任何一点理论基础就想搞新发明这不是扯蛋么。
<br/>　　想想以前的大唐，再看看这十来年的大唐，很多东西的确在发生着翻天覆地的变化，可那些都是有原始积累的，很多属于只隔着一层纸，点破了大家也就明白了，自然可以在最短的时间里搞出来。
<br/>　　可是电这个东西完全就是新生事物，以前从来没在大唐出现过，甚至经过了半年时间，长安还有一些人连听都没有听说过‘电’。
<br/>　　在这样的条件下，要搞研究这就是在拿钱在打水漂，不，不光是打水漂的问题，而是跟轱辘要孩子的问题，不是研究署不想去弄，而是根本就不知道从何处下手，不知道从哪里搞起。
<br/>　　李承乾听着小唐的吐槽，半晌没有说话。
<br/>　　他当然知道是自己有些心急，可是……，好吧，还真没什么可是的，大唐能有现在这个样子，他李承乾应该感到十分满足才是。
<br/>　　因为从某些方面来讲，此时大唐的科技水平几乎能用飞跃来形容，像铁甲舰这种钢铁巨兽就算是在十八世纪的辫子皇朝都不见得能独自建造，可它现在却出现在大唐的境内，正咆哮着奔向远方。
<br/>　　想到这里，李承乾叹了口气，一屁股坐到甲板上对小唐说道：“好吧，是我有些心急了，你不要生气。”
<br/>　　”不，我没有生气，我只是觉得你把自己逼的太紧了。”小唐摇摇头，同样坐到甲板上，看着河岸两侧随行的禁军护卫有些感概的说道：“人的精力总是有限的，你不可能把所有的事情在一生之中全都做完，当然，你也不可能做完。”
<br/>　　“我明白，有些时候我也很清楚这其中的道理，可是那种明明看到目标，却怎么也接近不了的感觉实在太折磨人了，总是会让你不自觉的敦促自己快一点，再快一点！”李承乾呵呵笑着，抬头看着飘满煤灰的天空。
<br/>　　每一个宅男心中都有一个梦想，这个梦想会随着时间的改变而改变，但却不会消失。
<br/>　　曾经李承乾刚到大唐的时候只想好好活着，顺便摆脱单身狗的命运。
<br/>　　可随着时间一点点发展，他慢慢发现自己有改变这个世界的‘能力’，凭借着前一世的记忆，他一点一点改变着身边的一切，改变着大唐的一切。
<br/>　　现在，他只想在有生之年把大唐变成自己所熟悉的那个世界，虽然这根本不可能实现，但人总要有些理想，不是么？如果没有理想和咸鱼又有什么区别？
