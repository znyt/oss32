　　第480章
<br/>　　其实李行哉并不能读懂这场胜利之后隐藏着什么阴谋诡计。但他流浪这么多年，见过许多人，目睹过许多事，看来看去，就认定了一个道理。
<br/>　　这世间没有任何好事可以轻而易举获得，有的话，那也只可能是坏事。
<br/>　　诸多帝国军中的大人物，聚在城主府的大厅，正享受着这场战争带来的喜悦。
<br/>　　“那程大雷也不过是一介莽夫而已，真知道兵者，乃诡道也。战场之上，拼得更多的是计谋，而不是蛮力。”尉迟离捋着胡须，两只眼睛眯成一条线。
<br/>　　“如今我们攻占黄沙城，已经占了程大雷的大本营，他这几日便成了无家之犬，只能在草原上逃窜。”崔相道。
<br/>　　“再加把力，便能将贼人一网打尽，替帝国除了这场大患。”
<br/>　　诸人你一言我一语，互相吹捧。事实上，在一开始，程大雷都算不上帝国的心腹大患。真正的心腹大患，则是在场就坐的每个人。李行哉并没有参与他们的讨论，觉的有些索然无味，独自离开了大厅。众人议论纷纷，倒是也没人察觉到李行哉的离去。
<br/>　　而外间的街道上，士兵你来我往，也正是热闹。他们正忙着划分每路兵的位置，清点仓库中程大雷留下的物资。
<br/>　　“六殿下……”
<br/>　　李行哉皇子的身份还是有用的，无论哪路诸侯的兵，都要给他几分面子。见到他独自一人走在街道上，停住脚步后纷纷行礼。
<br/>　　“喔，你们在忙什么？”
<br/>　　“启禀殿下，我们正将仓库中的物资运出。”
<br/>　　“走，带我去看看。”李行哉突然有些好奇，想要看看程大雷究竟留下多少东西。
<br/>　　城主府的大厅内，诸人正在议事，还是互相吹捧的话居多。
<br/>　　“报……”一名小校闯进大厅：“启禀诸位大人，蛤蟆军在城外击溃了扬州李星大人的队伍。”
<br/>　　诸人都沉默下来，诸人暗地里搞鬼，让李星面对程大雷的主力部队，如此，相当于将李星卖了。
<br/>　　良久的沉默后，尉迟离叹了口气：“原也该想到，程大雷竟然如此狡猾。”
<br/>　　崔相顿了顿，接着话茬道：“都说扬州兵的战斗力弱，却也是没想到，竟然弱到这种地步，五万打一万，还被打了个落花流水。”
<br/>　　“是啊，是啊。”
<br/>　　诸人说着话，都笑了起来，大厅内的气氛又活络起来。
<br/>　　“报……”又是一名传讯的小校：“派出去追击溃兵的队伍，中途遭遇程大雷的大股部队，全员两千人，全军覆没。”
<br/>　　“呃……”尉迟离愣了愣：“敌人杀威竟然这么强？”
<br/>　　“尉迟将军莫要吃惊，程大雷知道黄沙城被占，定然恼羞成怒，击溃我们的追击部队也是正常。但依我看，帝国儿郎也是有血勇的，他们吞下这么大队伍，定然也是死伤惨重。”
<br/>　　“是啊，是啊。”
<br/>　　众人听罢，都赶忙附和崔相的话，说些程大雷是秋后的蚂蚱，蹦跶不了多久了之类的话。
<br/>　　“报……”
<br/>　　众人听到这声音，下意识都是一激灵，不知道这次，又有什么糟糕的消息传来。
<br/>　　“又发生什么事了？”
<br/>　　“程大雷率领大股部队向黄沙城赶来，如今距离黄沙城还有二十里。”
<br/>　　“好！”尉迟离拍案而起：“他来得好，我方以逸待劳，定要将其全歼在城下。”
<br/>　　轰！
<br/>　　此刻众人耳边忽然听到一声巨响，感觉天摇地晃一般，从头顶扑索索往下落粉尘。大家心慌意乱，好不容易才算是镇定下来。
<br/>　　“发生什么事了？”尉迟离怒道。
<br/>　　立刻有人出去，查明刚才的原委，不多时，有人急匆匆跑回来，着急道：“启禀诸位大人，仓库发生爆炸，所有物资都被焚毁了。还有……”
<br/>　　“还有什么？”
<br/>　　“六殿下也在火场之中，现在身受重伤。”
<br/>　　“什么！”崔相心中一惊，立刻从椅子上站了起来。
<br/>　　“是真的么？伤得严不严重？”
<br/>　　尉迟离口中问着，心中却颇有几分欢天喜地。假若李行哉在爆炸现场死掉了，李乐天可就少了一个有力的竞争对手。
<br/>　　众人急匆匆赶往事发的仓库，见房屋倒塌了一片，火场内的余烬还未熄灭，正有士兵忙碌着灭火。
<br/>　　有人向尉迟离等人汇报刚才发生的事情。负责清点物资的士兵进入仓库后，外面不知里面的人遭遇了什么，就听得轰然一声响，也就是眼前这种状况了。
<br/>　　士兵死伤惨重，横七竖八的尸体摆在一起，死相惨不忍睹。
<br/>　　当然，这些不是崔相关心的，他最关心的还是李行哉的情况。一向老态龙钟的崔相，展现了从未有过的矫健，口中喊着：“六殿下在哪里，六殿下在哪里？”
<br/>　　这同样也是尉迟离所关心的，他关心李行哉究竟死了没有。
<br/>　　“丞相，丞相……”
<br/>　　有微弱的声音响起，士兵抬着担架，将李行哉抬到诸人面前。以李行哉的身份，自然不需要亲自进入仓库，所以事发时他离现场颇远。
<br/>　　但他还是被一根飞起的石块砸到腿，左腿小腿被砸断，大夫给他做过简单的固定。
<br/>　　看到这一幕，崔相略略松口气，口中不停道：“殿下无事就好，殿下无事就好……”
<br/>　　尉迟离感觉有些失望，也只是道：“无事就好，无事就好。”
<br/>　　心里却想，怎么可以没事呢，为什么石头不落得准些，直接砸破他的脑袋。不过砸断腿也好啦，就是不知道还能不能接上，反正没听说过，有那个瘸子当皇帝的。
<br/>　　“殿下莫要担心，现在安心养病要紧。”崔相。
<br/>　　李行哉躺在担架上挥挥手，打断了崔相的话，道：“丞相，速速退兵吧……”
<br/>　　崔相一怔，道：“殿下何出此言，我们刚刚占领了此地……”
<br/>　　“我已经问过，城中所有的水井都已被堵死，仓库爆炸……这都是程大雷之前安排好的。”李行哉抓紧崔相的手，道：“你们难道还没发现，是程大雷故意让我们占领这里的。黄沙城本就是为我们准备的牢笼，或者……坟墓。”
<br/>(本章完)
