　　如意君的实力，虽然不如颜古龙帝，却也是曾经叱咤九域的强者。
<br/>　　夜天君本身便就是天资极高，先天神体，如今在加上得到了如意君的洞府宝藏。
<br/>　　甚至有可能那先天玉如意，都落入到夜天君的手中。
<br/>　　这个消息，对楚风眠来说，可不是一个好消息。
<br/>　　“怪不得如今夜天君没有来找我的麻烦，一旦他度过生死大劫，只怕是对我来说，也是一个祸端。”
<br/>　　楚风眠心中暗自想到，夜天君在北邙圣域中，已经消失很久了。
<br/>　　如今听玄北圣者的话，夜天君应该已经是前去寻找机会，度过生死大劫了。
<br/>　　夜天君对寒月漓，已经是窥探已久，尤其是楚风眠如今还杀了不少圣党的人，跟这夜天君，已经是结下了死仇。
<br/>　　夜天君一旦突破，对楚风眠来说也不是一个好消息。
<br/>　　不过楚风眠的心中，也并没有太多的慌张，夜天君如今在寻找机会，度过生死大劫。
<br/>　　但是这生死大劫，对越是天资极高的武者来说，就越是难以度过，以夜天君的资质，只怕是也要引出千雷皇劫来。
<br/>　　想要度过，并非容易的事情。
<br/>　　而楚风眠，如今也已经是达到了神力境的极限，距离度过生死大劫，也只差一步之遥。
<br/>　　楚风眠现在都能感觉到，若是他真的有意想要度过生死大劫的话，只怕是瞬息之间，就足以将劫雷引来。
<br/>　　只是楚风眠的心中，却是已经感觉，他距离要度过生死大劫，还相差一点。
<br/>　　还差一点，才能让楚风眠真正的完美，有着足够的把握，度过生死大劫。
<br/>　　楚风眠心思还在思考之际，玄北圣者笑了笑开口道。
<br/>　　“这一次虚空海六座古洞府出世，我们北邙学院不少的核心弟子，也都得到了极大的好处，在北邙圣域中的一些天才，他们的实力，也都已经达到了神力境极限的地步，都不容易对付。”
<br/>　　突然间，玄北圣者话锋一转的询问道。
<br/>　　“七日前，在众圣堂中，引出传承的人，是不是你？”
<br/>　　七日之前，众圣堂中出现的动静，已经是引起了北邙学院武术高层的注意，这一次引出传承，更是让许多隐藏的老古董，都是纷纷现身出来。
<br/>　　一般有弟子，进入众圣堂中，引出传承来，虽然值得高层注意。
<br/>　　可是像是一些老古董，却不会关注，因为对他们来说，这个世间已经没有多少事，比起他们修炼参悟更到的境界要紧。
<br/>　　除非是北邙学院遭遇劫难，不然这些老古董，都是很少会出现。
<br/>　　但是这一次，却是不同。
<br/>　　因为这一次的传承，却是北邙学院的创始者，北邙圣君的传承。
<br/>　　这北邙圣君的石像，是曾经北邙圣君亲自留下的，上面蕴含的剑术，极为深奥，北邙学院都从未有人参悟出来过。
<br/>　　可如今，居然是有人得到了他的传承。
<br/>　　玄北圣者，本来也是得到了这个消息，才会出关，却正好听闻了关于楚风眠的事。
<br/>　　“不错。”
<br/>　　楚风眠点了点头道。
<br/>　　得到传承这种事，就算是楚风眠自己不承认也没有任何意义，因为众圣堂中的一举一动，都会被高层放在眼中的。
<br/>　　查也足以轻易查的出来，也不至于隐藏什么。
<br/>　　“此话当真？”
<br/>　　玄北圣者眼神中似乎也有些不可置信，北邙圣君的传承，居然就被楚风眠得到了。
<br/>　　听到玄北圣者的话，楚风眠并没有在做任何回答，而是突然一指，向着玄北圣者点了过去。
<br/>　　这一指中蕴含的剑术，便是楚风眠从北邙圣君上参悟出来的剑术，一指猛然向着玄北圣者刺杀而去。
<br/>　　玄北圣者的实力，还要远远超越陆千刃，但是在这一指剑气面前，他的心中，甚至都跟感觉到了几分慌张，似乎是有些威胁。
<br/>　　“碰！”
<br/>　　玄北圣者，猛然大手一抓，这一道剑气，在快要接近到他的身边的一刻，便是被直接捏碎。
<br/>　　但是玄北圣者的额头上，都是冒出了几分冷汗，流了下来。
<br/>　　这一道剑气，居然是令他，都感觉到了紧张。
<br/>　　“这是……好玄妙的剑术！这剑术，的确是跟北邙圣君上的剑术，一模一样！”
<br/>　　玄北圣者猛然站了起来，脸色惊叹的说道。
<br/>　　他曾经也是无数次的进入过众圣堂，也参悟过北邙圣君的石像，虽然他无法参悟出来其中剑术。
<br/>　　但是上面蕴含的力量，玄北圣者还是清楚的。
<br/>　　跟楚风眠刚刚那一指剑气上的力量，一模一样。
<br/>　　这种剑术，简直是玄妙之际，惊为天人，楚风眠才不过神力境的境界，而玄北圣者的实力，要比起楚风眠强大百倍，千倍，万倍。
<br/>　　这样的一剑，居然是令玄北圣者，都感觉到了几分紧张，甚至是感觉到了一种死亡的气息。
<br/>　　这一剑的厉害，玄北圣者心中清楚无比。
<br/>　　他能够有惊无险的化解，也是因为楚风眠的实力不够，剑气威力不够，却并不代表这剑术的威力。
<br/>　　相反，能够让楚风眠以如今的实力，都足以令他感觉到危险，这剑术，绝对是玄妙到了极点。
<br/>　　只要是楚风眠的实力再进一步，度过生死大劫，只怕是玄北圣者想要接下那一剑，也并非一件容易的事。
<br/>　　“你居然真的得到了北邙圣君的传承！”
<br/>　　玄北圣者一愣，随后大喜道。
<br/>　　“好，不愧是我玄北的弟子，如此资质，简直是超越我所见到过的任何人！”
<br/>　　“这次归墟天开启，本来我还没有打算让你前去的，现在看来，以你的实力，进入归墟天，也是绰绰有余了。”
<br/>　　归墟天？
<br/>　　听到这个有些陌生的名字，让楚风眠有些一愣。
<br/>　　在他的记忆中，还没有知晓过这个名字。
<br/>　　“你不知道？看来还没有人告诉过你。”
<br/>　　北邙圣者看着楚风眠的惊讶，笑了笑道。
<br/>　　“这归墟天，是三千年前，九域发现的一处域外空间，是曾经一些远古时代宗门遗址的所在地。”
<br/>　　“归墟天中，有着无数远古宗门遗留下来的宝物，没三年，便是开启一次，这一次开启，也快了。”
<br/>　　“归墟天开启，九域各大门派都会有着名额，可以进入其中，我们北邙学院也有着名额。”
