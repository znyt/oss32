　　第391章
<br/>　　身无彩凤双飞翼，心有灵犀一点通。
<br/>　　莫将难脸上的表情越来越不好看，肌肉拧起，小臂绷紧。可陷入程大雷手中的刀，却不动一分一毫。
<br/>　　就连莫将难身后两名随从脸上，也露出惊愕神色。二人不敢相信，世间竟有人能用两根手指，接住莫将难势若雷霆的一刀。
<br/>　　可这一幕却在眼前真实发生了。
<br/>　　“我已经表现了足够的涵养，希望你不要得寸进尺。”
<br/>　　咔吧！
<br/>　　随着程大雷的声音，钢刀被程大雷两指硬生生折断，他手一挥，将刀尖扎在地上。
<br/>　　莫将难一张脸变得面如死灰，败并不可怕，可怕得是败得这么惨。程大雷展现出来的势力，让他明白自己和程大雷的差距，那几乎是穷尽毕生心血都无法跨越的。
<br/>　　可他毕竟是万马堂的首领，习惯了高高在上，今日当着这么多人，面子实在没处放。
<br/>　　“姓程的，山水有相逢，等下次见面，我必报今日之仇。”
<br/>　　程大雷撇撇嘴，忽然向前跨出一步。莫将难还没动明白怎么会事，程大雷已出现在他面前。忽地，程大雷抓住他的衣领，原地揪起来，直接将他丢出七八米外。
<br/>　　莫将难摔了个结结实实，五脏六腑都要快被震裂。两名手下过去，急急忙忙将他从地上搀起。
<br/>　　“大哥，你没事吧，大哥。”
<br/>　　莫将难吐出一口淤血，看了程大雷一眼。见程大雷目光望过来，立刻避开眼神。在两名手下的搀扶下上马，一溜烟不见了踪影。
<br/>　　待三人走出好远，回头已看不见琴川关的人，如此才长长的松了口气。
<br/>　　“大哥，你身体没大碍吧？”手下问。
<br/>　　莫将难脸上苍白，显然已经受了不轻的伤。关键是今天的事对他心灵的打击是巨大的，西北绿林之首，今天却在程大雷面前连一招都没有走完。
<br/>　　盛名之下无虚士，今日，莫将难才承认程大雷的了得。
<br/>　　“哼！”莫将难从鼻翼中冷冷哼了一声，道：“程大雷想做缩头乌龟，就由得他去做。我们兄弟去长安城，让天下英雄都知道程大雷的懦夫行径。”
<br/>　　“走！”
<br/>　　三匹马在茫茫荒野中消失不见。
<br/>　　……
<br/>　　夜，凉州城，一场欢宴正在城主府内举行。
<br/>　　宴会上，有凉州牧宋伯康，兵马将军戴白，长史陆洛，以及凉州城大大小小的贵人……，而今天宴会的主人，自然而然就是帝国皇子李行哉。
<br/>　　诸人席地而坐，在大厅中央，有西域舞娘摇曳着健美的腰肢，载歌载舞。
<br/>　　帝国上层都是很潇洒的，酒喝到现在，大家都有些醉意，便将清醒时的威严，礼节丢到一边。有人击剑而歌，有人陪着舞姬跳舞……其中最风流的当属李行哉，他左拥右抱，怀里是粉嘟嘟的舞姬，年龄至多也不过十八九岁，时而抱着这个饮一口小酒，时而吃一吃那个嘴上的胭脂。
<br/>　　福德勒今天也在宴会中，他距离李行哉并不远。只是无人愿意与他说话，当然，他也不愿意和这些人说过。只是坐在那里，冷眼旁观这一幕。
<br/>　　从遇见李行哉开始，还是第一次见到他如此模样，越来越像个帝国大人物。
<br/>　　或者，他本来就是。
<br/>　　福德勒并不知道，六皇子当年在长安，有两样事最出名，一是废物，其二就是风流。
<br/>　　此刻他杯来盏去，酒一杯杯灌入肚子，脸上已泛起红光，眼睛却仍清亮得很。
<br/>　　“殿下，请满饮此杯，今晚我都已安排好了，就有珠儿翠儿伺候殿下就寝，为殿下解一解风尘。”宋伯康。
<br/>　　珠儿翠儿就是陪着李行哉吃酒的舞女，二女都是花样年纪，看上去娇滴滴的惹人喜爱。
<br/>　　李行哉哈哈一笑，将杯中酒一饮而尽，又端起一杯酒，面向诸人。
<br/>　　“诸位都是帝国的栋梁之臣，今日解国之难，全仰仗格外，这杯酒，我敬大家。”
<br/>　　“殿下放心，为国为民，死而后已！”
<br/>　　众人欢宴，不知何时才休，李行哉被搀扶着回了房间，一搁在床上就呼呼大睡。
<br/>　　他和福德勒住的，是城主府内一套封闭的院子，里面雕梁画栋，曲檐回廊分外雅致，李行哉住的是正房，而福德勒则住在偏厅。
<br/>　　李行哉呼呼睡着了，福德勒却难以入梦。想李行哉睡觉时有侍女陪着，自己却只能孤枕冷被，即便福德勒心里也有些不平衡。
<br/>　　一阵夜风吹动窗棂，福德勒瞬间握住枕头下的腰刀，两只眼睛星辰一般发亮。
<br/>　　“谁？”
<br/>　　“别怕，是我。”李行哉平复着呼吸。
<br/>　　福德勒怔了怔，发现李行哉虽然酒气熏人，说话却十分清醒，不是刚才酩酊大醉的模样。
<br/>　　“你怎么过来了，舍得下你那美娇娘。”福德勒不无嫉妒道。
<br/>　　李行哉撇撇嘴，道：“别废话了，等到了长安，你要什么我与你什么。但这里……我们不安全。”
<br/>　　“这都是你家的地盘，又有什么不安全的。看他们对你态度多好，刚来就大摆宴席，还有美人侍寝。”
<br/>　　福德勒酸溜溜说了一句，发现李行哉脸上半点没有以前玩世不恭的样子，是前所未有的严肃。
<br/>　　“可我并不是为了美酒与美女来的，长安之围未解，谁有心在这里欢宴饮酒。”
<br/>　　福德勒一怔：“你的意思是？”
<br/>　　李行哉叹了口气：“帝国是李家的帝国，可凉州未必是李家的凉州呐。依我看，宋伯康根本没有出兵的意思。”
<br/>　　“既然不想出兵，却为何要接你回来？”福德勒困惑道。
<br/>　　李行哉略沉默片刻，道：“只等长安城破，挟皇子以令诸侯。”
<br/>　　帝国不缺少反抗者，但同样，李家也并不缺少支持着，毕竟李家这龙椅已经坐了一百多年。长安一破，各地诸侯必定打着剿灭逆贼的名义揭竿而起。到那时候，一个帝国皇子的名头就相当值钱了。
<br/>　　今日看到宋伯康的表现，李行哉就大概猜到了这一点。
<br/>　　宋伯康既然敢打这主意，就证明了两件事。第一，长安城很有可能守不住，第二，他是真把自己当废物了。
<br/>　　“我们得走？”福德勒。
<br/>　　“嗯，必须趁夜离开。”
<br/>　　“回琴川关。”
<br/>　　“不，去长安。”
<br/>
