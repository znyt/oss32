　　“哪里有什么线索。”叶少阳反问。
<br/>　　“前两句话啊，遥想孔雀王，佛祖身中藏……这个典故，大家都知道吧。”
<br/>　　叶少阳道：“这个为知道，说是当年佛祖被孔雀王吞了，从后背划了个口子出来的，佛祖觉得好像自己被孔雀王生了一回，于是认她做佛母，就是这个典故吧。”
<br/>　　小马抢着说道：“这个我也知道，西游记里也演过，所以佛祖管大鹏鸟叫舅舅呢，咦，说的不会就是鲲鹏吧！”
<br/>　　说着用脚踢了踢被定身了躺在地上不动的鲲鹏。
<br/>　　没人搭理他的胡言乱语，要说起来，鲲鹏的确跟西游记里那只大鹏鸟（假如西游记里的大鹏鸟是真实存在的）最多只是同族而已。
<br/>　　林三生把四句话一起读了几遍，思索片刻，恍然大悟地道：“九星阁，就在这个‘小四哥’的身体里！前面两句话是典故——佛祖在孔雀王身体里，暗示的就是九星阁也在这个小四哥的身体里！”
<br/>　　叶少阳皱眉道：“在人身体里怎么建修所？”
<br/>　　“不是修所在人身体里，是洞府在人身体里！”
<br/>　　叶少阳刚是有点懵，听林三生这么一说立刻明白了，洞府就是一会围观世界，小到可以只有一颗石子那么大，但只要开辟洞府的人足够强大，原则上能开辟出无限的空间，任何生灵只要进入这个微观世界，也就进入了它的规则之中，体形什么的都不是问题。
<br/>　　但洞府外壳的体积越是小，开辟围观世界的难度越大。传闻在佛国极乐世界，一粒沙都是一个世界，乃是西天阿弥陀佛凭借自身强大无匹的实力所开辟的。
<br/>　　星月奴实力不如佛祖，但也是罕见的强者，沙子这样夸张的不行，但鸡蛋大小的洞府开辟一个还是没问题的，这么大的东西完全能放在人身体里，甚至那人还不会发现……
<br/>　　“星月奴这一手，也是聪明的，要说安全，把洞府放在人身体内，这才是最安全的，而且外人绝对想不到……现在一切问题都清楚了，只差一个最关键的——小四哥是谁？”
<br/>　　小四哥是谁？
<br/>　　大家面面相觑。
<br/>　　叶少阳走到鲲鹏面前，伸手揭掉他脸上的定魂符，一只脚踩在他胸前，问道：“认识什么小四哥吗？”
<br/>　　“不认识，认识也不告诉你。”鲲鹏阴笑起来，“你有本事读取我的记忆，就自己去那个人啊。或许……你可以抽魂、镇魂，用各种严刑来对付我，你大可以试试啊！”
<br/>　　“你也是狠的。”
<br/>　　叶少阳知道这些对他都没用，拔出了七星龙泉剑。
<br/>　　鲲鹏早就做好了必死的准备，看见这一幕，淡然一笑，“叶少阳，你杀我没什么，你难道连你那个和尚兄弟也不要了？”
<br/>　　四宝！！
<br/>　　叶少阳一怔，上前一把掐住他的脖子，“四宝呢！”
<br/>　　鲲鹏得意冷笑，“哦，我以为你真的什么都不担心，原来你也有所顾忌。不过，我可是一个字都不会告诉你的。”
<br/>　　“你以为说了这个，我就不杀你了？”
<br/>　　“动手好了。”鲲鹏闭上了眼睛，连话都懒得说了。
<br/>　　叶少阳把心一横，就要扎下去，被林三生拦住，“先别，杀了他没什么用处，留着他，等见到星月奴或许可当个人质。”
<br/>　　“就他？星月奴能在乎他？”小马表示疑问。
<br/>　　“他总归跟了星月奴几百年至少，也是名义上的大弟子，试试吧，反正杀了也没用处。”
<br/>　　叶少阳想了下，还是听了林三生的话，用定魂符贴在他额头上。
<br/>　　四宝，果然是落在星月奴手中了。
<br/>　　大伙听到这个消息，都担心得很，在一起讨论起来，林三生安慰大家，从鲲鹏的话中可以听出，四宝肯定还活着，而且没什么大问题，八成也是被星月奴抓去当人质了。
<br/>　　听他这么说，大伙稍微放心，继续讨论那个什么小四哥。
<br/>　　包子犹豫再三，对大伙说道：“那个……我们时间不多了，在鲲鹏的记忆中，九星叠气大阵要七七四十九天完成，如今差不多还有两三天，一旦完成，她就证道了，她开创这个世界，也就是想要在这里当霸主，千秋万世……”
<br/>　　叶少阳一干人听的心惊不已。
<br/>　　“她不是去称霸人间吗？”吴嘉伟道。
<br/>　　“应该是以这里为基地，徐图人间。”林三生猜测道，环顾左右，道：“如果真让她证了道，再加上她身边那些帮手，我们虽然人多，但怕也不是对手……必须赶在她完成九星叠气阵之前干掉她！”
<br/>　　叶少阳道：“我支持，但是她的洞府在哪儿呢？”
<br/>　　“找啊。”
<br/>　　“好，这个任务交给你了。”叶少阳拍拍他的肩膀。
<br/>　　林三生耸肩。“走访吧，把消息传下去，看有没有人能提供线索。”
<br/>　　说完，林三生召集自己带来的那些手下，找叶少阳要了几张手写的紫符，每个人分了一张，作为信物，找各个将军或者掌兵的，让他们把队伍分配好，统一编制，等候命令——现在他们最不缺的就是人手，但人太多打仗未必就厉害。
<br/>　　林三生最懂这个，交代完之后，又找来几个传令兵，传达了那四句谜语和解释，发动所有人一起寻找线索，这个时候，面对这种毫无头绪的事，他也没什么办法可想。
<br/>　　“对了，包子，那诛仙四剑呢？”叶少阳想起这茬，问道。
<br/>　　包子摇摇头，表示鲲鹏的记忆里没有这方面的线索。
<br/>　　大伙于是猜测，八成是在无色海褪去时候，他们暗中有人拿着剑走了。于是林三生又传达了一个命令下去，动员所有人搜索圣灵会的余孽，凡是有线索赶紧来报。
<br/>　　“那咱们先在这住下吧，好在这皇宫就是地方大。”叶少阳环顾左右，看到坐在龙椅上的钟馗，说道：“钟天师，您老有什么打算？”
<br/>　　钟馗叹了口气，“能有什么打算，来了这里，只有先待着了。”
<br/>　　叶少阳叫来大内总管，给大伙分配和住的地方，基本上都在一个宫殿里。叶少阳看到张无生还坐在台阶上，捂着心口，怔怔发呆，于是问他怎么样了。
<br/>(本章完)
