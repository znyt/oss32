夏夕绾的话一落下，全场抽吸。

<br/>        天哪，夏夕绾在说些什么？

<br/>        她知道她是什么人么，她可是从乡下回来的医学废才，她怎么能如此大言不惭，厚颜无耻的说救人的是她？
<br/>
<br/>        “夏夕绾，你知道你在说些什么么？”李玉兰是最先反应过来的，她直接跳了出来。

<br/>        夏振国也从震惊里缓过神，今天夏夕绾表现的太平静了，原来她就是等着在电视台采访的时候语不惊人死不休，她真的太狠了！

<br/>
<br/>        果然，主持人和工作人员们都来抨击夏夕绾了，
<br/>
<br/>        夏夕绾，原来你是打的这个主意来参加采访的，你会金针封穴那我名字就倒过来写！
<br/>
<br/>        妍妍是圣黎院的高材生，她肯定会金针封穴，你一个才高中毕业的土包子从哪里学的金针封穴？
<br/>
<br/>        夏夕绾，你完了，今天你把事情闹大了，这可是直播，八千多万人在看着呢，现在全网都在骂你！

<br/>        没有人相信夏夕绾，夏夕绾一点都不奇怪，她澄亮的瞳仁里闪烁着细碎的笑意，“真的假不了，假的也真不了，我说是我救了老夫人，那就是我救的。”
<br/>
<br/>        说着夏夕绾看了夏妍妍一眼。
<br/>
<br/>        夏妍妍心里一慌，夏夕绾如此笃定的样子充满了自信，整个人神采奕奕，她都有些怀疑救人的究竟是不是夏夕绾了。
<br/>
<br/>        但是夏妍妍迅速恢复了理智，怎么可能是夏夕绾，她不过就是一个医学废才。
<br/>
<br/>        现在见大家都在骂夏夕绾，夏妍妍反而开心了，她觉得夏夕绾真是蠢，在直播的时候自取其辱。
<br/>
<br/>        “绾绾，”夏妍妍迅速露出了悲伤难过的情绪，“你为什么要这样说，不要撒谎了，救人的明明就是我啊。”
<br/>
<br/>        这时突然有人跑了进来，“不好了，重症监护室里的徐老夫人又发病了，她的心跳不停往下掉，就快休克了，妍妍，周主任让你立刻到重症室为徐老夫人金针刺穴。”
<br/>
<br/>        夏妍妍双眼一亮，她觉得老天都在帮自己，现在她要大显身手了，“绾绾，你说救人的不是我，那你现在就跟我去重症室，看我是怎么金针刺穴二救老夫人的！”
<br/>
<br/>        ……
<br/>
<br/>        重症室，徐老夫人一直在昏迷，还没有苏醒，现在她旁边的心跳监测仪已经发出了刺耳的“叮”声，徐老夫人情况危急，快休克了。
<br/>
<br/>        周萍迅速看向夏妍妍，“妍妍，你来了，现在徐老夫人情况很不好，你快点给徐老夫人金针刺穴。”
<br/>
<br/>        李玉兰开口了，十分委屈道，“周主任，你不知道刚才在摄影棚里发生了什么事情，夏夕绾竟然污蔑妍妍，她说自己才是救老夫人的那个人，现在要我家妍妍金针刺穴救老夫人是可以的，但是周主任要为我家妍妍正名，并且严惩夏夕绾！”
