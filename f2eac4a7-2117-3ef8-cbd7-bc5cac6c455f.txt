　　胖大叔的话，让李清照将信将疑，无毒蛇，还能已吃毒蛇为食？
<br/>　　这有点没道理吧，两种蛇，无毒蛇咬对方一口顶多流血，可有毒蛇咬你一口，你就中毒而亡了。
<br/>　　而且这蛇体型也不算大，也就四尺左右，一米有余，只是样子透着凶狠跟霸气。
<br/>　　但在怎样，还是无毒蛇，充其量狐假虎威罢了。
<br/>　　真的可以逆天抗毒？将一切毒蛇的毒是视为无物吗。
<br/>　　“小丫头，我这黑王蛇可是蛇界的一股清流，你要真不信，找你的长辈随便问问，你就知道了”
<br/>　　胖大叔对自己的蛇充满自信，继续说：“这黑王蛇是来自墨西哥，在当地鼎鼎大名，别说佘山，就连全国都找不到一条，我也是费了九牛老虎之力才弄到手”
<br/>　　“我准备等祭祀结束，到时候献给蛇王大人，放生佘山，那样我们佘山也能再多一种蛇品种了”
<br/>　　胖大叔的蛇是不是真那么厉害，李清照没有眼见为实，但他的话却让李清照肃然起敬，举起大拇指，道：“大叔大气，为佘山做贡献，我看这蛇品相极佳，恐怕也价值不菲吧”
<br/>　　“钱都是小事，只要蛇王大人能感受到我对它老人家的敬仰就好了”
<br/>　　“大叔棒棒哒”
<br/>　　李清照基本上相信了这条蛇的不凡之处，连忙跟这条黑王蛇来了一个大合照，然后跟胖大叔打了一个招呼离开了。

<br/>　　她抽空跟直播间的观众互动几下，小丫头就穿梭在人群中，不断的采访一些非常珍贵的毒蛇，甚至其中不乏珍稀品种。
<br/>　　珍稀品种的毒蛇可不得了，此蛇就等于网红一样，会被很多人围观，人气也非常高。
<br/>　　一条蛇起码百万价值，谁都想看上几眼，平日想看也根本看不到。
<br/>　　也就蛇王节，大家才将自己的好蛇拿出来展览炫耀一番。
<br/>　　李清照看到珍稀品种的蛇，更是打了兴奋剂一样，高兴的脸上溢出花来。
<br/>　　连续采访了十几条好蛇，忽然，李清照脚步停滞，被一道风景给不经意的吸引。
<br/>　　不过这次吸引她的，到不是什么珍贵的毒蛇，而是一位年轻的帅气小哥哥。
<br/>　　只见前方一位年轻男子闲庭信步的跟着大部队，脸上微微噙着一抹放荡不羁的笑，他肤色白皙，五官清秀中却透着一股俊俏，俊俏中还带着一股浓郁的邪魅之气。
<br/>　　浑身散发的气质很复杂，像是很多种气质的混合体，有着独特的邪魅不羁。
<br/>　　白话来说，就是这位帅哥骨子里带着一股坏坏邪邪的气质，眼尖的人或许会看出来，他跟蛇身上的气息有点像。
<br/>　　只是他这等容貌跟气质，很容易招惹女孩子的目光，忍不住的想多看几眼。
<br/>　　“哇，好帅的小哥哥”
<br/>　　李清照瞬间被吸引，黑宝石般的眼眸转了转，对着直播间灿然一笑：“嘻嘻，大家想必看蛇也看累了吧，清照顺便带大家看看帅哥”
<br/>　　“我们要看美女”
<br/>　　弹幕跳出来，清照瞬间无视。
<br/>　　“你好小哥哥，我能采访一下你吗”
<br/>　　年轻男子走着走着，忽然被眼前窜出来的一位美少女给拦路了，看了她一眼后，微微的蹙眉起来。
<br/>　　“怎么会是她”
<br/>　　男子心中诧异了下，但是神情没有一丝异常流出。
<br/>　　李清照作为新时代的祖国花朵，女搭男，根本没有一点压力，况且这位小哥哥长的实在特别。
<br/>　　“小哥哥你好，我叫李清照，是位户外主播，我想采访你一下可以吗”
<br/>　　年轻男子深深看了一眼她的手机，淡淡一语：“不可以”
<br/>　　啪...
<br/>　　李清照瞬间被打脸，一下子愣住了，这还是今天她采访的那么多人后，第一次被拒绝。
<br/>　　不，不是今天，是今年第一次搭讪被拒绝。
<br/>　　要知道李清照本身长的就很俏美可人，加上青春无敌的气息，几乎无人会拒绝她这样的美少女，很多人恨不得被她每天采访呢。

<br/>　　一位女粉丝立刻花痴般的弹出了一个弹幕。
<br/>　　“这男的牛叉什么，敢拒绝我们女神，是不是在装，心里肯定乐开花了吧”
<br/>　　“这叫帅吗，长的不伦不类，脸上还挂着坏笑，一看就不是好人”
<br/>　　男粉丝打抱不平。
<br/>　　一时之间，层出不穷的弹幕席卷而来，有夸着年轻男子帅气有个性的，有吃醋贬低的。
<br/>　　但也有中肯的。
<br/>　　“这小哥哥长的其实很好了，有点明星的气质”
<br/>　　“什么啊，他比明星还好看呢”
<br/>　　李清照被拒绝，回神后，也不气恼，做直播各种各样的事情都会遇见，更别说她这种户外的直播。

<br/>　　“本地人”
<br/>　　他冷漠的说了三个字。
<br/>　　李清照继续追问：“既然是白镇人，那你身上怎么没有蛇”

<br/>　　李清照现在才不管直播间，眼前的这位年轻男子给她很奇怪的感觉，甚至好像曾经见过一样。
