　　把枪送回去之类的事情都是小事，回到藏身之处后，舒天便来到一处事先约定好的联络点，将枪枝藏了起来，这些枪将在他们离开之后有专人负责回收。
<br/>　　而他们下一次如果再需要的时候，同样会有人再次送来，以便他们取用。
<br/>　　但是说实话，舒天是真的有些舍不得将这两枝枪放回去，毕竟这枪使着太带劲了，装上瞄准镜之后，别说是百步穿杨，就算千步他都有信心。
<br/>　　可是想归想，最后这两枝枪他还是要还回去，毕竟独孤玉凤说的没有什么错误，他们现在身处的环境的确是不大安全，如果这东西被一些有人心得去，所能造成的后果绝对是无法估量的。
<br/>　　……
<br/>　　话分两头，舒天藏枪之事先放到一边，却说那抢到了东西的鬼面和文士。
<br/>　　这两个家伙在混乱中离开之后，双双来到咸阳城中一处并不怎么起眼的客栈，各自定了房间之后，又再次聚到了一起，鬼面修罗用其难听的声音说道：“这位先生难道不想自我介绍一下？或者随意报个名字？老夫总不能用‘喂’来称呼你吧。”
<br/>　　结果不想那折扇男还真是喜欢装犊子，听了鬼面的话之后也不着恼，只是淡笑着说道：“名字只是一个代号，如果鬼面先生喜欢，叫我‘喂’也未尝不可。”
<br/>　　鬼面强压下心中想要狠狠抽那折扇男几记耳光的冲动，脸上露出一抹狞笑说道：“好吧，既然这样，那老夫便不客气了。”
<br/>　　“鬼面先生不必客气。”折扇男摆了个十分大度的姿态，末了继续说道：“某一直以为鬼面先生是独来独往，现在看来是某错了，原来鬼面先生也有自己的雇主！”
<br/>　　然而鬼面却一点没有给他这个面子，哼了一声说道：“‘喂’先生还是不要在这里套老夫的话了，你这一套对老夫没用，直接说，那份资料你需要什么条件才肯交出来。”
<br/>　　折扇男咂咂此，似乎有些犹豫，拿腔作势半晌才说道：“某不知道鬼面先生为什么会如此看重这份资料，不过某家建议最好是先生把资料交出来为好，否则明年的今日只怕就是您的大忌了。”
<br/>　　这鬼面修罗其实不是别人，正是那李毅恒身边的黄浦，鬼面修罗只是他在这几年在外面闯出来的绰号。
<br/>　　此人在隐王势力中的地位仅次于李毅恒，平日里颐气指使发号司令如何受得了那折扇男的这份闲气，闻言之后脸色变的极端难看，冷笑一声说道：“笑话，如果你真的有这份实力，刚刚在城东就该把老夫留下。”
<br/>　　但是那折扇男也不是吃素的，面对鬼面的冷笑，依旧保持着原来的笑容：“鬼面先生说笑了，如果你能保证在死之前不把手里的那份资料消毁，您可以试试能不能走出这间客栈。”
<br/>　　“哼，‘喂’先生说话真的很有意思，老夫真是佩服之极，不过想要引老夫上当，你的手段却还是差了些。”黄浦盯着那折扇看了半天，最终也没有起身尝试，只是用言语继续试探着他的深浅。
<br/>　　这个人的来历太过神秘，竟然连出身情报科的独孤玉凤都叫不出他的名字，这让黄浦不得不小心应付，生怕因为一时大意起了什么变故。
<br/>　　甚至说在某一时刻，他还怀疑过这个人是不是情报科丢出来的一个诱饵，专门用来对付自己这些野路子出来的家伙。
<br/>　　但是后来黄浦否定了自己的这个想法，因为在刚刚的那个情况下，如果情报科想要把他留下其实很容易，就凭那种数百步外可以打爆人的脑袋的步枪，他黄浦认为自己绝对不可能走出那片林子。
<br/>　　正是因为有了这样的想法，所以黄浦才相信折扇男并不是情报科的人，同样是因为这样的判断，黄浦也开始相信独孤玉凤的叛逃是真非假。
<br/>　　便是这样，黄浦与折扇男对峙了良久，直到外面的街道上隐隐传来喧闹之声，天色大亮的时候，折扇男才缓缓开口说道：“不如这样吧，我们彼此交换好了，我用我的一份资料，换取鬼面先生你的那一份，你看如何？”
<br/>　　“老夫为什么要跟你换呢？这里是大唐的国土，没道理让你一个高句丽人占了便宜吧？”黄浦阴森森的笑着，看那表情像是已经确定了折扇男的身份一般。
<br/>　　而那折扇男也并没有否认鬼面的猜测，只是淡淡一笑便继续说道：“鬼面先生这话说的就错了，这份资料其实我们谁也不知道是真还是假，若是假的，我们在这里争论还有什么意义？您说是吧？”
<br/>　　黄浦深深吸了一口气，不可否认的说，那折扇男说的一点没错，这份资料是真是假谁也不知道，只是根据独孤玉凤的要求来看，她给出来的东西应该是真的。
<br/>　　这是这也仅仅是应该，并不是百分百有把握的事情，没有经过检验，谁也无法保证这种事情的真假，用一份假情报来骗敌人自相残杀的事情黄浦也不是没有干过。
<br/>　　所以在略一犹豫之后，黄浦点了点头，随即说道：“可是你怎么保证你给出来的东西是真的？万一你给我的东西是假的呢？”
<br/>　　折扇男似乎早有准备，听了黄浦的话之后立刻说道：“这个好办，我们彼此手中都有一半的资料，将这份资料的缺口合在一起自然可以分清真假，然后我们再各自派人把对方的那一份抄下来，这样就可以保证是真的了。”
<br/>　　“好吧，既然如此，那就这么办。”黄浦在考虑了一会儿之后，终于点了点头。
<br/>　　事实上此时的他也没有什么太好的办法，毕竟有一半的资料掌握在对方的手中，他又的的确确是十分需要这份资料，所以只能同意对方的要求。
<br/>　　至于说硬抢，如果是当时在现场的话，硬抢的确可以，毕竟东西就摆在那里谁也做不得假，可是现在事过境迁，黄浦也不法肯定那份资料就一定在折扇男的身上。
<br/>　　如果他现在再想硬抢，万一资料不在折扇男身上的话，便一定会被销毁，到时候若是再想找出来可就不是那么容易了。
