　　四季城外，一支规模颇大的商队正在赶路。
<br/>　　为首的是两个老者，洞玄一重修为；后面是十数车商物。
<br/>　　商物周围，还有数十护卫，皆是先天武者。
<br/>　　如此强大的武者力量，根本没有任何人敢打他们的主意。
<br/>　　然而，最让两个老者放心的，并不是自己的修为，而是马车上竖着的一张旗帜。
<br/>　　只要旗帜在，便没人敢打他们商物的主意。
<br/>　　因为，旗帜上，写着‘慕容’二字。
<br/>　　这是盛宝商行的货物，无人敢来抢，也无人敢得罪北山慕容家。
<br/>　　两个老者，也正是百武城盛宝商行中的执事。
<br/>　　一个名叫慕容智，一个名叫慕容雄。
<br/>　　两人慢悠悠地赶着路，似乎丝毫不急。
<br/>　　慕容智忽然说道，“也不知昨日慕容胤执事他发什么火，从四季城白家回来后，就一直黑着脸。”
<br/>　　慕容雄笑道，“你没听说吗？昨日他没能炼制出雪魄丹，反倒是他一直冷嘲热讽的易霄炼制出来了，当着那么多人的面被打脸，不生气才怪。”
<br/>　　“还有就是我们商行进驻四季城之事，也因易霄，而被白家拒绝了。”
<br/>　　“易霄？”慕容智皱眉道，“可是那个紫炎？传闻打败了无忧公子的武道天才，没想到竟还是一位厉害的炼药师。”
<br/>　　“切。”慕容雄不屑道，“天才又如何，现在慕容胤执事恨不得要他的命。你等着吧，那易霄没几天好日子过了。”
<br/>　　两人闲聊着，忽然，一道冷冰冰的声音从空气中诡异出现。
<br/>　　“哦？是吗？”
<br/>　　“谁，谁在说话？”慕容智脸色一冷。
<br/>　　“哪来的鼠辈，还不赶紧现身？”慕容雄冷喝一声。
<br/>　　他的话音刚刚落下，身旁便诡异地出现了一道人影。
<br/>　　“想偷袭？找死。”慕容智大喝一声，一掌轰向人影。
<br/>　　人影的速度极快，似乎早有预谋般，也不知做了什么，只见身影一闪，便已经快速退去，没给慕容智攻击他的机会。
<br/>　　慕容智刚想追击，却猛地瞳孔一缩。
<br/>　　因为，在他身旁的慕容雄，胸口心脏上，竟被穿刺了一个小洞，鲜血汩汩直流。
<br/>　　慕容雄双眼一黑，甚至不知道自己身上发生了什么，已经失去了知觉。
<br/>　　“慕容雄。”慕容智大惊失色，堂堂一个洞玄一重武者，竟然被秒杀了？
<br/>　　“你是何人？”慕容智心中惊怒交加，看着前方的人影，质问起来。
<br/>　　“呵呵。”人影淡笑一声，冷声道，“慕容家的人，如此善忘吗？刚才不是还在说易某没几天好日子过了吗？”
<br/>　　人影边说着，凝聚出一股澎湃的紫炎火海，将这四周包围了起来。
<br/>　　人影，正是萧逸。
<br/>　　“紫炎易霄！”慕容智认出了面前之人。
<br/>　　“为何拦我慕容家的商队，又困住我们，还杀了一位执事，你想挑衅我们慕容家不成？”慕容智冷声说道。
<br/>　　紫炎火海，将所有护卫和车队都包围了起来。
<br/>　　慕容智嘴上强硬，但实则心里没有丝毫底气，要不然也不会拿自己的家族来压萧逸。
<br/>　　他看不出易霄的修为，只觉易霄身上毫无气息。
<br/>　　但他知道，无忧公子曾经斩杀过一位洞玄境武者。
<br/>　　而易霄打败了无忧公子，刚才又秒杀洞玄一重的慕容雄。
<br/>　　所以在他看来，易霄起码有洞玄三重以上的实力。
<br/>　　自己一个洞玄一重，就算加上身后的护卫，也绝对打不过一个洞玄三重。
<br/>　　面对慕容智的质问，萧逸没有回答。
<br/>　　此举，让得慕容智心里更加惊慌。
<br/>　　“易霄，别以为你是武道天才，四品炼药师，就能如此嚣张。”
<br/>　　“你可知我们运送的是何等重要的商物？你又可知我们乃是要和暗影楼作交易？”
<br/>　　“识趣地便快快离去，否则，哼，我若当你想抢夺这批商物的话，你就等着被北山郡两大霸主势力追杀至死吧。”
<br/>　　在慕容智看来，自己这番威胁，足以吓退易霄了。
<br/>　　北山郡两大势力的追杀，没人承受得住。
<br/>　　“哈哈哈哈。”萧逸大笑一声，“威胁我？告诉你，我要杀的，就是你们，要抢的，就是这批商物。”
<br/>　　说罢，萧逸脚步一动，立刻朝慕容智攻去。
<br/>　　他身上紫炎密布，看起来威势十足。
<br/>　　“不好。”慕容智脸色一变，自觉自己不是易霄的对手。
<br/>　　但出于武者的身体反应，他还是全力一拳轰出。
<br/>　　本来，这只是随手而为之，没想到....当他的拳头打到易霄身上时，立刻感觉到不对劲。
<br/>　　“噗....”萧逸猛地吐出一口鲜血，脸色惨白。
<br/>　　能修炼到洞玄境的，都不至于太蠢，立马反应了过来。
<br/>　　“不对，你根本没有洞玄境的实力。”慕容智先是一惊，而后马上看出了萧逸的实力。
<br/>　　“是又如何。”萧逸冷笑一声，“现在才反应过来，迟了。”
<br/>　　慕容智不明白萧逸的话什么意思，但下一瞬间，他懂了，也重新变回了恐慌的样子。
<br/>　　因为，恰在他一拳打在萧逸身上时，萧逸手上蓦地出现了一把散发着浓郁紫芒的神剑。
<br/>　　神剑刺出，慕容智根本来不及收拳后退。
<br/>　　“嗤”的一声，如同刺穿一张薄纸那么简单。
<br/>　　慕容智的心脏，瞬间被洞穿。
<br/>　　“怎...怎么可能....”慕容智满脸不可置信之色。
<br/>　　“死。”萧逸冷喝一声。
<br/>　　冰鸾剑的极致冰冷，在慕容智体内快速肆虐，将他体内的内脏和器官瞬间摧毁。
<br/>　　慕容智明显感受到自己身上的生机在不断消失，他恨阿，若是自己早看出易霄的实力，根本不会死。
<br/>　　没过几秒，他也如同慕容雄那般，双眼一黑，失去了知觉，而后死去。
<br/>　　萧逸看着慕容智死去，心头冷笑一声，随即便是体内火烧般的疼痛。
<br/>　　刚才慕容智的全力一拳，打得自己体内五脏六腑都受了重伤。
<br/>　　洞玄境，与洞玄境以下的力量，差距非常大。
<br/>　　当初钟无忧能杀掉一个洞玄一重武者，有很多因素，靠的是中品灵器、地阶武技等等一系列底牌。
<br/>　　可想而知洞玄境有多强。
<br/>　　正常来说，慕容智那一拳，足以杀掉任何洞玄境以下的武者。
<br/>　　幸好，萧逸的幽魂面具，为他抵消了一部分力量；真气外衣，又抵消了一部分。
<br/>　　最后靠着强悍的肉体，才勉强撑住。
<br/>　　饶是如此，仍旧是重伤的下场。
<br/>　　但，能杀掉慕容家两个洞玄武者，再大的代价也值得付出。
<br/>　　正常来说，他就算借助紫炎，实力全开也杀不了一个洞玄境武者。
<br/>　　但，幽魂面具隐藏气息的特效，再加上他的暗杀手段和藏身本事，哪怕他就距离慕容智和慕容雄咫尺之遥，两人也发现不了。
<br/>　　最后靠着冰鸾剑无坚不摧的锋利，才一举暗杀了慕容雄。
<br/>　　最后硬扛慕容智一掌，也杀掉了慕容智。
<br/>　　只要给他机会，让冰鸾剑能刺出，他将无人能挡。
<br/>　　冰鸾剑，作为这个世界最强大的紫色武魂，而且还是最深色的深紫色，没有东西能抵挡它的锋芒。
<br/>　　当初杀不了慕容墨，是因为萧逸仅仅是后天境，根本没有机会。
<br/>　　而现在，半步洞玄战力、速度，却很轻松就能觅得机会。
<br/>　　“呼...”萧逸吐出一口浊气。
<br/>　　紫炎火海的包围内，数十护卫紧张而恐慌地看着萧逸。
<br/>　　两个洞玄境执事被杀，让他们丝毫不敢反抗，连动弹半分都不敢。
<br/>　　“哼。”萧逸冷哼一声，大手一挥，紫炎火海开始收缩。
<br/>　　“不要阿，易霄大人，饶命阿。”
<br/>　　不少护卫开始跪地求饶。
<br/>　　“哼，求他做什么，他杀了我们，家族会为我们报仇的。”
<br/>　　“紫炎易霄，你会死得比我们更惨。”
<br/>　　也有硬气的护卫，谩骂着。
<br/>　　萧逸冷声道，“我死不死不知道，但你们现在，死定了，怪就怪你们是北山慕容家的人。”
<br/>　　话音落下。
<br/>　　在烈焰手套的增幅下，紫炎火海将这数十护卫以及商物完全吞噬。
<br/>　　冰鸾剑的存在，绝不能泄露出去。
<br/>　　当然，刚才自己的速度很快，冰鸾剑一刺便收，能发现他动作的，只有洞玄境武者，先天境还没有那个眼力发现得了。
<br/>　　而慕容雄还未反应过来便被杀了。
<br/>　　也只有慕容智，最后反应过来，看到冰鸾剑的存在，但也死了。
<br/>　　无论如何，这剩下的所有护卫，也将死去，便不必担心了。
<br/>　　所有护卫被杀，商物被毁，萧逸将这一地尸体的武魂吸收，而后取了他们身上的乾坤袋。
<br/>　　嗖，一个跳跃间，快速朝远方离去。
<br/>　　恰在此时，远处一大队人马快速袭来。
<br/>　　为首的，竟然是钟无忧，后面跟着一群黑衣遮面，气息骇人的黑衣人，显然是暗影楼的杀手。
<br/>　　“易霄，果真是易霄，远远便看到了他的紫炎火海。”钟无忧来到后，看着遍地的尸体，以及快速离去的萧逸，咬了咬牙。
<br/>　　“公子，在下去将他擒来。”一黑衣老者对钟无忧说道。
<br/>　　“去什么！”钟无忧呵斥了一声，“看不懂易霄的意图吗？”
<br/>　　“哼，那家伙，偏偏在今日抢我们和慕容家交易的货物，摆明了是在报复。”
<br/>　　“报复前些天暗影楼杀手的伏击，也报复昨日慕容胤的针对。”
<br/>　　“你若出手去追他，这算什么？岂不是说本公子怕了他？要靠暗影楼的力量对付他？”
<br/>　　钟无忧冷冷地看着已经远去的萧逸的背影，咬牙道，“易霄，本公子会找到你，亲手打败你。”
<br/>(本章完)
