　　第227章
<br/>　　小衣外套上软胄，软胄外套上盔甲，护喉、护肘、披膊、护心镜……一样也不能少。
<br/>　　杨龙停在仆人的服侍下穿上盔甲，想了想，又将经久不用的护身刀攥在手中。
<br/>　　却见他：
<br/>　　身着亮银甲，脚踏虎头靴，怪蟒吞珠光灿烂，吞头兽面金环嵌，刀藏冰霜，人放杀机，当真是威风赫赫，不愧帝国战神。
<br/>　　李善言呆呆的看着这一幕，呐呐道：“主公，需要这个样子么？”
<br/>　　“怎么不需要，小心做事一万年都不多，麻痹大意一次就够了。”杨龙停攥着手中刀：“我这个样子，肯定安全。”
<br/>　　“那么……主公不觉得热么？”李善言。
<br/>　　“这个，的确是有些热啦。”
<br/>　　这样的天气，李善言只穿着布衣都已经满头大汗，何况是身上套着上百斤重的铠甲，事实上，再刚才穿衣过程中，杨龙停已经浑身透汗。
<br/>　　最后，他还是把沉重的铠甲脱了，换了一身布甲，当然里面的软胄是不能脱的。即便闷得有些透不过来气，但出于安全考虑，杨龙停觉得这并不重要。
<br/>　　门外早有队伍等待，礼部本来的安排，是杨龙停骑着高头大马，在长安街上转一圈，让长安城的百姓瞻仰一下战神大人的风采。可是这样的安排却被杨龙停否了，公然游街，还是没有遮拦的马，这要是有人暗杀我怎么办，在高处射来一箭我岂不是就报销了。
<br/>　　于是高头大马换成封闭的轿子，保护他的禁卫军从三百人提到一千人，凡行过的街道，两边都立着全副武装的禁卫军，高处还必须有手持强弓劲弩的人警惕。
<br/>　　可即便这样，也无法抵挡长安城百姓的热情，以及天下士子对战神大人的崇拜。
<br/>　　多少人新潮澎湃，多少人眼含热泪，透过禁卫军的阻隔，将鲜花投到大街上。
<br/>　　“让我们看一眼吧。”
<br/>　　“战神大人，请您一定要保重身体啊。”
<br/>　　“一定要多喝热水，不要熬夜。”
<br/>　　鲜花满道，为了今天的事，城里多少花店被买空，又有多少千金小姐，因为去城外采花而被磨破玉手。
<br/>　　穿过漫长的街道，进入皇城的瓮城，今天将在这里决出武状元。
<br/>　　进入这里后，杨龙停才算是彻底松口气，自己现在身在帝国的中心，距离自己最近的是明帝陛下，刺杀自己的人再大本事，应该也进不来吧。
<br/>　　杨龙停到的时候，程大雷已经等待好久。他的官职比杨龙停低很多，这样的场合自然要提前到场，听候安排。因此，各种繁复的仪式程大雷都经历过，他的耐心一点点被耗尽，烦得几乎都忘了今天来这里的目的。
<br/>　　当杨龙停出现后，程大雷隔着好远向那边望了一眼。
<br/>　　一时间，那种不安的感觉又一次笼罩杨龙停，他目光环视一周，想要确定不安感来自何处。是因为天气太热，自己中暑了么。
<br/>　　而杨龙停并不知道，此刻还有一个人，和他有类似的感觉。金问道坐在高台上，心情一瞬间就不好了，这种被噩梦支配的感觉究竟来自何处。
<br/>　　今日明帝也到场了，包括朝廷文武百官，以及戎族的谈判团，外面还围着很多士子。当然，这些士子的出身都是经过严格审查的。
<br/>　　程大雷没有资格坐在看台上观礼，他是站在场地中央，不知朝廷抽什么疯，要他担任今天的考官。
<br/>　　以程大雷几乎没有的政治敏锐性，自然不会知道，这是明帝为了抬举他。过了今天，他，屈九江，柳轻名将成为帝国即将升起的将星，未来，不可限量。
<br/>　　武状元的人选将在屈九江和柳轻名两人中间决出，考试一共一个三个环节：箭法，马战，问策。
<br/>　　箭法和马战容易理解，所谓问策就是由明帝亲自提问，考他们对国家大事，军事战略的看法。而也就在最后一个环节，明帝将会决定他们谁是这次的武科第一。
<br/>　　第一就是第一，第二是不会有人记得的。
<br/>　　第一轮是箭法。
<br/>　　铛！
<br/>　　程大雷重重敲了一声锣，有气无力吐出两个字。
<br/>　　“开始”
<br/>　　箭法比的是马射，每人箭篓里有十支箭，快马奔过，一炷香的时间将箭射尽。距离他们一百步外立着稻草人，根据他们射中的部位计分，最后决出胜者。
<br/>　　柳轻名率先出场，他身下一匹大黑马，快马奔过时，马蹄溅起黄土，抽弓搭箭，羽箭破空而出。
<br/>　　“柳轻名，十箭中靶，头部一，喉部一，肩部三，胸部一，下身四。”
<br/>　　报靶的小校将柳轻名的成绩报出，不仅是为了让程大雷听见，也是为了让周围的士子听见。
<br/>　　柳轻名是有资格骄傲的，马上射箭考的不仅是箭法还有骑术，和身下马匹的配合，能在起伏的马上获得如此成绩并不容易。
<br/>　　柳轻名将弓箭卸下来，恢复体力为下一场做准备，同时也想要看看，屈九江能不能超过自己。
<br/>　　屈九江是一个文士，擅长诗词歌赋，琴棋书画，至于他的武艺众人并不了解。不过有勇气参与这次武举，本事应该并不弱。
<br/>　　他骑着一匹白马出场，这一幕不知便惹得多少人羡慕。公子白马，在天下人之前争一争第一，酷得一笔。什么叫第一，第一就是我比你们所有人都强，更何况今天争的是整个帝国的第一。
<br/>　　这场武举在帝国的瞩目程度，超乎很多人想象。据程大雷所知道的消息，长安城内已经有人开下盘口，大把人把银子压在二人身上，赌的就是谁能在这一场比试中夺魁。
<br/>　　“屈九江，十箭中靶，头部二，喉部一，肩部二，胸部一，下身四。”
<br/>　　成绩报出来后，可以说引起一阵轩然大波，程大雷也愣了愣，这成绩他自己也没有想到。
<br/>　　这次比试，头部记三分，喉部记三分，胸部记三分，肩部以及躯干记二分，下身记一分。
<br/>　　屈九江的成绩竟然比柳轻名还好，还真是扮猪吃老虎啊。
<br/>　　程大雷撇撇嘴，重重的击了一下铜锣，口中高喊一声。
<br/>　　“第一场箭法，胜者，屈九江。”
<br/>　　
<br/>(本章完)
