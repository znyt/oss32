　　霍斯曼德结束了跟玛丽科娃的视频通讯后，拿起桌子上的电话，拨通一个内线电话，说道：“克罗恩，来我办公室一趟。”

<br/>　　“Queen的命令，把Derrick手里‘水晶珠’的配方弄来。”霍斯曼德靠在椅子上，似乎在说一件无关紧要的小事，对克罗恩说道：“在没有拿到完整的配方前，记住留活口。拿到东西以后处理干净，钱会在你们抓到Derrick之后，先打一半到你们的账户里，拿到东西后，再把剩下的一半打给你们。”
<br/>　　克罗恩拿起桌子上Derrick的照片，什么话也没有说，转身离开了霍斯曼德的办公室。克罗恩知道Derrick之前一直都是玛丽科娃面前的红人，就连霍斯曼德都要给他三分面子。
<br/>　　现在突然要自己解决掉他，无非就是因为他的利用价值没有了，或者说是在利益上跟玛丽科娃起了冲突，克罗恩之前在美国海军陆战队远征直属侦察连（Force-Recon）服役，给美国政府做过的“脏活”、“黑活”、“湿活”可不在少数，其中的关键和猫腻，克罗恩很清楚，也很了解。
<br/>　　……
<br/>　　Derrick从走出霍斯曼德办公室的那一刻就清楚知道自己的处境，他不会愚蠢的真的等着霍斯曼德上门来跟自己谈条件，他很清楚玛丽科娃是绝对不会答应自己那两个条件，但是玛丽科娃绝对不会对自己手里的“水晶珠”配方放手，所以等到玛丽科娃找上自己的时候，就是自己的死期。
<br/>　　Derrick不会傻傻的等死，他现在唯一的保命符就是他手里的“水晶珠”配方。
<br/>　　回到实验中心，Derrick把事情进行了交代，借走了自己助手的汽车钥匙，迅速离开了实验中心。
<br/>　　克罗恩赶到实验中心大楼的时候，Derrick已经驾驶着自己助手的汽车离开。
<br/>　　很警觉啊。
<br/>　　克罗恩轻轻眯了眯眼睛，按住对讲机说道：“监控室，找到Derrick离开时驾驶的那辆车，其余人去武器室，我们有事情做了。”
<br/>　　走出实验室的时候，克罗恩停下脚步看了眼正在观察室房间里那张大床上酣战的钱鑫和陆岑，对Derrick助手说道：“Derrick真是个天才，居然能让亚洲人变得如此勇猛，这玩意儿没有副作用吗？”
<br/>　　“按照目前的服用剂量和频率，房间里那个男人很快就会因为心脏骤停而猝死……”助手回答道。
<br/>　　助手的话音刚落，跟陆岑缠在一起的钱鑫心脏出现了骤停，从床上滚落到地上，猝死了。而房间里的陆岑却并没有因为钱鑫的猝死感到恐慌，她不断抚摸着自己的身体，发泄着身体中的欲望。
<br/>　　克罗恩指了指观察室里的陆岑，对助手说道：“里面那个女人长得还不错，你可以考虑进去跟她发生点什么，我想她不会拒绝你的。”说完，克罗恩便走出了实验室。
<br/>　　助手扭头看了眼观察室中一丝不挂，一副欲求不满的陆岑，整个人顿时一激灵，这女人现在完全就是“榨汁机”，普通男人绝对无法满足她，除非自己也使用“水晶珠”，但这东西就是一把锋利无比的双刃剑，每次服用后都会对身体造成损害，当这种损害累积到一定程度的时候，就是房间里孙鑫的下场。
<br/>　　在“维伦恩”生物制药公司中，有一间一百多平米的房间被用来作为武器室，里面存放了世界上各种轻武器和所有口径、类型的弹药。
<br/>　　“小伙子们，这次行动我们要活口。”
<br/>　　克罗恩把一个麻醉弹弹匣塞入手枪中，把手枪塞进腰间的快拔枪套中，拿起桌子上的战术折刀，按下机簧扣，刀刃“噌”的一下弹起，在空气中闪着寒光。克罗恩在手中挽了一个刀花，收起刀刃把战术折刀贴身收好，说道：“所以这次，我们只能用橡胶弹和麻醉弹。”
<br/>　　一名保安看了眼自己拿在手中，.45口径的弹匣，叹了口气，连同自己手中的HK45手枪一起放回到了架子上，重新挑选了一把手枪，拿了几个麻醉弹弹匣。
<br/>　　……
<br/>　　Derrick驾驶着自己助手的车“维伦恩”生物制药公司离开后，在往自己家行驶而去的时候，用手机拨打了达美航空公司的订票电话。
<br/>　　“我需要一张机票。”Derrick一边驾驶着汽车，一边对电话那头的客服人员说道。
<br/>　　“请问您要去哪里？”
<br/>　　电话那头传来一个甜美的声音。
<br/>　　“不管是什么地方，我需要一张能在最短时间内离开这里的机票！”Derrick大声吼道，同时加深了脚下的油门，汽车引擎发出低沉的咆哮声，往前冲去。

<br/>　　Derrick想也没有多想直接就确定了自己要逃离的方向，逃到中国去，就算是玛丽科娃手段再厉害也不可能抓到自己！那里可是“雇佣兵的禁地”，是最安全的地方！
