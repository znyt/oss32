
<br/>　　<br/>　　    菜刀文借着一泡尿的理由遁走了，出门右拐直接打车回家了。<br/>　　<br/>　　    茅小草咬了咬嘴唇，嘴里一个劲的念叨着‘姐儿要矜持，矜持’但还是不由自主的走了过去，然后王惊蛰就眼睁睁的看着她坐在自己身边，很熟络拿起瓶酒倒上了，然后举起来说道：“喝一个？”<br/>　　<br/>　　    王惊蛰胳膊机械僵硬的端起杯子，轻轻的和对方碰了下，两人同时一饮而尽。<br/>　　<br/>　　    一杯酒喝完，场面瞬间就诡异的寂静了下来，旁边是DJ震耳欲聋的喊着麦，四周的年轻男女合着节拍和音乐扭动着身子，唯独他俩面面相觑，两耳不闻窗外事，耳边响起驼铃声。<br/>　　<br/>　　    因为王惊蛰和茅小草都不属于主动出击的那号人物，一杯酒喝完接下来就不知道从哪找突破口再整出一句台词来了，所以真像菜刀文所说的那样，他俩真就是靠着眼神来沟通了。<br/>　　<br/>　　    半天之后，王惊蛰看着对方绿汪汪的大眼睛，觉得人家敬了杯酒之后自己怎么着也得该还回去吧。<br/>　　<br/>　　    下一刻，王惊蛰满了杯酒，说道：“再喝一杯？”<br/>　　<br/>　　    “唰”一饮而尽，小草舔了舔诱人的嘴唇，脑袋里忽然想起好几句话，酒壮怂人胆啊，酒能让人失去理智是不是也能让人恢复记忆呢，喝酒能断片那是不是断片之后就思路如泉涌了什么的，总之酒是好玩意啊。<br/>　　<br/>　　    “哗啦”小草又倒了一杯，王惊蛰顿时皱眉了，这是川中一带喝酒的规矩么，两个人见面得先喝几杯才算开场？<br/>　　<br/>　　    于是，就在这种诡异的气氛中，桌子上放着的两打酒在不到半个小时的时间内，被王惊蛰和茅小草都给一扫而光了。<br/>　　<br/>　　    直到茅小草喝的有点醉眼朦胧了，晃着空空的酒瓶子冲着适应声喊道：“再来一打……”<br/>　　<br/>　　    王惊蛰才憋不住的整出了今晚跟她的第二句对白：“你是来蹭酒喝的？”<br/>　　<br/>　　    茅小草转动着僵硬的脑子，忽然把脸凑到了王惊蛰的面前，两对眼神瞬间火花带闪电的闪耀了起来。<br/>　　<br/>　　    “你原来还是那么不会说话”茅小草磨着牙说道。<br/>　　<br/>　　    王惊蛰尴尬的挠了挠脑袋，说道：“那我换个话题吧……还不知道你的名字呢”<br/>　　<br/>　　    “小草，我叫茅小草”茅小草笑的宛若一头狡猾的小狐狸，这个场景似曾相识啊。<br/>　　<br/>　　    王惊蛰“哦”了一声，没什么反应。<br/>　　<br/>　　    小草眯着眼说道：“是不是感觉名字太普通了，充满了泥土的芬芳？”<br/>　　<br/>　　    “呃，那倒不是，其实我的名字也一样，我叫王惊蛰，我生于惊蛰那一天”<br/>　　<br/>　　    茅小草死死的盯着他的眼睛，期望着能够在他的眼眶里面，看到一抹恍然大悟的神采，因为刚刚那一番对话，是他俩刚见面时曾经如出一辙的对白。<br/>　　<br/>　　    可惜的是，王惊蛰的眼睛里干净的很！<br/>　　<br/>　　    “喝酒，喝酒……”茅小草拢了拢头发，露出了小臂上的那颗守宫砂。<br/>　　<br/>　　    王惊蛰瞄了一眼，随即没有任何反应的举起了杯子，茅小草在心里叹了口气：“默默无语两眼泪啊……人生哪能多如意，万事只求半称心就可以了，喝吧喝吧，都在酒里了。”<br/>　　<br/>　　    另外一头，川中某高档小区外，齐泉和丁武坐在一辆车里，在他们后面还停着两辆，一辆中坐着老马，午桥和王生，后面那辆里还有四个人。<br/>　　<br/>　　    这个小区，就是王生从李杨嘴里挖出来的，吴洋藏身的地方，他住在三号楼二门的三楼，屋内有两人，除了吴洋之外剩下那个他也不知道叫什么。<br/>　　<br/>　　    齐泉打开手扣，里面放着一把黑色的仿六四和一个弹夹，子弹也被压满了。<br/>　　<br/>　　    丁武看见六四之后就愣了，似乎没想到齐泉的车里居然还会有把枪。<br/>　　<br/>　　    别看陈重身份很特别，还流着红色的血脉，但他下面一样有群边缘人士，就比如齐泉这一种的，因为有些事可以在台面上办，泡个澡喝个酒就能办明白了，但有些事用这种方式就不行了，还得采取特事特办的节奏，比如动个粗什么的。<br/>　　<br/>　　    就拿国内那些资产达几十亿和百亿以上的地产公司来讲，别看那些掌舵者又是首富又是委员什么的，他们公司下方一层层的往下，总有些见不得光的黑色人士为他们服务的。<br/>　　<br/>　　    “会用么？”齐泉把枪拿出来，递给了丁武。<br/>　　<br/>　　    丁武犹豫了下，接过枪后说道：“这玩意没用过，不过以前在老家的时候，家里有把单管，基本上原理都差不多吧”<br/>　　<br/>　　    一般人拿枪都得吓一跳或者是有新鲜感，但丁武手里握着这把仿六四的时候，却是满满的激动和和憧憬。<br/>　　<br/>　　    齐泉说道：“熟悉下，摆弄一会就明白了……小武，知道我为啥把枪给你么？我还得告诉你，过会如果有机会的话，你最好还得争取先开枪”<br/>　　<br/>　　    “开枪？这里是居民区啊，到处都是人”丁武愕然问道。<br/>　　<br/>　　    “把你喝的那瓶矿泉水倒干净了，枪口对准瓶口再开枪就行了”<br/>　　<br/>　　    丁武哦了一声，说道：“泉哥，你是让我在老板面前露脸呗？”<br/>　　<br/>　　    齐泉说道：“你知道老板下面，像我这样的人有多少么？我只能告诉你有很多，而这些人里至少有九成以上都希望自己能有为老板开枪的机会，因为只要枪一响陈重就能记住他了，你说这叫投名状也行，叫露脸也可以，总之这个开枪的机会很难得，因为能让老板用这种手段的时候已经不多了”<br/>　　<br/>　　    “谢了泉哥，我知道该怎么办了”丁武把枪别在了身后说道。<br/>　　<br/>　　    “啪，啪”齐泉拍了拍他的胳膊，笑道：“如果不想或者不敢，你也不要勉强，毕竟开不开枪就是往前和往后一步的事，你自己的选择最重要”<br/>　　<br/>　　    丁武说道：“以后我要是好了，我得记着是泉哥你拉我起来的”<br/>　　<br/>　　    “呵呵，走了，下车吧，听老马怎么说”<br/>　　<br/>　　    丁武和齐泉从车里下来，来到后面那辆车外，老马放下车窗后跟齐泉说道：“地址和目标我都给你了，我们不上去，在下面给你压阵，那个叫吴洋的人最好留下活口，另外一个不用管，然后把善后做干净就行了”
