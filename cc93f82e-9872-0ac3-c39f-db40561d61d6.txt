　　数十名古帝的联合攻击之下，纵然是八劫古帝，想要从中强行传过去，都无比困难。
<br/>　　陷入其中，稍有不慎便是一个陨落的下场。
<br/>　　“看来这大型宝藏，倒是没有开启的钥匙。”
<br/>　　楚风眠看着那宝藏入口开启之处，上面的七彩霞光也已经是消失不见，心中暗自想到。
<br/>　　在赤红大陆开启的大型宝藏，许多都是要得到钥匙，才有着资格，今日其中去争夺的。
<br/>　　但是眼前这，显然是已经是直接开启了，人人都可以进入其中，如此情况之下，谁先进入其中，谁便是就有着最大的机会可以得到其中的宝藏。
<br/>　　“走！”
<br/>　　楚风眠身形一闪，观察了一下，便是丝毫没有犹豫，一下子向着那宝藏的入口之处冲了过去。
<br/>　　楚风眠冲过去的一刻，一下子也被无数武技所笼罩过去，这些武者现在都是在争夺这先进入宝藏之中的资格。
<br/>　　不会放任何一个人过去。
<br/>　　楚风眠冲入过去的一刻，无数杀招笼罩在了楚风眠的身上，纷纷轰击而来。
<br/>　　“神树之力，加持我身！”
<br/>　　楚风眠清喝一声，神树的虚影缓缓的凝聚在了楚风眠的背后，这一道道的杀招，打入到了这神树的虚影之中的，当即是被全部化解。
<br/>　　楚风眠的身形毫不退缩，盯着无数杀招，直接向着入口处冲了过去。
<br/>　　偶然几道武技，躲避开了神树的抵挡，在轰击到楚风眠的身躯之上的一刻，楚风眠的身上却是出现了一层血肉晶体化为的屏障。
<br/>　　几道武技，根本无法撼动这现在的楚风眠身躯，刹那之间楚风眠就步入到了铜门之内。
<br/>　　“不好，有人进入了！”
<br/>　　“不要在这里僵持了，追上去，不然宝藏我们谁也得不到！”
<br/>　　看到楚风眠冲入到了铜门之中，那些武者也不在僵持，他们也清楚，在僵持下去，只是给楚风眠做嫁衣。
<br/>　　无数武者，争先也冲入到了铜门之中。
<br/>　　“大型宝藏？”
<br/>　　“好东西，又是一座大型宝藏开启了，看来是我先到的，这宝藏归我了！”
<br/>　　突然之间，从天而降一股庞大的威势，随即一道身影冲了下来，这是一名老者，他看到那大型宝藏的时刻，眼神之中露出了无比的兴奋。
<br/>　　“一群蝼蚁一样的人物，给我滚开！”
<br/>　　看着下方乱糟糟的一团，这老者突然出手，挥手之间，天崩地裂的一拳轰击而出，几名在他前方的古帝，当场被凌空轰杀。
<br/>　　“九劫古帝！”
<br/>　　“快退，有九劫古帝来了！”
<br/>　　人群看到这老者的出手，脸色大变，不少人都是纷纷退去，如今的时代之中，半步天人境的强者已经是无比罕见了。
<br/>　　九劫古帝，便是近乎无敌的人物，强横到极点。
<br/>　　“算你们识相。”
<br/>　　那老者看着人群散去，也停下了轰杀，身形一闪，直接步入到了铜门之内。
<br/>　　另一边，铜门之内，楚风眠刚刚步入其中，便是感觉到了另有乾坤，进入到了这铜门之内，出现的不是楚风眠从外面所见的宫殿。
<br/>　　而是一座桥梁，一座独桥，这独桥的两边，尽是深邃的黑暗。
<br/>　　同时刚刚踏入到了其中，楚风眠就感觉到了一种无比巨大的压力，仿佛是双肩上，有着两座大山一样，在压迫这楚风眠。
<br/>　　虽然对现在楚风眠的实力来说，这还不算什么影响，可楚风眠感觉到，他每上前一分，这身上的压力，也就会变得更加的巨大。
<br/>　　“这就是这大型宝藏的考验吗？”
<br/>　　楚风眠目光闪烁，一边思考，一边身形的速度却是在不断的加快，他现在是第一个步入到这铜门之中的。
<br/>　　楚风眠必须要抓住这个先机。
<br/>　　隐隐预约之间，楚风眠感觉得到，在这独桥的尽头，应该就是真正宝藏的所在。
<br/>　　楚风眠的身上，十二万九千六百滴远古战龙精血，全部被催动了起来，一时间化为了一条远古战龙的模样，向着前方猛冲而去。
<br/>　　刚刚跟随楚风眠进入到了这铜门之中的几名武者，一下子都是被远远的甩开，在这里的压力，足以是让古帝，都寸步难行。
<br/>　　许多七劫古帝，刚刚踏入其中，都走不了几步，都没有力气了，只能够是退去。
<br/>　　想要得到宝藏，机缘，实力两点，缺一不可。
<br/>　　进入这铜门，考验的是机缘，而想要得到其中真正的宝藏，所凭借的，就只有实力。
<br/>　　轰隆！
<br/>　　就在楚风眠继续前行的一刻，一股天地之力，突然笼罩在了这独桥之上。
<br/>　　一尊强者降临了。
<br/>　　这种天地之力，足以影响整个独桥，唯有九劫古帝可以做得到。
<br/>　　“有九劫古帝来了！”
<br/>　　楚风眠目光之中露出了几分冰冷，九劫古帝，是他的真正对手。
<br/>　　在这赤红大陆之中的九劫古帝，虽然没有那么多，但是总会碰到几个的。
<br/>　　尤其是这大型宝藏出世，霞光数百里都看的清清楚楚，自然会引来九劫古帝。
<br/>　　这一尊九劫古帝降临之后，也是催动了全力，向着楚风眠追逐而来，这速度丝毫不弱于楚风眠。
<br/>　　不过幸好的是，楚风眠比起这九劫古帝提前进入到了铜门之中，现在已经是占据了先机。

<br/>　　楚风眠打量了一眼，身形一闪，直接来到了这宫殿前方，手心之中凝聚灵力，猛然一道龙爪轰击而去。
<br/>　　“破！”
<br/>　　咔嚓！
<br/>　　随着这一道龙爪的横扫，上面的阵法纹路纷纷破碎，古老的大门之上，打开了一个缺口，露出了宫殿之中的模样。
<br/>　　在这宫殿之内，一株半人高的树枝，躺在上面，随着古老大门被打开的一个缺口，无比精粹的生命力，从中散发出来。
