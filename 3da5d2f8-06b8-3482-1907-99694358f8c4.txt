　　轰！轰！轰！轰！轰！
<br/>　　嘭！嘭！嘭！嘭！嘭！
<br/>　　写字楼走廊垃圾桶中那些塑胶炸药在计时器归零的瞬间，同时发生爆炸。
<br/>　　十公斤重的塑胶炸药，已经可以把写字楼一层楼给摧毁掉；塞在楼层走廊垃圾桶中的塑胶炸药绝对不止十公斤，而且还是二十几层楼同时发生爆炸，这样的情况……足以用惊天动地来形容了！
<br/>　　整栋写字楼所有楼层的玻璃墙全都被震碎，碎裂的玻璃墙碎片在空气中闪烁着刺眼的光芒，朝着四周飞射而出，这一刻，这些玻璃碎片彻彻底底成了杀人利器！
<br/>　　马路上，K国快速反应部队和战备一队队员全都撤走了，但是之前被拦截下来的那些替身尸体和汽车还留在了马路上，而这些替身尸体和二手车就成了玻璃碎片的攻击目标。
<br/>　　铿！铿！铿！铿！铿！
<br/>　　玻璃碎片不断打在那些七横八竖停在马路上的汽车上，原本就有些老旧的汽车瞬间被打得坑坑洼洼，有些地方更是瞬间被玻璃碎片给打穿，倒在车厢内的替身尸体上也是又多了几个伤口，有几辆就停在写字楼前的汽车更是被吹飞起来，在空中打着转，然后狠狠地倒扣砸落在地面上。
<br/>　　哐！
<br/>　　砸落在马路上的汽车瞬间变形，车顶凹陷下去，框架变形，原本已经接近报废的二手车，现在是彻底报废了，只能被送入废车场，当做废铁来处理。
<br/>　　那辆载着罗昊、林晓阳、郑钰这些人撤离的MPV虽然跑的飞快，但还是受到了波及，接近半吨塑胶炸药爆炸所产生冲击波不是闹着玩的，气浪席卷而立，还在马路上疾驰的MPV被吹得失去了控制。
<br/>　　驾驶着MPV的战备一队队员保持着作为一个特战队员应该有的冷静，紧握着方向盘，脚下点着刹车，努力保持着汽车的平衡；这个时候如果一脚刹车踩到底，绝对就是整辆车侧翻，然后车里这么多人生死不明的情况……
<br/>　　吱——！
<br/>　　原本七人座的MPV被超载塞入了十个人，超载超重这个时候倒是帮了忙，令得车子在马路上滑行了几十米后，总算是停了下来。
<br/>　　MPV内，罗昊甚至都闻到了车轮摩擦后所产生的橡胶味，水泥马路路面上留下了黑色的橡胶拖痕，仿佛是在告诉车厢内的人，就在刚才，就在十几秒钟之前，他们经历了一段怎么样的生死体验。
<br/>　　写字楼的玻璃墙已经全都消失了，写字楼内部也是被炸得一塌糊涂，写字楼外的马路上看到最多的，还是地面上的玻璃碎片，还有就是被吹翻倒扣在马路上的汽车，以及被甩出汽车，被地面上碎玻璃划得面目全非的替身尸体。
<br/>　　“傻小子，你们情况怎么样？”赵天宇握着对讲机问道，他握着对讲机的手心中忍不住渗出了汗水，就在十几分钟前，他失去了一个兄弟，他不想在十几分钟后又失去自己的学生。
<br/>　　子午牺牲让云熙晕倒，如果罗昊再出事情，会让这丫头发疯的！
<br/>　　赵天宇一开始就想要询问罗昊他们的情况，但是爆炸对无线电信号产生了干扰，他无法跟罗昊取得联系。
<br/>　　“没事，就是被吓了一跳！”
<br/>　　对讲机中清楚的传来罗昊的声音，听到罗昊还能跟自己开玩笑，赵天宇也是跟着松了一口气。
<br/>　　“没事就立刻给我滚回来。”赵天宇骂了有一句，随后又说道：“小熙那丫头因为子午的事情晕倒了。”
<br/>　　很快，罗昊、林晓阳、李毅峰三个人就回到了指挥中心，现场留下了慕容飞、郑钰和战备一队的队员。
<br/>　　罗昊急匆匆的走进指挥中心，对赵天宇问道：“师父，小熙她怎么样？”
<br/>　　“没什么大碍。”赵天宇回答道：“这几天她本来就没休息好，加上受到子午的事情刺激晕了过去，现在被送去休息了。”
<br/>　　“我去看看她。”罗昊说道。
<br/>　　赵天宇点头同意了。
<br/>　　罗昊离开指挥中心后，赵天宇的视线又回到了写字楼的方向，那栋写字楼算是彻底毁了。这一系列的爆炸把这栋装修考究的写字楼彻底打回解放前，不说写字楼里的装修，这样的爆炸下，就算写字楼的结构再稳固也不可能不受到破坏，如果质量稍微差一点的楼房估计直接塌了……
<br/>　　轰隆！
<br/>　　赵天宇还在琢磨那栋写字楼遭到的破坏程度，写字楼就突然坍塌下去，用现实给了赵天宇最好的回答。
<br/>　　这栋二十五层楼高的写字楼就这么变成了一堆废墟，而在那堆废墟中埋葬着他的一个兄弟！他的名字叫子午！
<br/>　　赵天宇问过子午，他原来的名字叫什么，子午只是微微一笑，回答他，原来的名字他自己已经都忘了，生活在网络世界里的这么多年，他已经习惯了“GHOST”这个名字；但是因为需要将子午的档案入库，姓名那栏又不可能填写“GHOST”或者“子午”这种一看就是代号的名字，子午必须得有自己的名字。
<br/>　　子午在想了一个晚上后，告诉赵天宇他的名字叫“阮钊”，他原来的名字确实忘记了，只记得自己姓“阮”，至于为什么叫“阮钊”，因为“钊”这个字取了“赵”的谐音，子午说，他的命是赵天宇救得，就叫阮钊。
<br/>　　现在这个一晚上时间给自己取了一个并不怎么响亮名字，平时更多时候都叫代号，没有成家、没有孩子、一旦面对电脑就瞬间像变了一个人的家伙死了，就埋葬在那堆废墟中，而自己却无法将他带回国去。回去以后，在他的墓穴中也只能放上他一套常服，为他立下一个衣冠冢。
<br/>　　赵天宇紧握拳头，一拳砸在墙壁上，发出“咚”的一声闷响。
<br/>　　……
<br/>　　K国，某合资油气厂。
<br/>　　这座油气厂建造于上世纪九十年代末，是由中方和K国合资建造的，虽说是合资建造，K国方面只是出了这么一块厂房用地和建厂后的员工，最主要的技术骨干还是中方这边的人，即便是到了现在，还有中方的技术人员在这里工作。
<br/>　　智天使驾驶着黑色宝马朝着油气厂的方向驶来……
