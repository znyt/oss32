　　“你也是炼药师？”白管事直视萧逸，沉声问道。
<br/>　　“不错。”萧逸淡淡地点点头。
<br/>　　“几品？”白管事认真地问道。
<br/>　　萧逸刚要回答，一旁的慕容胤冷声道，“白管事，你还真的相信一个臭小子的话？”
<br/>　　“叶铭是裂天剑派炼药堂首席，我们等也就等了，可易霄嘛，哼，与他多费唇舌，不过是浪费时间罢了。“
<br/>　　慕容胤边说着，边露出了高傲的神态。
<br/>　　他虽是四品炼药师，但修为却只有洞玄三品。
<br/>　　此人在炼药上颇有能力，可在武道修炼上，只能算平庸。
<br/>　　起码以他这个年纪来说，并不出色。
<br/>　　加之他是个心胸狭窄之辈，所以尤其见不得其他武道天才。
<br/>　　易霄最近声名鹊起，连北山榜第十的无忧公子都打败了，无疑是个超级天才。
<br/>　　可是，在慕容胤看来，这个天才，不过是不知从哪冒出来的臭小子罢了，无背景，无势力。
<br/>　　那不屑的目光盯着萧逸，似乎在说，天才又如何，无背景，无势力，在成长起来之前不过是一只蝼蚁罢了。
<br/>　　萧逸心头冷笑一声，真不愧是北山慕容家的人，个个都是小人。
<br/>　　一旁的方鹤则说道，“紫炎易霄是吧，我劝你还是闭上嘴巴，退回一边去吧。”
<br/>　　“当然，方某没有轻视你的意思。起码在武道修炼方面，方某当年如你这般年纪时，不如你。只是，慕容胤是个小人，你若得罪了他，麻烦可大咯。”
<br/>　　“方鹤，有种再说一遍。”慕容胤脸色一冷，脸上露出了杀意。
<br/>　　“两位，还是改天再处理私人恩怨吧。”叶铭一直都是神态自若，淡淡地说道，“白家主还等着我们炼制雪魄丹。”
<br/>　　白管事反应了过来，再次问道，“易霄，你是几品炼药师？”
<br/>　　萧逸没有回答，而是问道，“雪魄丹，可有丹方？”
<br/>　　白管事点点头，沉声道，“丹方、材料，一切都准备好了。只等有能之人炼制雪魄丹。”
<br/>　　“那便好。”萧逸微微一笑。
<br/>　　“你到底是几品炼药师？”
<br/>　　萧逸耸耸肩，自信道，“不比慕容胤差。”
<br/>　　“什么？”白管事顿时一愣。
<br/>　　在场其余武者则是一惊，“不比慕容胤执事差？这岂不是说易霄也是四品炼药师？”
<br/>　　随手众人便是再次一笑。
<br/>　　“开什么玩笑，看这易霄的年纪，比叶铭还小，怎么可能是四品炼药师。”
<br/>　　“怕是这易霄被慕容胤执事说了几句，心里不忿，故意说高了吧。”
<br/>　　“易霄，你还是退一边儿去吧，别浪费大家时间。否则，惹恼了几位炼药师大人，你吃不了兜着走。”
<br/>　　“要是耽搁了白家的时间，就算你是紫炎易霄，也必死无疑。”
<br/>　　听着周围的质疑和哄笑，萧逸充耳不闻。
<br/>　　白管事再次问了一句，“易霄，你真的是四品炼药师？”
<br/>　　事关雪魄丹，白管事不得不慎重。如今若能再多一个四品炼药师，对白家来说无疑是极好的事。
<br/>　　“还是那句话，不比慕容胤差。”萧逸淡淡地说道。
<br/>　　“小子你找死。”慕容胤顿时勃然大怒。
<br/>　　在慕容胤看来，易霄三番四次说不比自己差，根本就是在挑衅自己。
<br/>　　“慕容胤，收起你的气势吧。区区洞玄三重，杀不了我。”萧逸冷冷地说道。
<br/>　　“哼。”慕容胤冷哼一声，就要出手。
<br/>　　一旁的白管事立刻拦下，道，“慕容胤执事，稍安勿躁。”
<br/>　　说罢，白管事看向萧逸，沉声道，“易霄，雪魄丹的材料太稀罕，我们白家一时半会儿也无法准备太多。”
<br/>　　“你若真有本事，自然是好的。若是乱说一通，浪费了材料，休怪我白家对你不客气。”
<br/>　　“不过，你也是武道天才之一。若真的炼制失败了，便拿你替我家小姐抵命。”
<br/>　　“抵命？”萧逸目光一凛，随即冷笑一声，“白管事，易某告辞。”
<br/>　　“嗯？”白管事脸色瞬间一冷，“你刚才在耍我？”
<br/>　　慕容胤更是冷笑道，“哼，告辞？我就说这臭小子虚有其表，不过是在浪费时间。”
<br/>　　在场武者也是纷纷摇头，“我就知道，这易霄是在捣乱。”
<br/>　　“哎，现在才要退去，跟戏耍白家有何区别，这易霄今日怕是惨了。”
<br/>　　其实萧逸是有把握的，只是，他不喜白家人的态度。
<br/>　　萧逸直视白管事，道，“敢问白管事，炼制这雪魄丹，慕容胤他们可有把握？谁又敢说一定能炼制成功？”
<br/>　　“他们若失败了，也是抵命吗？”
<br/>　　“白管事说我耍你，倒不如是是你欺我。”
<br/>　　“既然如此，易某告辞便是。相信没人会为了你白家那区区报酬，拿自己的性命作赌。”
<br/>　　说罢，萧逸转身便欲离去。他不喜白家人的态度，便不再多留。
<br/>　　不是怕了白家，仅仅是因为不喜欢。
<br/>　　天才，是有傲气的，萧逸更是如此。
<br/>　　“哼，白家是你想来就来，想走就走的吗？”白管事怒喝一声，就要出手。
<br/>　　恰在此时，内堂中，一位中年人凭空而现，面露不悦之色。
<br/>　　“为何如此吵闹？叶铭还没到吗？”中年人一来，便对着白管事质问起来。
<br/>　　“参加家主。”白管事见了中年人，立刻行礼。
<br/>　　在场武者，也立刻行礼。
<br/>　　包括方鹤和慕容胤，也微微鞠躬，礼貌地叫了一声，“白家主。”
<br/>　　只有叶铭，仍旧神色自若，没有动作。
<br/>　　现身的中年人，正是白家家主，这四季城的城主，在整个北山郡都是响当当的大人物，破玄境武者之一。
<br/>　　白管事对白家主说了一遍刚才的事。
<br/>　　“哦？”白家主微微点头，而后道，“易霄小兄弟，且慢。刚才是我的下人失礼了，还望不要放在心上。”
<br/>　　白家主的态度，很平淡，但丝毫没有半分恶意。
<br/>　　“白家主愿意相信易某的炼药本事？”萧逸停下脚步，转身问道。
<br/>　　“不信。”白家主毫不犹豫地摇摇头。
<br/>　　“不过。”白家主认真地说道，“我相信能打败无忧公子的人，绝不是蠢货。换言之，你是个聪明人，你不会在我白家如此紧急的时刻还来戏耍我白家。”
<br/>　　“这证明，你有那个本事。”
<br/>　　白家主轻声道，“如今请不来五品炼药师，时间紧迫，多一个四品炼药师，雪魄丹的炼制成功率便多一分。”
<br/>　　“易霄小兄弟，慕容胤执事，方鹤先生，叶铭小兄弟。你们四人尽管炼药便是，无论成功与否，我白家都承你们的情，绝不计较半分。”
<br/>　　白家主认真地看了四人一眼。
<br/>　　“白家主言重了。”四人同时说道。
<br/>　　萧逸也拱了拱手。
<br/>　　“炼药前，四位可有什么要求，只要是我白家能力范围内的，必定满足。”白家主说道。
<br/>　　叶铭、方鹤、慕容胤三人摇了摇头。
<br/>　　反倒是萧逸，站前一步，道，“白家主...”
<br/>　　几乎是在萧逸准备提要求之时，周围的武者已经脸色一变。暗道，白家主只不过是客气地问问，易霄竟然真地去提要求了。
<br/>　　慕容胤更是心头冷笑，臭小子，如此不识相，真真找死。
<br/>　　白管事眉头一皱，脸色一冷。
<br/>　　白家主则是摆摆手，道，“有什么要求，尽管说。”
<br/>　　萧逸说道，“白家主，能否为在下安排一个安静之处。那雪魄丹的丹方，在下要观摩半个时辰。”
<br/>　　萧逸注意到周遭武者的神色，虽说现在提要求不好，但还是要提出来。
<br/>　　“白家主，在下只是想更有把握地炼制雪魄丹，免得出意外。”
<br/>　　“呵呵。”白家主蓦然一笑，道，“易霄，你这算什么要求？”
<br/>　　白家主淡笑道，“我本来就打算带你们去安静的地方揣摩丹方。”
<br/>　　慕容胤冷笑一声，像看白痴一般看了眼萧逸。
<br/>　　叶铭则轻笑道，“易霄小兄弟，雪魄丹位列四品巅峰丹药，又是偏门丹药，可没几人炼制过。”
<br/>　　“就连我都要去研究一下丹方，才敢开炉炼药。自然是要提前找个安静的地方，观摩丹方的。”
<br/>　　方鹤也是轻笑一声，道，“小子，可以阿。那雪魄丹，我起码要研究数个时辰才敢说有把握，你竟妄言半个时辰，真是后生可畏。“
<br/>　　“额，是这样吗？”萧逸尴尬地摸了摸鼻子。
