　　楚风眠成为了天地异种，成为了天地之中独一无二的生灵。

<br/>　　楚风眠自己也不知道，根本无法猜测出。
<br/>　　楚风眠也只能够是从他身上的蛛丝马迹之中，可以猜测到，他未来的天人大劫，只怕是会跟任何人的都不一样。
<br/>　　会比起任何人的天人大劫，还要凶险万分。
<br/>　　楚风眠本来也是想要看看这狂龙帝的天人大劫，会不会跟他人有所不同。
<br/>　　现在看来，这狂龙帝的天人大劫，也是跟楚风眠在古籍之中见到过关于他人度天人大劫的场景一样。

<br/>　　他的天人大劫，也没有任何的变化。
<br/>　　现在看来，楚风眠未来去度天人大劫，也没有其他人的经验可以去看了，只能够是凭借他自己了。
<br/>　　毕竟度过逆天大劫，成就天地异种的人，哪怕是在历史长河之中，都没有记载过，只有一些隐秘之中，有提到过一两笔。
<br/>　　而且现在楚风眠更是以一人，掌控了龙之本源，吞天本源，两种本源之力，只怕是从未有人做到过楚风眠一样的事情。
<br/>　　楚风眠要走的道路，只有他自己开辟。
<br/>　　楚风眠脑海之中思考之际，另一边，那宫殿之上的劫云，也已经是开始散去。
<br/>　　狂龙帝的天人大劫，结束了。
<br/>　　以狂龙帝在这无数岁月之中的积累，他度过天人大劫的几率，几乎是百分之百，没有悬念。
<br/>　　这个结果，楚风眠也不吃惊。
<br/>　　楚风眠的目光看向狂龙帝，仔细的打量着。
<br/>　　如今眼前的狂龙帝，虽然只是刚刚度过了天人大劫，可已经是踏出了天人境的第一重，成就了仙人九阶中的仙人。
<br/>　　已经算是真正的羽化登仙了。
<br/>　　眼前狂龙帝的一身实力，已经是高深莫测，到了一种楚风眠都无法探察全部实力的地步了。
<br/>　　之前狂龙帝的一身实力的确强横无边，可楚风眠却还是可以一眼看出狂龙帝的全部实力。
<br/>　　现在楚风眠却是做不到了，这就是踏入天人境的转变。
<br/>　　楚风眠感觉到狂龙帝的一身实力，比起之前至少要强横了一倍。
<br/>　　当然这还是狂龙帝刚刚突破，还没有彻底熟悉力量，在闭关苦修之后，狂龙帝的实力还会再进一步。
<br/>　　不过楚风眠倒是注意到了一点，就是狂龙帝身上的力量，还都是灵力，而并非是仙力。
<br/>　　曾经楚风眠所交手过，那五行刀圣的师尊，林五行，其实出手爆发出的力量，也是灵力，而并非是仙力。
<br/>　　仙人九阶之中，每一阶都有着极大的差距，只有踏入到天人境四重，成就仙士的时候，才可以转化力量，将一身力量，转化为仙力。
<br/>　　仙人层次的力量，其实跟帝尊境，还有着几分相似，不过成就仙人，也有着一种强横的手段，那就是一念成术。
<br/>　　之前在跟林五行交手之中，楚风眠便是就见识过这一念成术的威能。
<br/>　　随随便便的一个字，一个念头，都可以化为媲美天级武技的威能。
<br/>　　所以一旦是踏入了天人境，一切的武技都没有任何的意义了，只有仙术，才有用。
<br/>　　凭借这一念成术，天人境的强者，就足以是几乎横扫所有天人境之下的人物。
<br/>　　“师尊。”
<br/>　　劫云消散，狂龙帝也恢复了人身，走了过来。
<br/>　　尹龙仙急忙的迎了上去，一脸喜色的说道。
<br/>　　“恭喜师尊痊愈伤势，度过天人大劫，成就天人。”
<br/>　　“都是多亏了楚公子。”
<br/>　　狂龙帝的脸上，都是露出了笑容，看向楚风眠，恭敬的说道。
<br/>　　“从今日起，老朽这一身老骨头，就为公子效力了，还请公子尽管吩咐。”
<br/>　　说着，狂龙帝便是向着楚风眠拜了下去。
<br/>　　这刚刚踏入了天人境，成就仙人的狂龙帝，在楚风眠的面前，却是如此的恭敬，行此大礼。
<br/>　　尹龙仙，海公主在一旁见状都是一幕，都有些想不明白这其中的门道。
<br/>　　“师尊，这是？”
<br/>　　尹龙仙有些惊异的说道。
<br/>　　狂龙帝现在踏入了天人境，可所谓是整个苍龙星域的霸主，未来横扫无敌，无人能敌，可现在居然对楚风眠如此恭敬，更是要为楚风眠效力。
<br/>　　让她一时间都是弄不明白。

<br/>　　他既然是曾经说过要为楚风眠效力，这个时候，也不会反悔。

<br/>　　楚风眠看着狂龙帝的表示，也是笑了笑，他也看得出来，以狂龙帝的性格，绝不会反悔。
