　　时间就在元始参悟道法，闭关修炼之中飞速流逝，转瞬已过三个元会，洪荒天地中，凶兽与众大神之间的战斗愈演愈烈，双方战得不可开交，天地间充满煞气。
<br/>　　同时，洪荒之中也崛起了一批神通广大之人，为对抗凶兽大劫增加实力。
<br/>　　在洪荒大战的同时，元始也修炼到了关键的时刻。
<br/>　　“要化形了吗？”元始从感悟中醒来，眼中闪过生死幻灭，宇宙重组、天雷地火等画面，只是一瞬间，就消失不见。经过三个元会的感悟修炼，元始的修为已经达到了身体承受的极致，能够看到，元始的元神清气从一开始只有几寸之高，到如今已有丈许之高，已经成长到了极限。他能感受得到，自己的化形劫就快要到了。
<br/>　　“终于要化形了，话说，没有一副好的身体，真是不习惯啊。我能感受得到，最多再有一百年，我就能化形了。好，很好！”元始激动道。
<br/>　　修为到了一定地步，不是靠积累就能进步的，想要达到质变，只有换一个更高级的身体，所以，只有经过化形，才能继续修炼。当然，也有一些妖兽，他们一直修炼本体，将本体当做是最强之道，并不屑于化形，他们只修本体神通，这也很强大，但这终究是有限制的。
<br/>　　相传，先天道体最适合修道。苍茫第一大神盘古就是先天道体，就连人族后来的创造也是依照盘古的先天道体而创造出来的。
<br/>　　先天道体也分级别，洪荒之中的先天魔神都是先天道体，这些道体可以追溯到混沌魔神之上，他们继承了混沌魔神的某些特征，修炼各种道，分别适合各种不同的道体。
<br/>　　如：元始的混沌魔神真身就适合修炼因果大道，魔祖的千手千眼圣魔真身适合修炼魔道，扬眉的空心杨柳适合修炼空间法则……但，在这些魔神之中，只有盘古的先天道体最适合修炼各种大道，所以，盘古才是三千魔神之中最强大的。
<br/>　　先天道体之中，威力最强的是盘古的混沌最高级别的混沌先天道体，盘古为混沌青莲孕育造化而生，是混沌最强，为混沌之子。之后才是各混沌魔神的魔神真身，他们的道体虽没有盘古的强大，但那也是最适合他们所修之道的道体。之后是洪荒天地现如今的先天道体。
<br/>　　元始既然要修炼众道合流的元始大道，那么，他化形之后的先天道体也早已注定，那就是参考盘古的混沌先天道体而出的先天道体。
<br/>　　人体阴阳二脉代表人的阴阳，道体负阴抱阳，阴阳合一。三大丹田，紫府、檀中，丹田是人体的天地人三才。双手双脚四肢代表着四象地水火风。五脏六腑代表五行六壬，头部七星，八脉对八卦，十二正经对应时辰年月，人体二十四节脊柱对二十四节气，三百六十五祖穴对应远古周天三百六十五大主星，人体一共有十二万九千六百大小经脉与十二万九千六百大**窍，对应一元会。整个先天人体，对应一个小宇宙。
<br/>　　这样的先天道体，在洪荒之中，最为契合天地万物，这样的道体，不论任何环境都适合生存。
<br/>　　经过三个元会的修炼，元始的元神身体早已按照这样的先天道体成长了，现如今只要化形而出，他就会拥有一具真正的先天道体，而不是只有元神的先天道体。
<br/>　　现如今元始要考虑的已经不是化形后的道体，而是在准备应付化形时的天劫。修行本是逆天而行，故而，修行总是伴随随着种种劫难，度过了，从此海阔天空，度不过，那就化作灰灰了去。
<br/>　　化形之劫会根据渡劫者的资质的好坏与修为的高低，还有身前所缠因果等，来判断天劫的大小。元始为盘古元神融先天玉清之气而成，天资惊人，如今的修为也已经堪比大罗金仙了，化形时天劫估计很是惊人，所以，他需要多做一些准备工作。他倒不是担心自己度不过去，而是在考虑，怎样借助此次天劫将自己修为提高到最大限度。
<br/>　　……
<br/>　　眨眼间，百年时间已匆匆而过，岁月无声无息，带走一切。
<br/>　　这一日，元始正在修炼元神，如今他的《神元九转》已经修炼到了第四转融术，一身元神修为强大到可怕。蓦然，他心有所感，却是化形的日子到了，在之前他不断压制，要将基础打炼到最强最牢，但化形之劫今日还是来了。元始心中不免闪过一丝兴奋，迎着天劫向山谷之上飞去。
<br/>　　这一霎那，天地轰鸣，整个昆仑山之上掀起了灵气风暴，无尽灵光充斥天地四方，无尽狂风迭起，元始站立虚空，倚然独立，任他东南西北风，岿然不动。
<br/>　　“轰隆隆隆——！”
<br/>　　顿时，在那虚空之巅，苍穹之上，乾坤倒转，日月失色，八方风云倒卷，一股恐怖的雷霆之力，直接出现，尽显天道威严。
<br/>　　这是属于元始的化形之劫，这劫，只能元始自己度过，谁也不能帮忙。这劫，不同于一般的雷劫，赫然是金色，透露着强悍的威严高贵，化作一条条金色雷龙，直奔元始的元神，轰轰而来。那是一股毁天灭地的威严，惊天动地！
<br/>　　“哈哈哈，化形之劫，那又如何，想让我屈服，不可能，来吧，我何惧之有？”元神震动，传出元始的狂笑，透着一股豪迈，迎着天劫冲去。
<br/>　　元始的意志何其惊人，在混沌之中走过无数岁月，体会无尽生死，能与盘古拼杀的人，又岂会惧怕这所谓的化形天劫，此刻他冲天而起，爆发出无尽威严，压向天劫。
<br/>　　一时间，昆仑山谷之上，化形天劫，惊天动地！
<br/>　　冲天的元神，闪烁着奇异的光芒，蕴含着惊天动地的力量，仿佛能够撼动苍天！
<br/>　　“轰隆！”
<br/>　　天劫仿佛被元始这种大胆行为激怒了，威力倍增，雷龙数量再次增加，咆哮着向元始的神魂冲来，势要将他毁灭！
<br/>　　“杀！”元始看似弱小的元神之体冲进狂暴的雷霆之中。
<br/>　　“大吞噬术！给我吞！”元始大叫，带着一股凶威，身边出现一个巨大的黑洞，它缓缓旋转，将元神周围的雷霆吞噬一空，之后，化为一个稍显平静的精纯雷液流入元始元神之中，滋养着他的元神，使之更加坚韧刚阳。
