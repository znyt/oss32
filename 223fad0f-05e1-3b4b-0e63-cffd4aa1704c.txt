        高中同班同学……
<br/>
<br/>        初恋……
<br/>
<br/>        这些都已经逝去了……
<br/>
<br/>        离他越来越远……
<br/>
<br/>        十八岁那年，季思妍离开后，傅昼景就再也没去回忆过那些事情，也从来不和高中时期的任何同学或是老师联系过。
<br/>
<br/>        在遇到童筱颖之前，他唯一所做的，就是用学习和工作麻痹自己。
<br/>
<br/>        自己是不受宠的私生子，自己爱过的女人又抛弃了自己，他一直以为只有自己才是这世界上最孤独的那个人，却没想到，还有筱颖和他同命相连。
<br/>
<br/>        筱颖的闯入，真的是一道美丽的意外。
<br/>
<br/>        起初在飞机上，他有点儿晕机，这会儿又坐上了小轿车，不禁困意袭来。
<br/>
<br/>        季思妍一边开车，一边瞥着副驾驶座上傅昼景的动静，见他似乎昏昏欲睡，于是主动把车内的摇滚乐换成了轻音乐。
<br/>
<br/>        很快，傅昼景便睡了过去。
<br/>
<br/>        等到他醒来的时候，季思妍刚好将车停好。
<br/>
<br/>        “醒呢？”季思妍一边说着，一边从车内收纳盒内掏出了两个包好的红包。
<br/>
<br/>        傅昼景四下看了看，见季思妍正将其中一个红包递给自己，不禁纳闷地反问：“你这是做什么？”
<br/>
<br/>        “你这是有多久没关注过高中的班级群消息了？今天是我们的高中班主任五十岁寿辰，在酒店里备了晚宴，特意邀请了我们这届他最喜欢的班的同学来参加他的寿辰。”季思妍微笑着说。
<br/>
<br/>        不然，她也不会突然跑去机场顺路接上他了。
<br/>
<br/>        傅昼景一脸迷茫，但还是信了季思妍的话，接过季思妍手里的红包后，才发现里面包了五千元现金。
<br/>
<br/>        “嗯，回头我还你。”傅昼景淡淡地说。
<br/>
<br/>        季思妍莞尔一笑：“就当我是拜师的学费吧！”
<br/>
<br/>        言下之意，她是不需要他还了，毕竟她知道，他最近手头有点儿紧。
<br/>
<br/>        傅昼景知道她季思妍不缺这点钱，她堂堂季氏集团的大小姐，怎会在乎这点儿钱。
<br/>
<br/>        反过来值得讽刺的是，他傅昼景如今连这点儿钱都拿不出来了。
<br/>
<br/>        每个月的工资，要供银行贷款，还要养筱颖，所以，他过得很窘迫。
<br/>
<br/>        傅昼景不再吭声，季思妍也知道他自尊心强，同样不再多说其余的废话。
<br/>
<br/>        两人一同下了车，进了酒店大门。
<br/>
<br/>        班主任的寿辰晚宴上，宴请的都是他曾经带过的班级里的学生。
<br/>
<br/>        傅昼景和季思妍一同出现在自己班的宴席桌前，其他同学见，一个个又开始起哄了。
<br/>
<br/>        “昼景、思妍，你们两什么时候请我们大家吃喜酒啊？”
<br/>
<br/>        “是啊！高中时期，你们两简直就是天造地设的一对！”
<br/>
<br/>        “就是，郎才女貌！没想到，你们这么久了还能在一起！”
<br/>
<br/>        大家七嘴八舌，无不是对他俩的祝福。
<br/>
<br/>        傅昼景刚想开口解释。
<br/>
<br/>        季思妍却抢先一步，微笑道：“那你们到时候可要准备大红包噢！红包不够大，我和阿景是不会收的！”
<br/>
<br/>        “那肯定的啊！”
<br/>
<br/>        “就是就是！红包小意思，最大的意思，就是想看咱们班的这对金童玉女，早生贵子！”
<br/>
<br/>        “你们就先别拿我和阿景打趣了，来来来，说说你们的事情啊！”季思妍笑容灿烂地转开话题。
<br/>
<br/>        以前在班上，季思妍的人缘特别好，和谁都能聊得上来。
<br/>
<br/>        而傅昼景在班上属于班草级别的人物，受女生欢迎不说，和男生的关系相处得也挺不错。
<br/>
<br/>        所以，他俩的结合，大家都是纷纷送祝福。
<br/>
<br/>        傅昼景坐在季思妍的身边，仍旧默不吭声。
<br/>
<br/>        就连班主任下席来端着酒杯回敬大家酒的时候，也忍不住当着大家的面调侃傅昼景和季思妍。
<br/>
<br/>        “你们两个读高中的时候就已经眉来眼去了，老师我是看在眼里的。好在，你们两都是好孩子，没有因为彼此喜欢而影响到成绩。一个考上了咱们滨江大学，一个又去了国外留学深造。如今，老师也期待喝你们两的喜酒！”班主任是个男老师，本身就喝了酒，带着微醺的醉意，就开始满嘴跑火车。
<br/>
<br/>        听着班主任这话，傅昼景脸上没有任何表情。
<br/>
<br/>        季思妍则一直向大家陪着笑脸，特别是在班主任说完这些话后，一个劲地向班主任道谢：“李老师，我得谢谢您当年的睁只眼闭只眼，没采取喊家长的强制手段拆散我和阿景。”
<br/>
<br/>        “我也祝你们所有人，有情人终成眷属，事业蒸蒸日上，家庭幸福美满。”班主任笑不拢嘴地举起了酒杯。
<br/>
<br/>        大家一一和班主任碰杯，祝班主任长命百岁。
<br/>
<br/>        傅昼景却很不喜欢这种感觉，全过程，他都被季思妍牵着鼻子走，却又不得不顾及季思妍的面子，配合她的谎言。
<br/>
<br/>        时间一分一秒的过去……
<br/>
<br/>        桌上的饭菜由最初的热气腾腾，到慢慢冷却，童筱颖端起凉掉的饭菜热了一遍又一遍，电话也打了一通又一通。
<br/>
<br/>        最终，电话始终都没打通，桌上的饭菜，童筱颖一口未动全倒进了垃圾桶中，然后她又去重新做了一份。
<br/>
<br/>        晚上十点，童筱颖的肚子饿得咕咕叫，此时，傅昼景的手机仍旧处于关机状态。
<br/>
<br/>        会不会是一下飞机就去公司加班了，所以把手机忘记开机呢？
<br/>
<br/>        童筱颖心里揣测着，于是拿起钥匙和手机，换了鞋去了一趟地利置业大厦。
<br/>
<br/>        在去的途中，童筱颖疾步如飞，心里脑海里想着的人全是傅昼景。
<br/>
<br/>        她气喘吁吁地跑到了地利置业大厦的大门前，却发现大厅的正门居然被从内部上了锁。
<br/>
<br/>        童筱颖不得不又跑到大厦前坪，看到大厦里没有一层的灯是亮着的，顿时有点点失落。
<br/>
<br/>        昼景不在公司的话，那又会去哪里？
<br/>
<br/>        童筱颖第一次发现，她好像一点都不了解昼景，就连他朋友和助理的电话号码，她都没有。
<br/>
<br/>        这关键时刻，她不知道该向谁求助。
<br/>
<br/>        那么……
<br/>
<br/>        会不会是飞机晚点呢？
<br/>
<br/>        童筱颖又抱着这样的侥幸心理，搭了一辆出租车，赶去了机场。
<br/>
<br/>        令她觉得可笑的是，她连他坐那趟航班回来的都不知道。
<br/>
<br/>        机场出口，童筱颖看着人来人往，心提到了嗓子眼。
<br/>
<br/>        她只希望，昼景快点出现在她面前。
