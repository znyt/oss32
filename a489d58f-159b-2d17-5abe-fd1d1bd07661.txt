孔真儿的家境并不好，但是她本人贪慕虚荣，跟着夏小蝶，夏小蝶会给她一些好处，比如夏小蝶用旧了的名牌包包还有衣裙，夏小蝶也会带她来1949这种高档酒吧来喝很贵的酒，这种纸醉金迷的生活就是她一直奢望的。
<br/>
<br/>        夏夕绾说她是夏小蝶身边的一条哈巴狗，这话完全正确。
<br/>
<br/>        但是孔真儿并不希望别人这样说她，她知道夏小蝶是一个无脑的草包，心里更是深深的艳羡妒忌着夏小蝶所有的好运，她并不喜欢夏小蝶。
<br/>
<br/>        还有夏夕绾，她更讨厌夏夕绾，在她看来夏夕绾这个土包子应该比她更低贱，但是夏夕绾将自己活得那样清丽璀璨。
<br/>
<br/>        孔真儿偷喝了两杯很贵的酒，就在她细细品尝酒味的干烈时，苏希走了进来。
<br/>
<br/>        看见苏希，孔真儿迅速起身，“苏…苏少，你怎么来了？”
<br/>
<br/>        苏希看了一眼包厢，并没有看到约他来这里的夏夕绾，他的目光落在了孔真儿的脸上，“夏夕绾人呢？”
<br/>
<br/>        孔真儿简直不敢直视苏希的俊颜，每个灰姑娘都会幻想有一天她的白马王子到来，爱上她，并让她成为人人艳羡的公主。
<br/>
<br/>        四大豪门之一的苏希俊逸温润，就是那个白马王子。
<br/>
<br/>        每当苏希跟夏小蝶在一起的时候，孔真儿就幻想过自己能替代夏小蝶。
<br/>
<br/>        孔真儿喜欢苏希。
<br/>
<br/>        这是第一次两个人独处，孔真儿的心跳几乎要跳出胸膛了，她水汪汪看着苏希，“苏少，夏夕绾刚才跟小蝶出去了。”
<br/>
<br/>        苏希蹙了一下剑眉，坐在了沙发上，“那我在这里等。”
<br/>
<br/>        苏希没有再看孔真儿，而是拿出了兜里的手机。
<br/>
<br/>        孔真儿一颗芳心砰砰乱动，她爱慕的看着苏希的俊脸，然后手忙脚乱的给苏希倒了一杯红酒，“苏少，喝杯酒吧。”
<br/>
<br/>        孔真儿走过去，将酒杯递给苏希。
<br/>
<br/>        但是她有些醉了，再加上自己喜欢的人就在面前，所以她脚一崴，整个人扑进了苏希的怀里。
<br/>
<br/>        ……
<br/>
<br/>        夏小蝶跟着夏夕绾出来了，但是夏夕绾什么都没有跟她说，只是自顾自的走着。
<br/>
<br/>        “夏夕绾，你究竟想说些什么？”夏小蝶没了耐心。
<br/>
<br/>        夏夕绾来到洗手间的门口，勾唇一笑，“我只是想邀请你跟我一起上洗手间。”
<br/>
<br/>        夏小蝶一愣，知道自己被夏夕绾给耍了，她当即骂道，“夏夕绾，你脑子有病吧！”
<br/>
<br/>        夏小蝶转身就回了包厢。
<br/>
<br/>        夏夕绾看着夏小蝶怒气冲冲的身影，挑了挑精致的柳叶眉，不知道孔真儿会不会让她失望呢？
<br/>
<br/>        ……
<br/>
<br/>        夏小蝶推开包厢门走了进去，“真儿，我…”
<br/>
<br/>        下一秒夏小蝶的声音就没了，因为孔真儿扑进了苏希的怀里，正痴痴的看着苏希的俊脸。
<br/>
<br/>        这一幕无比的暧昧。
<br/>
<br/>        夏小蝶一怔，一把怒火已经焚烧了她的大脑，她当即质问道，“孔真儿，你在干什么？”
<br/>
<br/>        骤然听到夏小蝶的声音，孔真儿所有的旖旎都被吓得烟消云散了，她迅速起身，一下子就撞上了夏小蝶那可怕的双眼。
