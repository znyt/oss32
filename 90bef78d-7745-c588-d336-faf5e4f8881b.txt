翌日清晨。
<br/>
<br/>        T大的学校门口，一辆劳斯莱斯幻影豪车缓缓停了下来，副驾驶座上的厉嫣然一脸娇羞甜蜜的看向驾驶座上的男人，“寒霆哥哥，我到了，谢谢你送我来学校。”
<br/>
<br/>        今天陆寒霆穿了手工版的白色衬衫黑西裤，经典男神的穿搭，衬的他英俊精致，卓尔不凡，戴着名贵腕表的大手随意搭在方向盘上，他温和的看着厉嫣然，“好好学习。”
<br/>
<br/>        “恩，那我下去啦。”厉嫣然打开了车门。
<br/>
<br/>        现在正是上学的高峰期，厉嫣然是T大的校花，本来就是全校瞩目的焦点，她一下车，很多女同学围了过来，
<br/>
<br/>        嫣然，你来上学啦。
<br/>
<br/>        嫣然，今天是谁送你来学校的啊？
<br/>
<br/>        厉嫣然一般都是坐着厉家的专属私家车来学校的，今天突然换成了劳斯莱斯幻影豪车，这辆世界级的豪车贵到令人咂舌，车屁股后面挂着的车牌也是吊炸天那种，那些女同学们偷偷往车里看了一眼，当即看到了男人精致英俊的侧颜。
<br/>
<br/>        呀嫣然，今天是陆总送你来学校的啊。
<br/>
<br/>        嫣然，你和陆总也太甜蜜了吧，这是要公开了吗？
<br/>
<br/>        最近帝都一直在传陆厉联姻，但是陆寒霆并没有在公开场合表过态，他都在公司忙，跟厉嫣然相处的机会并不多。
<br/>
<br/>        今天他愿意来送她上学，厉嫣然心里十分惊喜的，开着世界级豪车将她送到T大门口，让全世界知道彼此的存在，这已经是很高调的公开和表态了。
<br/>
<br/>        厉嫣然甜蜜蜜的看向陆寒霆，然后突然弯了腰，整个人钻进去，凑到陆寒霆的俊颊上就亲了一口。
<br/>
<br/>        哇哇哇！
<br/>
<br/>        周围的女同学们都炸开了。
<br/>
<br/>        厉嫣然漂亮夺目的小脸红红的，刚才她就想给他一个good-bye-kiss了，“寒霆哥哥，那我真走了。”
<br/>
<br/>        陆寒霆的俊脸上并没有什么情绪波澜，他点了点头，“恩。”
<br/>
<br/>        “那…那你下午能来接我放学吗？”厉嫣然期待又忐忑的看着他。
<br/>
<br/>        陆寒霆想了一下，“好。”
<br/>
<br/>        这一下厉嫣然心花怒放，“寒霆哥哥，那我们下午见，88。”
<br/>
<br/>        厉嫣然关上了车门，在一群女同学的簇拥下走进了T大。
<br/>
<br/>        陆寒霆收回目光，劳斯莱斯幻影豪车疾驰而去。
<br/>
<br/>        此时，夏家的豪车从迎面疾驰而来，两辆车擦身而过。
<br/>
<br/>        后座的夏夕绾透过蹭亮的玻璃车窗一下子就看到了那辆熟悉的座驾，陆寒霆的座驾一直没有变过，依然是劳斯莱斯幻影。
<br/>
<br/>        她的心脏蓦然一缩，这时只见劳斯莱斯幻影的驾驶车窗缓缓滑下了一点，男人那张熟悉的俊颜就这样闯入了眼底。
<br/>
<br/>        但是只不过是一瞬间，劳斯莱斯幻影就疾驰走了。
<br/>
<br/>        夏夕绾迅速出声，“夏爸，停车，快停车！”
<br/>
<br/>        夏邦当即踩下了刹车，路边停车。
<br/>
<br/>        “绾绾，你怎么了，发生什么事情了？”蓝烟关心的问。
<br/>
<br/>        夏夕绾来不及说话，她迅速拉开了后车门，跑了出去，拔腿就去追那辆劳斯莱斯幻影。
<br/>
<br/>        她追了一路，两条腿注定是追不上四个轮子的，劳斯莱斯幻影在前面的路口打了方向灯，准备左转了。
