　　第177章
<br/>　　“这帮没良心的畜生！”
<br/>　　程大雷掌中斧，腰中剑，骑着瘦马立在城头之下，身上锦袍玉带，看上去杀气凛凛，威风赫赫。
<br/>　　但自家人知道自家事，程大雷的本事也就三斧头，真如果打起来，一招不慎就是掉脑袋的事情。想自己还没有领略帝国大好山河，还没有和帝国各地的美女亲密接触，如此一命呜呼，岂不遗憾。
<br/>　　不过，程大雷也不是没有半点底气，他手中快剑设定就是百分百割喉。程大雷也想清楚了，能打的话自己就耍上一剑，耍完就溜。城门后，由秦蛮和赵子龙率领的骑兵随时接应。
<br/>　　想明白后，程大雷深吸一口气，冲着戎族大喊一声。
<br/>　　“戎族的渣渣们，那个来爷爷斧前受死！”
<br/>　　从戎族大营中，一个衣衫褴褛，表情麻木的汉子走出来。是步行，双手缠着铁链，身上伤痕累累。
<br/>　　赤眉将一柄枪丢在地上，道：“姓林的，杀了他，我便放了你。”
<br/>　　姓林……那就意味着不是蛮族。蛮族的名字很奇怪，有姓的大部分都是贵族，普通人取名字的时候特别随意。比如孩子生在大树下，就叫个『大木』，生在泥水里，就叫『黑石』，纵然有些崇尚帝国文化的蛮人会取一个武朝的名字，可若真是戎族人，该不会这么惨。
<br/>　　程大雷一怔，莫非是他。
<br/>　　他忙集中注意力在这汉子身上，瞬间眼睛一亮。
<br/>　　姓名：林冲（默默无闻的顶级武将）
<br/>　　年龄：33
<br/>　　技能：破浪七叠枪
<br/>　　隐藏属性：无
<br/>　　啊，我的林教头。
<br/>　　程大雷发出一声咏叹，难以抒发心中的慷慨激情。林教头果然还是来了，只不过却是以这种方式，现在落在戎族手中饱受虐待。
<br/>　　不，自己一定要救他脱离苦海。
<br/>　　林冲将地上的枪捡起来，纵然对阵，他手上的镣铐也没有被解开，可见戎族对他的忌惮。
<br/>　　程大雷看双方的距离，此时若让秦蛮冲出来，双方必然发生厮杀，到时候未必能救得林冲，反而城门有失守的危险。
<br/>　　林冲没有骑马，立在地下，抱枪行礼。
<br/>　　“抱歉，我还有使命未完成，今日得罪了。”
<br/>　　得罪的意思就是要杀自己，程大雷撇撇嘴，心道：纵然你本事高强，可伤成这个样子，不至于还看不起我吧。
<br/>　　一枪朝天式，向程大雷逼来。来得好快，程大雷差点着了道，顿时收起给林教头一个下马威的意思。
<br/>　　“林教头，是我，我是程大雷。”借着插招换式的缝隙程大雷道。
<br/>　　“我不是林教头，也不认识你，抱歉。”一招中平枪，掠着程大雷头顶飞过。
<br/>　　程大雷吓得一跌，情况不对啊，林教头明明是自己召唤出来的，难道不该像秦蛮那样纳头便拜么。系统这次的身份设定究竟是什么……
<br/>　　“难道你就甘心做戎族走狗，杀戮同胞！”程大雷只有招架之力，而无还手之功。
<br/>　　林冲脸涨得通红，枪势就缓了下来。
<br/>　　“我家公子流落此地，下落不明，我必须活着带他回去，抱歉。”
<br/>　　“你家公子是……”
<br/>　　“正是那相貌堂堂，武功高强的……林……少……羽！”
<br/>　　一枪直刺，逼得程大雷节节败退，他终于撑不住了，此刻林冲的枪已到眉心。
<br/>　　“住，我认得他！”
<br/>　　“嗯？”林冲停枪。
<br/>　　“少羽就在我山寨，此时与我是结义兄弟，少羽的大哥就是我的大哥，林大哥。”
<br/>　　“这个……我是他叔叔。”
<br/>　　“呃，没想到你这样的人也会占便宜。”
<br/>　　程大雷看看周围，道：“林教头，今日没办法救你，我先回去，一两天必然救你出来。”
<br/>　　“你该不会诓我吧？”
<br/>　　要不要这么警惕！程大雷无语，道：“少羽胸口有块红色胎记，现在你信我不信！”
<br/>　　“啊，这么隐秘的事情你怎么知道，难道少羽他有了龙阳之好……”林冲打量着程大雷：“眼光这么差！”
<br/>　　“我香蕉你个吧啦！”
<br/>　　程大雷心中忽然拨转马头，向城门飞奔，边跑边大声喊着。
<br/>　　“那贼，今日你我不分胜败，等本将军回去填饱肚子，改日再战！”
<br/>　　不分胜败……明明是人家把你打得抬不起头好不好。
<br/>　　对阵双方，心底同时发出一声感慨。
<br/>　　我从未见过如此厚颜无耻之人。
<br/>　　“关门，关门。”
<br/>　　程大雷马不停蹄的回到蛤蟆寨，将各家头领聚到一起。
<br/>　　“大当家，打得好啊，只看得我眼花缭乱，再有十几个回合，大当家必杀了他，大当家怎么不继续打下去了？”徐神机。
<br/>　　程大雷白了他一眼，心道：取笑我有意思么。
<br/>　　“那人是一个我失散多年的朋友，如今陷在戎族大营中，请诸位过来，是看怎么把他救出来，诸位有没有什么法子？”
<br/>　　“失散多年的朋友，大当家你今年才多大，失散多少年了？”徐神机。
<br/>　　程大雷有些不想理他。
<br/>　　“在戎族大营，想救他出来怕是难如登天。大当家，您这位朋友重要么？”刘悲。
<br/>　　“无比重要！”
<br/>　　程大雷坚定道，没有林冲就没办法升级山寨，没办法升级山寨就没有附加属性。
<br/>　　“那么只好大举进攻，与他们展开白刃战，趁机把大当家的朋友抢回来。”秦蛮。
<br/>　　程大雷摇摇头，秦蛮的智商也就和徐神机相差无几，真如果白刃战，山寨的山贼根本不是戎族的对手。
<br/>　　“不若由几个高手趁夜偷营，把人从敌人那里偷回来。”刘悲道。
<br/>　　程大雷想了想，这或许真的是一个办法，戎族知道己方不会出城，防守未必森严。由秦蛮、赵子龙、高飞虎、小白狼等几个好手行动，成功的概率还是很大的。
<br/>　　只是，这一切都是程大雷的想当然，毕竟对方大营的情况自己摸不清楚。万一救不出来林冲，再折上几个，便得不偿失了。
<br/>　　众人又胡乱出了很多注意，类似于挖个地道什么的，自然而然都被程大雷否了。
<br/>　　“大家想，继续想！不计代价，不计成本，接下来我们的任务就是怎么把这个人救出来，我把这次任务代号命名为……拯救大兵林冲。”
<br/>　　
<br/>(本章完)
