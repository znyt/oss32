　　田不易听到有人大喊小徒弟的名字，就知道苏重又闯了祸。
<br/>　　架起遁光立刻朝着事发地点飞去。
<br/>　　可刚飞没一会儿，就看到迎面飞来一道白色遁光。其后跟着五六个人。只看那遁光驳杂色彩，就知道不是好路数。
<br/>　　“师傅快跑啊！他们人多！”苏重迎着田不易大喊。
<br/>　　田不易脸皮抽动，你当我不知道要跑吗？五六个和他同级别的存在，一头撞上去，他也得凉啊！
<br/>　　“闪开！”田不易一声大喝，一道火红剑光轰然斩出。
<br/>　　苏重遁光猛然偏转，红光擦着身边飞过。灼热气息让苏重一阵口感舌燥。心里啧啧称奇，老田这是拿出真本事啦！
<br/>　　身后五六个魔门之人，猛然迎来剑光冲击。其中一个倒霉家伙正好挨上，一下就给从天上劈了下来。
<br/>　　就在几人惊疑不定迟疑之时，田不易调转剑光，伸手拉过宋大仁，一溜烟的的跑了……
<br/>　　放完大招就跑，真刺激！苏重哈哈大笑，拐了个弯也跑了。
<br/>　　苏重遁光迅捷，田不易法力深厚。追踪之人只是迟疑片刻，就再也找不到两人踪迹。
<br/>　　一连飞出数百里，苏重落下遁光。然后隐匿身形，又悄然摸向河阳城。
<br/>　　他也不进城，就躲在城外。
<br/>　　“六尾，这次能不能给他们一个狠的，就看你的了！”苏重这是要杀个回马枪！
<br/>　　经过刚才那么一闹，魔门想再藏在河阳城内已经不可能。
<br/>　　“来了！”六尾闭着的眼睛突然睁开道。
<br/>　　苏重连忙看过去，果然有五六个黑衣人鬼鬼祟祟的从城墙上跳下来。
<br/>　　“和我猜测的一样，这些魔门的人，不敢直接驾驭遁光飞走，只能分散开悄悄逃离！”苏重嘿然一笑，气息悄然消失不见。
<br/>　　他竟一脚踩进万人往的陷阱，要不是他修炼凶戾气，说不定就真被缚灵网抓住，这会儿已经任人宰割了！他可不是被打了还不还手的性子。
<br/>　　这五六个人很紧张，在野外快速行进。
<br/>　　“这次到底是怎么回事，我们藏的好好的，为什么通知我们撤离。”队伍中的人轻声抱怨。
<br/>　　“据说是我们的据点被一个青云弟子发现了，计划失败，我们只能撤离。”他旁边的人小声回道。
<br/>　　“要我说，逃什么逃，就直接在河阳城开战。那些青云的人不是喜欢保护普通人吗？有这顾忌，咱们一下给他们来个狠的不就完了！”一个矮粗身材男人狠声道。
<br/>　　“还有那小子，叫什么来着？李二蛋！哈哈……谁会叫这个名字！”矮粗身材男子嘿嘿笑。
<br/>　　“你听错了，那不是李二蛋，是李尔旦！”
<br/>　　“那还不是一样，哎，你特么是谁！”
<br/>　　不知道什么时候，他们身后竟然多了一个人，竟然谁都没发现！
<br/>　　“我特么是你爸爸！”苏重一拳就砸在这个矮胖子脖颈上，直接给打折掉，死的不能再死。竟然喊他李二蛋，这种家伙还能留着？
<br/>　　“敌袭！”领头人一声大喝。剩下的人迅速后退，一边后退一边手掐法诀，准备召唤法宝攻击。
<br/>　　苏重怎么可能让他们从容进攻，他就站在原地，笑眯眯看着。
<br/>　　黑夜中一道白色细线蜿蜒飞舞，瞬间穿过众人身旁。
<br/>　　噗嗤！
<br/>　　一声异响，空气中血腥气弥漫。几人眉心几乎同时出现一个孔洞，全都被剑丸穿脑而过。
<br/>　　随着修炼时间越长，剑丸越发犀利迅捷。而且由于被融入一颗道种，更加收发由心。
<br/>　　就像现在，在别人还在掐诀沟通法宝的时候，苏重就已经催动剑丸收割敌人性命。
<br/>　　“六尾！洗地！”苏重淡淡道。
<br/>　　地面泥土顿时软化成泥，慢慢把几具尸体包裹，拉入地下消失不见。
<br/>　　然后苏重拉着六尾快速离开。河阳城很大，魔门分开撤离，不可能也不会只走一条路。
<br/>　　他就带着六尾再河阳城周围转悠，半夜三更，凡是遇到的鬼鬼祟祟的家伙，肯定都是魔门的人。
<br/>　　果然，没过多久就有十多个人翻过河阳城，朝着黑漆漆大山快速奔跑。等到他们走近，苏重一下跳出来：“此山是我开，此树是我栽！要想从此过，留下小命来！”
<br/>　　十多个人直接就蒙了，你半夜三更出来抢劫就不说你了。还留下小命来？这是直接不给活路啊！
<br/>　　“哪来的蠢贼，杀了他，我们快走！”零头之人冷哼一声，偏转方向立刻就往林子里钻，就连动手杀苏重都嫌累的慌。
<br/>　　几个人人一溜烟穿过苏重身边，最后一人笑嘻嘻靠近苏重：“大半夜都不忘出山打劫，这么敬业简直就是我辈楷模，下辈子投胎来我们鬼王宗，你肯定能出人头地！”
<br/>　　“不去，出了头也得掉地上！”一道白光闪烁成丝，从背后追上十多人，绕着脖颈一转。
<br/>　　噗通噗通……十多个人头滚落在地。
<br/>　　最后一人脸都白了！
<br/>　　“你看，我说过的，要想从此过，留下小命来，谁让他们不听来着？”
<br/>　　“我没过去！”噗通一声，人头落地。
<br/>　　苏重往前迈了一步，越过此人：“现在你不就过去了。六尾，出来洗地啦！”苏重吆喝一声，再次转移战场。
<br/>　　直到天光微明，苏重才带着六尾回山。
<br/>　　刚走进竹屋，迎面就碰上了怒气冲冲的田不易。看着自家小徒弟身上气息，田不易倒吸一口凉气。这一身的煞气，简直比魔门弟子还像魔门弟子！
<br/>　　“徒弟，你跟我说实话，庚金剑气真的没问题？”刚才还满腔怒火，现在却全是担忧。
<br/>　　他不用问都知道苏重这一夜干什么去了。他自己都带着其他几峰长老打了个回马枪，斩落了两个魔门实力长老。怎么还不明白，自家小徒弟和自己一样，反头又杀了回去。
<br/>　　只是这一身透体杀意，到底是杀了多少人？
<br/>　　苏重低头看了看身上衣服，黑色衣服干净如新没有丝毫血迹。在看田不易神情，就知道肯定是杀戮太过，引起田不易注意。
<br/>　　“师傅放心，我清楚我在干什么。”苏重镇定道。
<br/>　　他闯荡位面数百年，心志坚定，一夜杀戮还动不了他的心。
<br/>　　“这段时间，你就老老实实的给我呆在山上吧！”田不易下了禁足令。
<br/>　　弟子太天才也不是好事，误入歧途的妖孽弟子已经不止一个。他不希望苏重成为下一个。
<br/>　　苏重笑着接受，最近他也没打算再搞什么幺蛾子。昨天晚上那么一闹，什么魔门诡计都不可能成功了。
<br/>　　一来青云做了防备，诛仙剑阵威力足以震慑天下。二来吗，昨天晚上魔门可是死伤惨重。
