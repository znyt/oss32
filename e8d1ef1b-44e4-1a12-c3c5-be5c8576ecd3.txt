　　楚风眠以雷霆之势灭杀了陈家众人，已经是在这一群北邙学院高层的心中，种下了恐惧。
<br/>　　如今看到楚风眠冰冷的目光，一个个都是瑟瑟发抖起来。
<br/>　　“楚风眠，你也是出身自北邙学院，今日的事，我们的确也是被逼无奈，无可奈何，你放了我们，我们愿意认你为主，帮你掌管北邙学院。”
<br/>　　那北邙学院院长看向楚风眠突然脸色一变开口道。
<br/>　　“你只是北邙学院的一名弟子，想要执掌北邙学院可并不容易，我们可以帮你，让你真正掌控北邙学院。”
<br/>　　“不错，北邙学院之中各个家族的势力也是错综复杂，院长的位置可没有那么好座的，你跟我们合作，你就可以执掌北邙学院。”
<br/>　　一名北邙学院的老祖听到这话，也是眼前一亮的对着楚风眠开口道。
<br/>　　“今日你如果杀了我们，你也只能是毁了北邙学院，不可能的得到！你回来北邙学院，不就是为了得到北邙学院吗？你跟我们合作，对谁都有好处。”
<br/>　　那北邙学院院长看着楚风眠一直没有说话，冷笑一声道。
<br/>　　“你们几个，也有跟我合作的资本？”
<br/>　　楚风眠眼神瞥了一眼，冰冷的开口道。
<br/>　　“跪下，打开道心，我可以饶你们一命！”
<br/>　　“你！楚风眠你不要太过分了！你今日难得真的想要鱼死网破？我们北邙学院，传承万年，什么风浪没有见过，我劝你还是……”
<br/>　　那阳皇刚刚开口道一半，突然间声音戛然而止。
<br/>　　一道龙爪，凌空轰击，这已经是成就帝尊的阳皇，也在这一爪之下，被凌空打爆，灰飞烟灭。
<br/>　　“还敢威胁我？”
<br/>　　楚风眠冰冷的开口道。
<br/>　　他可没有兴趣留这些人的性命，这些人早就背叛北邙学院了，留着他们也是一个祸害。
<br/>　　要是他们乖乖听话的话，楚风眠还可以留他们一命，毕竟他们都是北邙学院的高层，可以负责管理北邙学院。
<br/>　　可要是不听话的话，那这些人也没有存在的必要了。
<br/>　　“不臣服，就死！”
<br/>　　楚风眠眼神一横，扫了过去。
<br/>　　“楚风眠，你也是北邙学院的弟子，何苦闹得这么僵硬，你放了我们，也是结了一个善缘！你也许不知道，我们北邙学院可是有着大秘密，你敢杀了我们，你也得死！”
<br/>　　这北邙学院院长盯着楚风眠，突然间下了决定，冷声开口道。
<br/>　　“什么大秘密？”
<br/>　　楚风眠目光看了过去。
<br/>　　这北邙学院院长，却是默不吭声。
<br/>　　过了一下才开口道。
<br/>　　“你先放了我们！放了我们，我就告诉你！”
<br/>　　“轰！”
<br/>　　楚风眠眼神冰冷，听到这话，没有丝毫的犹豫，顿时运转起灵力来，猛然抓了过去，这北邙学院院长被直接捏在了手心之中。
<br/>　　“我没时间跟你们废话，臣服，或者死！”
<br/>　　楚风眠死死的盯着这北邙学院院长，眼神之中露出了几分杀意。
<br/>　　“你杀了我，你永远得不到这个秘密！”
<br/>　　北邙学院院长还心存侥幸，绝对会在跟他商量一二。
<br/>　　可是他的话音一落，突然间楚风眠手心灵力便是凝聚起来，挤压着他的身躯。
<br/>　　“既然你不愿意说，那你就去死吧！”
<br/>　　碰！
<br/>　　随着楚风眠手心一捏，这北邙学院院长当即被捏碎，化为了漫天血雾。
<br/>　　同时楚风眠运转起吞天秘术来，将他的一团精血，都给吞噬殆尽。
<br/>　　“啊！不，我是古帝，我怎么会死！”
<br/>　　这北邙学院院长凄惨的叫道，想要从楚风眠的手中逃出生天，可一切都是无济于事，一团精血，都被吞天秘术无情吞噬。
<br/>　　这北邙学院的院长，被楚风眠一下子轰杀，毫不留情。
<br/>　　既然不听话，那此人也没有存在的必要了。
<br/>　　至于所谓的秘密。
<br/>　　这北邙学院最大的秘密，楚风眠都知道，还有什么是楚风眠无法知道的，等到楚风眠掌控了北邙学院后，一切都可以查出来。
<br/>　　“院长！”
<br/>　　“小畜生，你不配做北邙学院的弟子！”
<br/>　　“当初我们就应该杀了你！”
<br/>　　“祸害！你才是祸害！”
<br/>　　看到北邙学院院长的死，阳皇等人，都是一脸悲愤大喊起来，他们不少人可都是当初很早的时代就跟随这院长的。
<br/>　　“看来，你们也不打算臣服了？那就都死吧！”
<br/>　　楚风眠脸色冰冷无比，手心一点，那些刚刚大喊的北邙学院老祖，阳皇，风皇等人，一个个身躯都是破碎。
<br/>　　他们的实力，可都是帝尊，在九域都算是大人物了，可在楚风眠的手里，就连蝼蚁都算不上。
<br/>　　仅仅是一个眼神，就足以将他们灭杀，一道道身躯凌空爆炸开来。
<br/>　　整个宫殿之内，都弥漫着血腥味。
<br/>　　楚风眠下手，从不留情，的确楚风眠是有着掌控北邙学院的心思，但是也不代表他会受到这些北邙学院高层的威胁。
<br/>　　杀了他们，楚风眠只是多花费一些手段罢了，但是留下他们，可都是祸害，未来楚风眠建立剑道门里，可不能有这样的祸害。
<br/>　　今日正好是大清洗一番。
<br/>　　整个宫殿之内，如今鸦雀无声，再也没有人敢开口，大部分的北邙学院高层，已经是全部陨落，神形俱灭。
<br/>　　在地上，只剩下两道身影，跪在了地上，如今在瑟瑟发抖。
<br/>　　“你们愿意臣服？”
<br/>　　楚风眠目光扫过，落入到这二人的身上，这二人，也是北邙学院的两位老祖级别的人物。
<br/>　　其中一人是武皇，而另一人则是北邙学院的邓灵副院长。
<br/>　　他们二人如今跪在地上，在刚刚都没有说话，已经是选择臣服了。

<br/>　　武皇，邓灵二人跪在地上听到的话，大声的喊道。
<br/>　　他们二人也被楚风眠刚刚的铁血手段给吓得半死了，楚风眠出手的确是太狠了，无所顾忌。
<br/>　　本来在北邙学院高层之中，认为可以跟楚风眠谈判合作的人，还不在少数。
<br/>　　毕竟他们才是北邙学院的高层，他们一死，北邙学院一定是会混乱一片的，不适合掌控。
<br/>　　陈家的人，也是因此，没有杀了他们。
<br/>　　可现在楚风眠的手段却是告诉他们，在楚风眠的面前，只有臣服，不然唯一的一条路，就是死。
