　　黑漆漆，灰蒙蒙，上没有天，下没有地，中间也没有人、神、佛、太阳、月亮、星辰，甚至没有岁月、时间的概念！
<br/>　　在这里，天也空、地也空，人也空，神也空，佛也空，日也空，月也空，星也空，时也空！
<br/>　　这里当然不是九空无界，而是著名的混沌虚空！在这里，除了混沌，就是混沌巨蛋，并没有别的东西，即便有别的东西，在混沌之中，也经受不住混沌气流的霸道冲刷，不空也得空！
<br/>　　元释发现自己仿佛正在做一个梦，梦中世界漆黑一片，空旷无垠，无边无际，找不到一丝光亮，没有一丝声音，有的只是无边的黑暗，无边的寂寞，但他却没有一丝不舒服，他好像徜徉在大海之中，无边无涯，乘着一条船，舒适的晒这太阳。他感觉自己的神魂一阵舒爽。
<br/>　　“呃——！”元释睁开眼，却开不见丝毫光亮。他摸了摸自己的身体，却仿佛一片虚无，什么也触摸不到。
<br/>　　“果然，自己现在只是灵魂状态！”元释不出所料，但语气中还是掩饰不住的失落。自己到底是死了？还是死了呢？他想起前世的种种，这一刻，仿佛过眼云烟，过去的种种，全都烟消云散。
<br/>　　经过初始的不适之后，元释的灵魂仿佛站在虚空之上，开始私下张望，看到的却是无尽的幽暗。没有想象中的无尽鬼魂，没有牛头马面，更没有黄泉河、奈何桥、望乡台、三生石……
<br/>　　“这是哪里？有没有人？”元释陡然大喊起来，四周的黑暗与未知，让他心里发慌，整个人的灵魂，看起来颤抖不停，仿佛下一刻就要消散。
<br/>　　“到底是怎么回事儿？我已经死了？变成了灵魂，这里又是哪里？我又是怎么会来到这里？本以为这次苏醒，可以轮回转世，完成上一世渡劫飞升的愿望。但，如今又是何种情况？”经过长时间的呼喊之后，元释沉默下来，整个人陷入了深思之中。
<br/>　　这种情况，对他来说，是未知的，他开始在黑暗中游荡寻找，希望可以寻找到一丝光明，想要看看，这里到底是什么地方？如此，不知过了多久，他的目光，从初始的期盼，渐渐地烦躁，慢慢呆滞，甚至快要接近绝望了。他修炼的魔功已经开始反噬自己的灵魂了。
<br/>　　正在此时，突然间，灵魂内一块白玉色的圆形印玺，发出一片耀眼的白光！
<br/>　　这一刻，已经走到绝望悬崖边的元释陡然间停住脚步，一下子清醒过来。
<br/>　　“咦！这不是那块我一直没有看透的元始印吗？它怎么会在我的灵魂之中？对了，我能依靠它在末法时代仅仅修炼三百年就修炼到渡劫成仙的地步，这就充分证明了它的不凡，而如今它又出现在我灵魂中，说不定我当今这种状态，就是与它有关？”想到此处，元释连忙将自己的意识，探入元始印中查探。
<br/>　　“嗡！嗡！嗡！……”当一缕意识，与元始印接触，元释的脑海开始嗡嗡作响。
<br/>　　“鸿蒙元始印？与鸿蒙终结印并列鸿蒙九宝之首？众道之元，万象之始？嘶？这是……”元释的眼中露出不可思议的神情。
<br/>　　原来，方才那一瞬间，当他与元始印相碰，他感受到一股大道本源的气息，涌入灵魂，刹那间，仿佛明白了什么，又仿佛什么都不明白。
<br/>　　震惊中，狂喜时，就在他想要继续深入，探寻这块“鸿蒙元始印”奥妙之际，突然之间元始印轰然一震，自灵魂中升腾，出现在黑暗空间，耀眼的白光闪烁，向四周扩散，蓦然照亮四面八方。
<br/>　　嗡！元始印陡然一震，耀眼白光快速扩展，向四周虚无弥漫，陡然之间，它化作一道流光，带着元释的灵魂，冲天而上，速度之快如同迅雷闪电。它划破虚无之空，噼里啪啦之间，如同要冲破这无尽黑暗，又好似要打破这命运的枷锁，追寻大道的本源……
<br/>　　在这片奇异空间，哪怕有着元始印耀眼白光的笼罩，可，依旧充满孤寂与冰冷。随着元始印，元释的灵魂渐渐升高，当达到一定高度之时，陡然间，他的灵魂一颤，却是他感觉到了自己仿佛被无数锁链困住，与此同时，一股来自高空虚无的威压，骤然间暴涌而至。
<br/>　　他的身体，上升百丈，一千丈……两千丈……五千丈……，升得越高，威压越强，束缚也越强，灵魂开始打颤，开始变得虚幻起来，本来就不明亮的双眼，更是开始暗淡。
<br/>　　“还没有到尽头？”元释咬了咬牙，四周还是无尽虚无，无边黑暗，哪怕是有着元始白光的遮蔽，但，元释还是明白，自己远没有脱离黑暗空间。
<br/>　　随着不断上升，元释感觉自己越来越难受，仿佛无数锁链正在束缚着自己，撕扯着自己的灵魂，不让自己上升。随着上升的越高，这种撕扯的力量也就越大。
<br/>　　如果不是有着一道元始印的光幕守护，他的灵魂，恐怕早已溃散分裂了。渐渐地，他的灵魂变得越来越暗淡，仿佛虚弱到了极致。当上升到八千丈的一瞬间，元释的灵魂陡然一震，如同撕裂。
<br/>　　“我还能继续上升！”元释的目光一定，拿出了他渡劫时与天夺命，与雷争锋，不服输，不认命，不妥协，不放弃，至死无悔的那一股狠劲，心中生出一种与命抗争，执掌一切的气概，他咬牙坚持。
<br/>　　九千丈之时，他的灵魂已经彻底黯淡下来，手、脚、身体似乎都已经消散了，只留下一个头颅还在坚持不懈，威压也越来越强，他的灵魂之中，仿佛出现了某种奇异变化，越来越强，越来越凝实，似乎正在进行某种蜕变。
<br/>　　“我看到了，一万丈，那个位置，一定可以超脱，九为极限，十为超脱，只要我在坚持片刻，就一定能跳出去。”元释的状态十分不好，一不小心就会消散，但在这一瞬间，他那暗淡的眼神，陡然间露出前所未有的强烈目光。那是一种坚定不移的执着！
<br/>　　越光明，越黑暗。越是接近目标，元释所承受的威压也就越强，灵魂的痛苦也就越重。
<br/>　　意志不灭，灵魂不灭；执念不移，希望不灭！
<br/>　　随着不断上升，元释的周围，威压更加强大了，甚至灵魂也变得越来越热，仿佛快要融化一遍。但，元释眼中却是不顾一切，满是坚定执着，任何外界都影响不了他，他的面目满是狰狞，他大吼：
<br/>　　“啊——！”
<br/>　　“轰——”
<br/>　　他的灵魂被点燃了，燃起了白色的火焰，火焰摇摇曳曳，仿佛下一刻就会熄灭。但，不管外面威压如何强大，这火焰都不会熄灭。
<br/>　　这就是灵魂之火，也称希望之火，与十大真火之中的涅槃之火齐名，有着异曲同工之妙，甚至更胜一筹。
<br/>　　灵魂之火，是以不灭的意志为引，以坚定不移的内心为灯盏，以满怀希望的执着为核心，以灵魂为燃料，再借助近乎苛刻到不可思议的外界条件，才能点燃。
<br/>　　一旦点燃灵魂之火，灵魂不灭，则灵魂之火不灭；灵魂之火不灭，则灵魂不灭。而且它还能时时刻刻淬炼自己的灵魂，使灵魂更纯粹，使悟道更清晰。有了灵魂之火，诸邪辟易，不敢临身，更不敢夺舍。
<br/>　　至于涅槃之火，则为重生之火，拥有涅槃之火者，能起死复生，更进一步。一般它只会出现在不死凤凰身上。
<br/>　　因为有着强大的能力，所以，这两种真火及其难得，几乎绝迹。当然，这也是众修道者都极其渴望拥有的真火。
<br/>　　这些只是在瞬间就发生了，甚至元释都感受不到这些，因为他的眼中只剩下了一个目标，那就是——跳出去！为了这个目标，他忘了疼痛，忘了威压，忘了自己，忘记了一切。
<br/>　　轰——！
