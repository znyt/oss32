<br/>    暗星。
<br/>    
<br/>        暗星之所以叫做暗星，便是因为这里有着三头龙星宿最为出色的刺客，这里，是刺客的天堂，黑暗的圣地。
<br/>    
<br/>        之前的历代无面者的诏令发出的地方。
<br/>    
<br/>        不过无面者消失了之后，这里就变成了刺客天堂，接取着星空中各种暗杀任务。
<br/>    
<br/>        星空并不像一座大陆之上一般，可以随意发起战争，大陆内部的战争，星空是不管的。
<br/>    
<br/>        但是星辰和星辰之间的战争，却是众多大势力不允许的，这是对于他们统治的挑衅。
<br/>    
<br/>        一方面，大陆之间的内部消耗，可以有更多的灵魂收割，而且战争规模也不大，完全可以接受，不会造成毁灭性的效果。
<br/>    
<br/>        比方说，一场群星之间的大战下来，人的数量必定骤降，虽说这一百年之内的灵魂数量是以前的几倍，但是估摸着往后千年，这一片星辰，就不用想着有灵魂可以收割了。
<br/>    
<br/>        毕竟人都死的差不多了。
<br/>    
<br/>        这是杀鸡取卵。
<br/>    
<br/>        所以，无论是哪一个势力，都会控制着战争的局面，不至于太大，导致一处星空变成死地，没有任何灵魂可以收割。
<br/>    
<br/>        所以在星空之中，通常报复使用的手段，都是请人刺杀。
<br/>    
<br/>        而暗星，便是一个这样的位置，黑星位于孽龙星河，而且位置变幻不定，加上天庭和瑶池一直摩擦，没有必要再招惹一个敌人，所以也是一直放任暗星行事。
<br/>    
<br/>        一座黑色的大殿之中，丹成子脸上的面纱被解开，他终于是可以看见东西了，想要请暗星的人做事，必须经过暗星的人带过来，一路上，不能看到周围的景象，也不能展开感知。
<br/>    
<br/>        这样才能确保暗星的位置安全，也是暗星保护自己的手段。
<br/>    
<br/>        在大殿之中，最上方有着一把交椅，一个脸上有着一个痞子，背后披着黑色披风的痞子看向丹成子，“你有什么委托？”
<br/>    
<br/>        丹成子道：“我想要燕山财团的主事者的人头！”
<br/>    
<br/>        暗星之主看向丹成子，嘴角露出一抹笑容，“呦
<br/>    
<br/>        呵，大生意啊，你能出什么价钱？”
<br/>    
<br/>        丹成子道：“三百条天灵脉！”
<br/>    
<br/>        暗星之主琢磨了一番，然后认真道：“三百条天灵脉，只够买一个人的人头。”
<br/>    
<br/>        丹成子疑惑道：“燕山财团的主事者，价格这么高的吗？”
<br/>    
<br/>        暗星之主道：“是的，现在生意不好做啊，你要知道，现在天庭和瑶池都在注视着各处，天仙根本不敢动身，我们只能派出地仙去刺杀，既然是地仙嘛，肯定是地仙之中的巅峰强者，这样的强者，价格自然就贵了。当然，你也可以选择等到这几年过去，天庭和瑶池在鬼门星的战争结束，那个时候，就便宜多了。”
<br/>    
<br/>        丹成子心说等到那个时候，我自己都不知道能不能活，那些炼丹宗门恨不得活活吞了自己，怎么等得到那个时候。“这交易，我做了！”
<br/>    
<br/>        暗星之主笑道：“痛快！那你选一个吧。”
<br/>    
<br/>        丹成子思索了片刻，燕山之中还是陈明的地位最高，只要陈明一死，燕山必定大乱，“我要陈明的人头！”
<br/>    
<br/>        暗星之主点点头，然后道：“交了第一笔订金，我就会派遣地仙之中最强的杀手去！通知地字榜上那些排名靠前的刺客，他们有任务了！”
<br/>    
<br/>        燕山的小动作，自然是被天庭和瑶池的人发现了，这一边喊着元气大伤，一边招揽炼丹师，是哪门子的元气大伤？
<br/>    
<br/>        不过一大早，天庭的负责人便找到了陈明在河府的住所，天庭负责人问道：“这个院子就是陈明的居所吗？”
<br/>    
<br/>        旁边的两个侍卫道：“是的，大人！”
<br/>    
<br/>        天庭负责人道：“走，随我进去！”
<br/>    
<br/>        三人进入院子中，可以看得到陈明正躺在摇椅之上晒太阳，天庭负责人道：“燕山山主还真是好悠闲啊！”
<br/>    
<br/>        陈明本想躺着说话，他懒得起来，这个时候，脑海之中忽然响起了系统小姐姐的声音，“你触发了b+级剧情任务，连环刺杀，任务要求，活着！奖励：2亿灵识，30w声望。未知奖励。”
<br/>    
<br/>        陈明起身，伸手端起身前的茶壶，给天庭负责人
<br/>    
<br/>        倒了一杯茶，“天庭的大人，怎么有心思光顾我的小院？”
<br/>    
<br/>        天庭负责人在陈明的对面坐下，而后道：“你们燕山阳奉阴违，明着说没有实力继续炼丹，背地里却是在大量招揽炼丹师，难道以为我天庭不知道吗？”
<br/>    
<br/>        陈明扫了天庭负责人一眼，天庭的人，没有必要刺杀自己，他们直接动手就行了。
<br/>    
<br/>        那么到底是谁要自己的命？
<br/>    
<br/>        说实话，陈明并不是很关心是谁要自己的命这个问题，要自己的命的人多了去了，也不差一个两个，他关心的是，刺杀他的，到底是什么人。
<br/>    
<br/>        嗯，这个问题比较关键。
<br/>    
<br/>        陈明呵呵一笑，然后道：“大人在说什么啊，我怎么听不懂啊！”
<br/>    
<br/>        天庭负责人一掌拍碎了桌子，怒道：“陈明，你休要胡搅蛮缠，真当我天庭不敢动手吗？”
<br/>    
<br/>        陈明嘿嘿笑，“动手，试试看啊？”
<br/>    
<br/>        天庭负责人怒道：“你知道我天庭的主要力量都在和瑶池对峙，所以有恃无恐，但是，对峙结束之后，你们还能活吗？天庭大军降临而下，十个燕山也不够填命！”
<br/>    
<br/>        对峙结束之后，呵呵，对峙结束之后此地还有没有天庭都是一个问题。
<br/>    
<br/>        陈明喝完了一杯茶，却是有一个天庭的人伸手端起茶壶，似乎是准备给陈明倒茶，若是平常，陈明倒是觉得没什么，但是此刻，陈明却是感受到了不对的地方。
<br/>    
<br/>        天庭是来兴师问罪的，怎么可能会给自己倒茶。
<br/>    
<br/>        陈明的右手落下，落在两年蝉的刀柄之上，就在这时，倒茶的天庭兵卒，手中忽然出现一柄匕首，朝着陈明刺去！
<br/>    
<br/>        天庭负责人，在此刻都是吃了一惊，自己带来的人，怎么可能在这个时候刺杀陈明？
<br/>    
<br/>        他可没有下令，他看向身旁的两个兵卒，却是发现，这两个家伙，他从来没有见过！
<br/>    
<br/>        他的心中顿时明了，这不是他的人！
<br/>    
<br/>        陈明右手握住两年蝉，黑刀出鞘，刹那间一刀挥出，两倍速！
