
<br/>　　<br/>　　    “你看这夜色美不美······”其实，在这句话后面还有句话王惊蛰没有说出口，他想说的是：“夜美，人更美”<br/>　　<br/>　　    从东郊大野地里出来后，王惊蛰和茅小草打了辆车回到了长安市区，此时天色已然快到午夜了，茅小草靠着车窗拄着下巴，眼睛看着外面闪烁的路灯和霓虹灯招牌，不知道在想些什么。<br/>　　<br/>　　    可能是感觉两人之间的气氛比较沉闷和尴尬，王惊蛰没话找话的问道：“你看什么呢，想什么呢？”<br/>　　<br/>　　    茅小草叹了口气，说道：“我在想，你一会儿会不会要套路我呢”<br/>　　<br/>　　    “啊？”王惊蛰状若痴呆，没有反应过来。<br/>　　<br/>　　    茅小草又叹了口气，说道：“如果你套路我的话，我在想是一脚踹的你后半生不能生活自理，还是干脆拂袖而去呢？”<br/>　　<br/>　　    “啊？”王惊蛰不解的问道：“你为什么要打我？”<br/>　　<br/>　　    茅小草：“·····”<br/>　　<br/>　　    出租车司机在后视镜里，看了眼后面那对年轻男女，无奈的摇了摇头，脸上很有种忆往昔哥曾经书写过辉煌，哥也曾经年轻过的表情。<br/>　　<br/>　　    二十多分钟后，出租车开到一家宾馆门前，茅小草推开车门就下去了，王惊蛰付过钱后刚要下车，司机回头给他叫住了。<br/>　　<br/>　　    “哥们，兄弟送你一句话，待会去酒店前台给身份证的时候，你用食指在身份证上画个圈，记住背对着那个姑娘点哈”<br/>　　<br/>　　    “这是什么意思？”<br/>　　<br/>　　    司机眨了眨眼睛说道：“都是经验，这一招在长安的宾馆，酒店都是通用的，你就用吧，事成之后在心里感谢我一下就行了，大哥也就能帮你到这里了”<br/>　　<br/>　　    王惊蛰狐疑的下了车，进到宾馆里的时候，茅小草站在吧台前无聊的哼着小曲，低着脑袋摆弄着手指。<br/>　　<br/>　　    “您好，先生，小姐，请问是要开房么？”<br/>　　<br/>　　    “嗯，是的，你身份证呢？”王惊蛰拿出自己的身份证，又朝小草问了一句。<br/>　　<br/>　　    接过小草的身份证，王惊蛰递到了前台服务员的手里，忽然想到了自己下车前司机师傅交代的那句话，然后就鬼使神差的用食指在身份证上画了个圈。<br/>　　<br/>　　    服务员顿时笑了，露出一脸我明白了的表情，隐晦的朝着王惊蛰眨了眨眼睛，那意思是，放心吧我懂了。<br/>　　<br/>　　    王惊蛰：“？？？”<br/>　　<br/>　　    片刻后，服务员看着电脑说道：“不好意思，先生，我们酒店满房了，现在只剩下一张大床房了”<br/>　　<br/>　　    “唰，唰”王惊蛰懵逼的眨着眼睛，大脑瞬间当机了，紧接着他就听到了背后一阵磨牙的声音。<br/>　　<br/>　　    “你套路我？我拿你当兄弟，你居然要套路我······”小草姑娘咬着牙一把抓住王惊蛰的领子就给他拽了过来，手指戳着他胸膛说道：“呵呵，就你这智商还和我玩套路呢，如果咱俩的智商赛跑的话，我让你先跑二百米都能给你来个弯道超车，你有什么资本套路我啊，你个傻鸟！”<br/>　　<br/>　　    王惊蛰继续懵逼的说道：“不是，酒店没房间了，那也不怪我啊，谁住宾馆之前还得算一卦么？”<br/>　　<br/>　　    “你还敢不承认，还嘴硬？你的猪蹄子在身份证上画个圈圈干什么，这套路你当我不知道么？”<br/>　　<br/>　　    王惊蛰瞬间通透了，捂着脑门子无语的说道：“你别说了，太丢人了”<br/>　　<br/>　　    十分钟后，王惊蛰和茅小草来到了楼上客房，当然是两间房了，门对着门的两间。<br/>　　<br/>　　    “嘎吱”推开房门，王惊蛰跟个受惊的兔子似的，连忙钻进了房间里。<br/>　　<br/>　　    门外，茅小草抿着嘴笑了：“傻鸟，真是头猪”<br/>　　<br/>　　    “噗通”王惊蛰进到房间里后，就扑到了床上，把脸埋在了被子里，来回踢着两条腿说道：“好尴尬啊，好尴尬”<br/>　　<br/>　　    对面的房间，茅小草关上门，来到卫生间拧开水龙头放着热水，又哼起了小曲，这次的小曲哼的清楚多了：“女孩的心思男孩你别猜······别猜，别猜·······女孩的心思男孩你别猜，你猜来猜去就会把她爱”<br/>　　<br/>　　    一夜过后，日上三竿，由于昨天晚上睡得比较晚，今天起的也就比较晚了，王惊蛰叫了小草姑娘两次，最后都被她一句我要睡美容觉和你是一头猪么这句话给骂了回去。<br/>　　<br/>　　    直到将近中午的时候，茅小草才起来，梳洗打扮了一番后她从房间里出来，再看见王惊蛰的时候脸上也没什么变化，仿佛昨天的小插曲因为睡了一觉而烟消云散了。<br/>　　<br/>　　    王惊蛰故作镇定的问道：“睡得好么？”<br/>　　<br/>　　    小草翻了翻白眼。<br/>　　<br/>　　    两人从宾馆里出来，茅小草问道：“吃饭了，中午吃什么？”<br/>　　<br/>　　    “有一家泡馍很好吃，以前我来过两次长安都是是吃他们家的”王惊蛰没给茅小草反对的机会，直接拉着她上了一辆车就跟司机说去长安老城区，然后进了一条胡同找到了一家老店面。<br/>　　<br/>　　    “两碗泡馍，多放辣子”王惊蛰喊道。<br/>　　<br/>　　    片刻后，两碗泡馍上来，王惊蛰拿起筷子，说了几声吃吃吃后就埋头苦干起来，扒拉着泡馍夹起一筷子就塞满了嘴。<br/>　　<br/>　　    茅小草用筷子夹起一块牛肉，慢吞吞的塞进嘴里，细嚼慢咽着，王惊蛰“咕嘟”一声咽了嘴里的东西后，诧异的问道：“你不喜欢吃么？很好吃的啊”<br/>　　<br/>　　    茅小草很认真的看着他，说道：“我拜托你，下次请女人吃饭的时候，能不能先征求一下对方的意见？这么多辣椒，吃完你想让我脸上全是痘痘么？这么多的油，你想让我涨多少肉？你难道不应该先问问，你喜欢吃西餐，中餐，还是川菜，粤菜？”<br/>　　<br/>　　    王惊蛰眨着眼睛，理所当然又一本正经的说道：“我就是觉得这里很好吃啊·····”<br/>　　<br/>　　    茅小草无奈了，深深的叹了口气，肚子里翻来覆去的就腹诽着一句话：“来日方长，欠调教”<br/>　　<br/>　　    片刻后，王惊蛰忽然发现，小草姑娘吃的居然比他还欢快，一碗泡馍在以肉眼可见的速度消失着。<br/>　　<br/>　　    “我还以为你不喜欢吃呢”<br/>　　<br/>　　    “我不是不喜欢吃，我刚才只是在和你讲述一个道理”<br/>　　<br/>　　    茅小草吃完后，矜持的擦着嘴巴，问道：“接下来你要干什么去？”<br/>　　<br/>　　    王惊蛰寻思了半天说道：“还不知道，你呢”<br/>　　<br/>　　    茅小草也寻思了半天说道：“我也不知道”<br/>　　<br/>　　    两人之间忽然寂静下来，然后王惊蛰忽然忸怩着张了张嘴，吭哧半天也没放出个屁来。<br/>　　<br/>　　    小草姑娘斜了着眼睛问道：“你是不是想说，让我陪你仗剑走天涯啊？”<br/>　　<br/>　　    王惊蛰眼神闪烁飘忽的说道：“我也曾想仗剑走天涯，但是······安检没过去”<br/>　　<br/>　　    “你是猪么？”茅小草悲凉的叹了口气。
