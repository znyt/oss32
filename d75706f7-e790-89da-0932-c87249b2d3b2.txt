　　有了第一丝阳属性的元神，那就会有第二丝，第三丝……
<br/>　　渐渐地，阳性元神变得越来越多，此时，元始控制住黑鱼阴神，不让它与阳性元神碰撞，因为阳性元神还很小，经不起折腾。
<br/>　　此刻就需要元始耗费巨大的心力，他将那些阳性元神凝聚成一条白色小鱼。这条小鱼对比那黑色大鱼无疑是要小了不知多少倍。
<br/>　　元始将两鱼首尾相连，控制它们缓缓游动，渐渐地，黑鱼吐出的至阴元神被白鱼吸收，白鱼突出的阳性元神，被黑鱼吸收。
<br/>　　虽然黑鱼瞬间便会将那一丝丝阳新属性泯灭，但仍旧会留下一丝丝阳气在内。同样，阴性元神来到小鱼体内，一样会被消灭，但也会留下滴滴阴性，通过吸收更多的太阳真火，渐渐地阳性小鱼就会变大。
<br/>　　就这样，随着时间的流逝，两条鱼变得阳中有阴，阴中有阳，变得不再那么敌视了。
<br/>　　这就如同人一样，时间相处久了，两者也会变得熟悉了，都会被对方所感染。
<br/>　　两条鱼亦然，若是再给它们一些时间，它们究竟会变成什么？
<br/>　　……
<br/>　　太阳星上，扶桑树下。一个道人，正在闭关。
<br/>　　那道人名为元始。此刻，在他的元神之中，有一条白色阳性小鱼与一条黑色阴性大鱼。
<br/>　　周围无数的至阳之力涌入到元始的元神之中，被白色小鱼所吸收。
<br/>　　渐渐地，随着他吸收至阳之力的增多，白色小鱼渐渐变大，而黑色大鱼却在渐渐变小。直到某一天，两者达到平衡。之后，两条阴阳鱼不用元始的控制，它们也能在一起愉快地玩耍。它们首尾相连，相互追逐，演化阴阳大道。
<br/>　　至此，元始的阴阳元神小成了。
<br/>　　此刻，元始才将元神沿着扶桑神树往下延伸，去寻找那太阳本源。
<br/>　　与太阴本源一样，太阳本源一样在扶桑树下的本源空间之中，经过多年的适应，元始并没有废多大的劲，便找到了太阳本源。
<br/>　　太阳本源如太阴本源一般大小，但颜色却是通体赤金之色，本源之上，金色火焰焚烧着，同样一根扶桑树根扎根于太阳本源之上，与太阳星连成一体，两者命运相连，一荣俱损。
<br/>　　既然找到了太阳本源，之后便是接引太阳本源来撞开另一个太阳穴的事了。这种事元始做多么，既然做多了，那便熟悉了，废不了多少时间，元始便将穴窍打开，无数太阳本源如潮水般涌**窍之中，扩展这元始穴窍的空间。
<br/>　　渐渐地，将穴窍转化为金色神窍。
<br/>　　他全身金色火光四射，尤其是左边太阳穴之处，迸发出耀眼的金光。
<br/>　　无数的太阳本源之力被元始接引而来，注入到穴窍之中，同时也有一部分顺着他的经脉流遍全身，增强者他的肉体强度。当然，还有一部分太阳本源，来到元始的识海之中，进入到白色小鱼之中，增强者它的至阳之力。
<br/>　　黑白阴阳鱼在缓缓旋转，吸收着这股至阳之力。同时，元神之上，黑白两色光芒越来越盛，到最后，元始的头顶，直接出现一个巨大的黑白阴阳太极图，在不断地吸收着四周的太阳真火。
<br/>　　“轰隆隆！”
<br/>　　不知过了多久，元始的体内一阵轰然巨响，太阳本源停止了传输，元始全身金光大涨，之后，银光大涨，再之后，赤光、青光、黄光、蓝光、黑光、白光……各种各样的光彩照遍元始全身。
<br/>　　元始身上，三百六十五处神窍之处，在这一刻全部发出耀眼的光芒。
<br/>　　“不好！”
<br/>　　元始心道一声，连忙闪身来到因果莲台的混沌空间之内，刚才突破动静太大，他担心被人发现。
<br/>　　混沌空间之内，元始全身气势大涨，三百六十五道光柱从他身上发出。一瞬间，他的身上，从识海之处为起点，一条银色细线蔓延开来，穿过两大太阳穴，往下而去，每到一处神窍，它都会穿过，一直到脚底的涌泉神窍处。
<br/>　　最后，这条银色细线更是将元始全身三百六十五个神窍连接起来，之后，银光大涨，连带着神窍发出耀眼光芒。
<br/>　　全身被银色细线密布，如同一张大网，包裹了元始的全身。
<br/>　　混沌空间之内，混沌之气蜂涌般地朝元始涌来，刚到他的身边，便化作地水火风各种灵气，注入到元始肉体之中，经脉之中，各处神窍之中。
<br/>　　这一刻，三百六十五个神窍空间在迅速扩展，在增大，地水火风涌动而来，空间变得如同一处洞天。
<br/>　　三百六十五个神窍，意味着三百六十个洞天，里面储存的全是灵气法力，还有各种星辰本源。
<br/>　　星辰演化，洞天演变，一千里，两千里，三千里……
<br/>　　转瞬间，神窍洞天便增加到九千里之光，之后，它们才渐渐停止扩张。
<br/>　　九千里，是洞天的极限，之后便会演变成小世界。
<br/>　　轰隆隆！
<br/>　　伴随着一声巨响，元始全身气息大涨，这一刻，他清晰地感觉到，自己的肉身突破了混元的限制，真正达到了混元大罗金仙的地步。
<br/>　　同时，《玄黄不灭》第五重星辰入神窍修炼成功，他的星辰圣体大成。

<br/>　　“我实在是太强大了！”
<br/>　　元始感应着自身的实力，默默道。
<br/>　　看了一眼混沌空间，发现空间之中再次少了三分之一的混沌之气，元始实力突破的喜悦渐渐变淡了，真不知道要到何时，这些混沌之气才会再次回补过来。
<br/>　　元始心疼不已，话说这混沌空间虽然会自主吸收灵气转化为混沌之气，但那速度实在是太慢了。想让它自己补回这些混沌之气，真不知道要到何时？
<br/>　　不过，元始一般也用不到混沌之气，此次只是一个意外。今后，应该也不会用到，就让它慢慢不回来吧。
<br/>　　实力突破，元始便不再停留，出了混沌空间，来到扶桑树下。
<br/>　　同时，赤帝与扶桑树的争斗也结束了。
