　　五月初九，天空湛蓝，界牌关前。
<br/>　　灵鹫山圆觉洞燃灯道人、九仙山桃源洞广成子、少华山云霄洞赤精子、二仙山麻姑洞黄龙真人、崆峒山元阳洞灵宝大法师、夹龙山飞云洞惧留孙、乾元山金光洞太乙真人、五龙山云霄洞文殊广法天尊、九宫山白鹤洞普贤真人、慈航真人、玉鼎真人、道行天尊、清虚道德真君、云中子、孔宣、金翅大鹏迦楼罗、黄金道人、申公豹，以及杨戬、哪咤、金吒、木吒、李靖、韦护、雷震子、杨任、土行孙、洪锦、龙吉公主等人，俱皆到齐，此次前来，便是为了那红尘杀劫，封神量劫！
<br/>　　众仙高立关前一云层之中，却不见任何人影，四周空空如也，想来是截教众仙还没有到来。
<br/>　　正在此时，关前忽然阴暗下来，微风吹过，空气怪异流动，似乎有微不可见的火花在空气中噼啪作响。
<br/>　　众人放眼望去，一时间都说不出话来。只见前方驶来一艘广大无边的度人舟遮蔽了天空，携带着不可阻挡的天地之威，从天上压了下来。
<br/>　　那艘度人舟忽然升起万丈之高，之后悬浮不动，截教群仙都各自手持阵幡，飘飘然飞出舟来。但见那舟上火风嘹亮，白鹤飞翔，青鸾祥鸣，声音直上九天，四周云涛翻滚，如海浪般奔腾不休。四周旗幡林立，三山五的岳仙人，奇人异士等，数量足足有百万之众，在云海之中若隐若现，声势浩荡。
<br/>　　阐教众仙与对方隔空相望，看到对面那浩大声势，再回头看向自己一方，不过寥寥二三十人罢了，他们此刻却是感受到了那种寂寞的感觉了。燃灯叹道：
<br/>　　“今日方知截教之中有如此多仙人，我教也不过是屈指可数的数人而已。”
<br/>　　玉虚众仙看着对方，突然间自四面八方涌来无数云海，中央有四道绝世杀气分为黑、白、金、青，冲破亿万苍穹，杀气直上三十三天外，四道杀气相互交织，在虚空中乱转，最后化为一道光华笼罩下来，笼住这四方翻腾不休的云海。
<br/>　　又见六道太白元气翻滚不休，如六条雪白玉龙一般，鳞甲张开，在无尽云海霞光之中来回穿梭，反反复复，不时发出龙吟之声。
<br/>　　燃灯面色一变，对众人道：“百万群仙已是难以对抗，今日又有诛仙剑阵，截教圣人亲自主持，实在是难以抵挡。”
<br/>　　广成子道：“往日我等也曾听说过这诛仙剑阵，乃是诛仙、戮仙、陷仙、绝仙四剑演化而成，却未曾亲眼见过，我等如今已身在劫中，落入红尘，不若前往一观，也知这杀阵奇妙！”
<br/>　　燃灯劝慰道：“此乃绝世杀伐之阵，我等又何必去惹麻烦，在此等候师尊便是了。”
<br/>　　太乙道：“道兄所言，贫道并不赞同，此阵乃洪荒天地第一杀阵，我等已身入杀劫，如今去观摩一二，也好增长见识。”
<br/>　　其余众仙纷纷赞同，燃灯劝不住，众仙上前，到了阵前，驻足观看，只见阵内玄机奥妙，有无穷杀机，亿万杀气，颠倒乾坤，绝灭天地，心中骇然无比。虽然同属道教一脉，但截教与阐教并不相同，截教的阵法是天地一绝，众弟子如痴如醉，各有领会。
<br/>　　截教众仙见此，目露嘲讽之色，觉得阐教诸仙此时就好像乡下之人，毫无见识。忽然一道光华闪过，数道流光汇聚，成为一个人形，乃是截教龟灵圣母，她手持长剑，怒斥道：“玉虚门下，既来此地，又怎能只做那偷窥之事，谁敢与我大战三百回合？”
<br/>　　惧留孙大叫道：“龟灵圣母莫要嚣张，贫道前来会你！”
<br/>　　说着，惧留孙上前与之大战，不过百余个来回，惧留孙便稍显劣势，他正要取出捆仙绳对战，龟灵圣母好像早知他的意思，迅速取出一日月宝珠打出，明耀照人，如日方升，金光灿灿，惧留孙心知不能招架，就要躲开，这时，一道足足有数千丈的翻天大印从天而落。
<br/>　　“师妹小心！”
<br/>　　多宝道人突然道，紧接着，他取出一方宝塔，向上扔去，宝塔迅速变大，与那大印相撞，轰隆隆如天崩地裂，翻天印与宝塔都倒翻而回，狂暴的气息向着四面八方狂涌而出！
<br/>　　申公豹取出青莲宝光旗，放出亿万神光，将众人笼罩住，其余波动向着界牌关狂涌而去，势如海潮，澎湃如海啸，五百里界牌关，城楼万千，屋舍无数，都在这巨大的冲撞之中仿佛腐骨一般纷纷倒塌，界牌关外群山皆分崩离析，化为尘埃向着四方扩散而去，千里之内，山川皆夷为平地，去势仍然不止。
<br/>　　“哎，此界牌关也是命中有此一劫。”突然，一道无形手掌往下一压，霎时间风云停止，尘埃落尽，天空恢复清明。
<br/>　　“拜见师尊！”
<br/>　　截教众仙都对着来人恭敬拜道，来人是一个身穿青色道袍，坐着跬牛的青年，正是通天教主。
<br/>　　“嗯。”通天教主对众人点了点头，便进入度人舟中香榻之上，之后对着多宝道人道：“你去见一见你大师伯，下一封战书！”
<br/>　　多宝道人领命，出了度人舟，便向着玉虚门处走来，昆仑山元始心神一动，一步踏出，来到界牌关上，挥手间，空中云层之中便多了一金碧辉煌的建筑，元始入内，高居上方，众弟子皆入内，一番行礼，立于两旁。
<br/>　　“禀师祖，门外多宝道人求见！”哪咤来报。
<br/>　　“宣！”
<br/>　　哪咤领命，不一会儿，多宝道人进入殿内，对着元始行了一礼，道：“拜见大师伯，我奉我师命前来奉上战书一封。”说着，他将手中书信呈上。
<br/>　　元始伸手一招，将书信抓在手中，单眼一瞟，便已明了，他对着下方多宝道人道：“回去复你师命，明日我等去会你万仙阵。”
<br/>　　多宝道人领命退下，回去禀告掌教师尊。
<br/>　　大殿之中，元始突然扔出数枚玉牌，漂浮于众仙面前，看着诸仙疑惑的神情，元始道：“此乃护身玉牌，有此玉牌，可护尔等阵中无恙。”
<br/>　　诸仙闻言，大喜，忙接过玉牌，叩谢元始天尊。元始点头不语。
<br/>　　翌日清晨，元始领着阐教众仙来到万仙阵前，突然，一声牛吼传来，众人望去，只见一道身影自远处高空云层之中出现，他侧身骑着青牛，满头白发，手持一白色拂尘，身边一青年跟随，正是人教太清圣人。
<br/>　　“见过大兄！”太清圣人下了青牛，对着元始拱手一礼道。
<br/>　　“嗯。”元始淡淡点了点头。
<br/>　　“见过二师伯！”阐教众仙对着太清圣人拜了一礼。
<br/>　　“见过大兄、二兄！”与此同时，截教之中，通天教主亦走出度人舟，对着元始与太清一拜，同时，他将目光投向太清，不知太清来此所谓何事。
<br/>　　太清微微一笑，道：“老道来此，只是凑个热闹，并不会出手，权当与你们做个裁判。”
<br/>　　通天闻言，将目光看向元始，元始淡淡点了点头，不屑言语。通天道：“那小弟便在次恭迎大兄破阵！”
<br/>　　语罢，通天教主转身进入大阵，主持大阵核心。云雾翻滚，百万群仙分成两旁，只见阵门左右，各有两名万丈巨人，一人手持黑色号角，呜呜地吹响，另一边一巨人敲起战鼓，轰轰作响，其余百万众仙，唱起战歌。
<br/>　　战歌嘹亮，百万丈妖气滚滚升天，遮天蔽日，天地失色，乾坤震惊！
