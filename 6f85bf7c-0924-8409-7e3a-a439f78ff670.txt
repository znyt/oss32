　　第151章
<br/>　　顺着程大雷的目光望去，只见山谷内是一片热火朝天。
<br/>　　一人抱不过来的大树被伐倒，伴随着号子声拉入山谷。磨盘大小的山石从山坡上滚落，将成为修筑城墙的材料。谷内的河道在疏通，猪铁匠带着人在铁匠铺挥汗如雨，老弱妇孺齐上阵，每天的饭菜第一时间送到。
<br/>　　人究竟拥有多大潜力，或许只有生死存亡的关头才知道。五千人有多少潜力，在大刀将落在头顶时就见分晓。
<br/>　　在五千人全部动员上的情况下，山寨的工程进度有条不紊的向前推进。
<br/>　　但程大雷只有一个想法：太慢，太慢，太慢！
<br/>　　必须在开春之前，将蛤蟆寨生到三级，成为一座无法被攻破的堡垒。如果再给程大雷一些时间，他有把握给山寨每个人搞到全套装备，训练这群乌合之众成为一支令行禁止的精兵。
<br/>　　但程大雷如今没有时间，所以他觉得一切太慢，太慢，太慢！
<br/>　　山谷内有多紧张，就衬得飞虎寨就有多诡异。
<br/>　　“幽州王千秋万代，功与天齐！”
<br/>　　天还没亮，窗外就响起这样的喊声。裴喜之这一刻真连死的心都有，这几天在飞虎寨真可谓是度日如年，每天沉浸在对幽州王的歌功颂德之中。
<br/>　　早起要有晨跑，饭前要有祷告，睡前大家聚在一起，讨论如何才能成功，还有幽州王究竟有多伟大。
<br/>　　裴喜之都要疯了好不好！
<br/>　　他本打算和其他几人商量商量这次任务，可徐神机将他们安排在几十人睡在一起的大通铺上，全天十二个时辰无死角监控。
<br/>　　什么调查、打探、分化……现在裴喜之想都不敢想。
<br/>　　他来之前不是对这里发生的事情没有准备，蛤蟆寨要来暴的，他这次带来的人里，不是没有高手。蛤蟆寨要玩头脑，自己足以以一当千。但他没想到，蛤蟆寨既没有和他比头脑，也没有和他比暴力，他们比谁更无耻。
<br/>　　裴喜之技不如人，甘拜下风。
<br/>　　“徐军师，我已经来好久了，是不是见见程寨主。”
<br/>　　“程寨主拉肚子，改日改日，我先考考你幽州王做过的十件大事，看你能不能回答上来？”
<br/>　　“……”裴喜之。
<br/>　　“军师，昨日程寨主拉肚子，今天好点了么，要不要我去看望一下？”
<br/>　　“嘿，你早说呀，程寨主一大早就外出公干了。我先问问你，幽州王当你是在那一场战斗中大获全胜，击败戎族？”
<br/>　　“长冶谷战斗。”
<br/>　　“杀敌多少人？”
<br/>　　“八千。”
<br/>　　“他们都叫什么名字？”
<br/>　　“……”裴喜之。
<br/>　　终在一个夜里，裴喜之等人抱着大无畏的勇气，发动了保卫自己智商的一战。连夜逃出飞虎寨，不顾一切的逃回幽州城。
<br/>　　夜幕中，有人叩开徐神机的屋子。
<br/>　　“军师，军师，他们走了？”
<br/>　　“走了么！”徐神机翻身坐起：“快把厨房的猪肉拿来，好几天没吃肉，嘴里淡出个鸟来。
<br/>　　一天一夜后，裴喜之回到幽州城。当李善言见到他们时，顿时吓了一跳。只见这几人衣衫褴褛，双眼冒着绿光，每个人都活脱脱瘦了一圈。
<br/>　　“喜之，你这是怎么了，减肥成功啦？”李善言。
<br/>　　“是英明神武，神机妙算的军师大人么！愚蠢的我终于见到您了，像迷途的羔羊重回母亲的怀抱，夜行的旅人见到朝阳。”
<br/>　　“……”李善言。
<br/>　　“先给他们搞点吃的？”李善言也不知道到底怎么回事。
<br/>　　“蛤蟆寨打你们了？”李善言。
<br/>　　“我们是尊贵的军师大人派去的，他们肮脏的手指怎敢伤害我们一根头发。”
<br/>　　“那是把你们关起来了？蛤蟆寨的山贼如此大胆。”
<br/>　　“将狮子的胆子按在他们身上，他们也不敢这样对待我们，因为这样对待我们，就是冒犯军师大人的无上的威仪。”
<br/>　　“那你们是怎么回事？”李善言道：“还有，你能不能正常点说话。”
<br/>　　裴喜之一怔，确实蛤蟆寨究竟那里对自己不好了，或者那里做得不对了。一时间竟也不知该如何回答。
<br/>　　“英明的军师大人，请您开恩，允许我拙笨的口舌诉说连日来的经历……”
<br/>　　李善言忍着极大的恶心，听完了这些人的经过。最后愣了好大一会儿，发出一声感慨。
<br/>　　“这么无耻！”
<br/>　　“军师，裴大人怎么办，是不是给他们请个郎中？”有手下问。
<br/>　　“拖出去，一人四十大板，什么时候会说人话了，再带来见我。”
<br/>　　“诸位……”等裴喜之几个被带下去，李善言脸上挂出微笑，冲身边几个幕僚道：“没有帆的船无法航行，没有你们的智慧我无法做事，还有谁想去蛤蟆寨啊？”
<br/>　　说完，李善言恨不得给自己一个耳光，自己说话怎么也跑偏了。
<br/>　　李善言身边鸦雀无声，平时几个自诩聪明过人的也没有说话。裴喜之去了才十几日，就差不多被搞疯了，谁还敢再去。
<br/>　　“汝才，你走一趟如何？”
<br/>　　“尊贵的军师大人，笨拙的鸟儿……”
<br/>　　此人起身，在触及李善言目光时，识趣的坐下来，闭上嘴巴。
<br/>　　李善言冷哼一声，你再说下去，信不信我当场抽你。真以为我这读书人不会揍人么。
<br/>　　他想了想，忽然冷笑道：“罢了，无论程大雷打什么鬼主意，试一试他便知道了。来人，传我的信过去，让蛤蟆寨调一千人去打伏虎山。”
<br/>　　风雨亭后，杨龙停选择了程大雷，封他为游骑左将军。而其他绿林匪人有些实在吃不下去饭，便加入幽州军。而一些人，因为仗着手下人马众多的缘故，没有给幽州王面子。
<br/>　　其中便包括伏虎山的熊老大。
<br/>　　杨龙停早有消灭熊老大的意思，一来熊老大手下七八百人，想打下来也不是什么难事。二来，杨龙停也需要杀鸡儆猴，给幽州王绿林道见见信。
<br/>　　近几日，正在筹划这件事。李善言写了一封信，快马送到青牛山。如果程大雷肯出兵，就是狗咬狗，两方巨贼是两败俱伤。如果程大雷不肯出兵，他的心意差不多也就试探出来了。
<br/>
