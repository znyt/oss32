　　鸿蒙量天尺！
<br/>　　后天第一攻击神器，鸿蒙量天尺，又称天地玄黄量天尺！是开天之初诞生的太初紫气混合天地玄黄母气、开天玄黄功德形成的一柄利尺。
<br/>　　这里所说的后天第一攻击神器，并不是攻击力第一，而是开天后第一个形成的攻击神器。与天地玄黄玲珑宝塔一样，都为后天第一，只不过一个是攻，一个是守！
<br/>　　而元始所要参悟的太初玄黄之道，没有什么比这蕴含太初玄黄母气的鸿蒙量天尺更好的了。
<br/>　　元始打量着眼前这柄神尺，只因这尺子紫金闪烁，攻击之时能够释放出耀眼紫光，形成领域禁锢敌人，有点像鸿蒙，故称鸿蒙量天尺！
<br/>　　突然，元始心念一动，只见他手中的鸿蒙量天尺在一寸一寸的分解开来。一缕缕太初紫气与玄黄之气被还原出来，飘散到本源空间之中。紫光闪烁，明灭不定，犹如一尊君王降临凡尘，充满高贵之气。
<br/>　　天地初始，唯有紫气东来！
<br/>　　元始印在刻录太初紫气之中的道则神韵，收录太初紫气的道韵。天地有太初，紫气唯东来。
<br/>　　“当——”
<br/>　　蓦然之间，空中骤然响起“当”的一声响，就如同暮鼓晨钟，悠悠扬扬的响起……
<br/>　　元始听着这钟声，猛然怔住了，脑海之中浮现出一种类似“醍醐灌顶”这般震动，他明显感受到，就这么一声钟响，似乎已经让自己看到了自己轮回千百世！
<br/>　　千百世的漫长岁月光阴，在这一刻化为一抹流光，在元始心间快速流过。
<br/>　　似真实，似梦幻，似实在不虚，又似乎难以捉摸……
<br/>　　余音袅袅之际，空间中紫气大涨，一道道宏大的紫气，自元始为中心，向着四周翻腾而起，如云浪一般疯狂涌动。此刻，元始就仿佛是一团东升的昊阳大日，缓缓升起。一道浓郁到了极点散发着璀璨光芒的紫气，在元始周遭缭绕旋转。
<br/>　　元始心中，沉浸在对于道的领悟之中，不可自拔。
<br/>　　“天地太初，紫气东来！”
<br/>　　“万物造化，玄黄天地，是为万物之母，滋润天地苍穹……”
<br/>　　……
<br/>　　幽冥血海深处，冥河刚一回到家中，便急忙打开之前抢到的荡魂钟，查看里面的东西。他之前已经突破到了准圣圆满，就是因为没有鸿蒙紫气，才不能突破成圣。为了成圣，他不惜冒着得罪妖族、镇元子而抢来这红云的荡魂钟。
<br/>　　只要能够成圣，他愿意不惜一切代价。
<br/>　　打开荡魂钟，冥河道人满怀激动之色的查看里面的空间，一想到自己马上就可以成圣了，他就激动不已。
<br/>　　“没有？怎么会没有？”
<br/>　　冥河愣住了，竟然什么都没有，那我不是白忙活了吗？他不相信，立马再次查看一遍，将荡魂钟一寸一寸地搜寻好几遍，还是没有看到别的东西。
<br/>　　“怎么会没有？怎么会没有？老天不公啊！”冥河有些恨声大叫道。
<br/>　　“冥河，给我出来！”
<br/>　　这时，冥河道场之外，镇元子追了进来，对着他便是一阵猛喝道。
<br/>　　“冥河！你给我出来！别以为躲在里面就没事了。”
<br/>　　“镇元子，你想要干什么，这里可是我的地盘？”
<br/>　　冥河皱着眉头走了出来，看着一脸焦急的镇元子大声道。
<br/>　　“冥河，将红云的东西拿出来！”
<br/>　　镇元子不管其他，直接要冥河拿出红云的荡魂钟。
<br/>　　冥河皱了皱眉头，知道那荡魂钟中并没有鸿蒙紫气，仅仅荡魂钟，只是一件上品灵宝，他冥河还不差这么一件灵宝，但若是他给了镇元子，当他发现里面其实什么也没有，也一定会将这一切归咎与自己，若是不给他，又会得罪镇元子。
<br/>　　一时间，他陷入了两难的境地，而且，更麻烦的是，天地间其他人也会认为是自己拿走了鸿蒙紫气，虽然事实上他并没有拿走，但别人并不知道啊，也没人会相信自己的话。
<br/>　　既然给也不是，不给也是，那便不给了！
<br/>　　“道友，我只拿到一件荡魂钟，并没有得到里面的鸿蒙紫气！”冥河皱着眉对镇元子说道。
<br/>　　“冥河，你别想吞没红云的东西，那鸿蒙紫气也就算了，我不与你计较，但若是红云的其他东西你也要吞没，我定不会罢休！”镇元子现在正沉浸在红云死了的伤痛之中，哪还能听得起这些事，开口便是威胁冥河。
<br/>　　“哼！我冥河还从没有怕过谁？说了没有就是没有，想要做过一场，谁怕谁啊？”冥河冷声道。
<br/>　　“冥河，你确定不还？”镇元子也变得不耐，冷声道。
<br/>　　“是有如何？”
<br/>　　“杀！”
<br/>　　镇元子二话不说，挥动灵宝地书，头顶人参果树，便向着冥河杀去，红云是为了他而死，他心中充满愧疚，红云死后留下的遗物，他不能让它落在其他人手中。
<br/>　　“哼！”
<br/>　　冥河冷哼一声，挥手之间，便出现两柄闪烁着无边杀气的神剑，正是阿鼻、元屠两大先天杀剑。
<br/>　　“血海大阵，起！”
<br/>　　冥河大喝一声，只见周围虚空凝固，时光扭曲，一道道巨大的血浪翻滚而来，将四周禁锢，无数血神子涌出血海，杀向镇元子。
<br/>　　“杀！”
<br/>　　冥河手持杀剑，杀气纵横，冷光冰寒，出手之间，毫不留情，他修的便是血杀之道，杀人不留情是常有的事。
<br/>　　剑光与地书相撞，发出“咣当”的响声，血海倾覆，将镇元子包围，冥河早已是准圣圆满，而镇元子还停在准圣后期，再加上之前与帝俊大战过一场的原因，大战时间久了，镇元子渐渐落入下风。
<br/>　　“土生木，青木通天！”
<br/>　　镇元子一声大喝，通体气势大涨，他头顶的人参果树快速长大，变成一具通天巨木，青光流转，洞穿虚空，攻向冥河。
<br/>　　“来得好！接我一招，血剑道：血海漫天！”
<br/>　　冥河不惊反喜，手中阿鼻剑挥起，气势大涨，血光乍现，血海翻腾，如九天之上落下一道血色瀑布，铺天盖地般的落了下来，血色瀑布之中，无数枯骨冤鬼挥动着手中兵器，杀向镇元子。
<br/>　　镇元子的通天巨木被血河冲垮，无数血光变成一道牢笼，将他包围住，若不是他有着地书，他早就被擒住了。
<br/>　　“血剑道：血光乍现！”
<br/>　　冥河不停，继续挥剑，剑光凝丝，剑意化线，杀气聚点，点动成线，冥河将剑光、剑意、杀气凝聚成一黑色圆点，圆点带着无尽血光，洞穿虚空，在虚空中出现一条细细的血线，直直地没入镇元子体内。
<br/>　　“噗嗤——”
<br/>　　镇元子一口鲜血喷出，脸色变得苍白无比。他狠狠的盯着冥河，恨声道：
<br/>　　“冥河，此事不会就此罢休！”
<br/>　　接着，他化作一道青色流光，穿过冥河的血海大阵，向着血海之外飞走，却是直接走了。
<br/>　　冥河收起杀剑，默默地看着镇元子离去，他并没有前去阻挡，不是他不想，而是阻挡不了，镇元子修为不差他多少，若是完全拼命，他还拦不下来。
<br/>　　他知道，自己今后会有许多麻烦了。冥河不语，看着手中的杀剑，眼中杀气一凝，良久，转正回到血宫之中。
