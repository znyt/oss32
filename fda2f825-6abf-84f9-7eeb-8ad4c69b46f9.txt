　　“找到你们了。”萧逸眼眸一亮，随后又一冷。
<br/>　　原本在他这些天接触邪修的判断中，邪修，是一门职业，而不是称呼。
<br/>　　二者的区别，在于称呼仅仅是对他们的行事方式的形容。
<br/>　　而职业，则是他们有着自己的特殊方式。
<br/>　　如体修武者，走的是体修一道。
<br/>　　魂师，走的是魂力一道。
<br/>　　炼药师，不仅走的是炼药一道，还有控火一道。
<br/>　　邪修，同样也有着他们自己独特的修炼一道。
<br/>　　也就是说，不同的武者，因他们修炼的一道不同，而有着极大的区别。
<br/>　　太阴太阳之眼，既然能堪破天地武道至理，自己也尝试过从浩瀚武道中，窥探出一丝自己需要的武道力量。
<br/>　　如当初在山洞之内窥探堪破‘十界灭生火’的武道力量，从而找到它与自己体内各种武道的契合点。
<br/>　　那么，必然也能从天地武道中，找到这些邪修所修一道的特点。
<br/>　　他先看一眼邪道阵盘，便是想记下阵盘之内的邪道之力。
<br/>　　之后看向天空，又看向地域周遭，便是试图通过自身感知，看能否通过武道力量的方式，感知出这个地域内修习这一种邪道的武者。
<br/>　　果然，被他寻到了。
<br/>　　会修习这一种邪道的武者，只可能是邪修。
<br/>　　萧逸面具之下的脸庞，陡然闪过一丝笑容。
<br/>　　刚要看真切些，眼眸却忽然一疼，让他不得不收回了太阴太阳之眼。
<br/>　　……
<br/>　　萧逸闭眼休息了一阵，再次睁开眼睛，凝聚出太阴太阳之眼。
<br/>　　这一次，萧逸细细感受。
<br/>　　太阴太阳之眼，在睁开的一瞬，仿佛与这片天地产生了沟通联系。
<br/>　　难怪上一次在山洞中时，能够窥探天地至理。
<br/>　　不过，上一次在山洞时，自己并没有放出感知。
<br/>　　这一次，萧逸放出了感知。
<br/>　　以他现今实力下，感知覆盖数千里，轻而易举。
<br/>　　可是，在太阴太阳之眼沟通天地之时，此刻放出感知，他脑海中的感知范围，竟在不断飙升。
<br/>　　“太阴太阳之眼，竟还有这般功效？”萧逸脸色一惊，随后又一喜。
<br/>　　在眼睛沟通天地之时，他仿佛自身也置身这片天地之上。
<br/>　　放出的感知，竟也在天地之力下，变得范围辽阔惊人。
<br/>　　不过片刻，整个感知范围，竟然覆盖至整个地域的恐怖范围。
<br/>　　整个地域啊，这是多辽阔的范围？
<br/>　　不过，这种状态下的感知，并无法感知到所有东西。
<br/>　　感知到的，只是他现今眼睛沟通天地时，所窥探之天地武道有关的东西。
<br/>　　他现今窥探的，是邪修一道的天地武道。
<br/>　　感知的偌大个地域范围内，感知到的，则是一团团黑气。
<br/>　　这些黑气，便是邪修所在的位置。
<br/>　　脑海中明显可以看到，这些黑气在移动，证明着，这些邪修在移动。
<br/>　　“既是邪修一道能感知，那别的呢？”萧逸带着惊喜，试了试。
<br/>　　眼睛，再次看向天空，但却并非窥探邪修一道，而是窥探控火一道。
<br/>　　果然。
<br/>　　感知中看到的，不再是一团团黑气，而是一团团火光。
<br/>　　这一团团火光，便是这片地域的控火武者。
<br/>　　“嘶，厉害。”萧逸倒吸一口凉气，再度试了试。
<br/>　　收回目光，重新再看，这一次，他窥探的是剑道。
<br/>　　果然。
<br/>　　感知中看到的，是分布在这片地域范围内的一道道剑气。
<br/>　　剑气，或凌厉、或绵长、或狂猛。
<br/>　　想必，这些便是这片地域内的一个个剑修。
<br/>　　不同的剑修，在天地窥探之下，如同一道道不同的剑气。
<br/>　　“嘶。”不多时，萧逸忽然咬了咬牙。
<br/>　　眼睛，充斥着剧烈的疼痛，让他急急收回了目光。
<br/>　　萧逸蓦地感到脸庞上有一股湿润之感。
<br/>　　他知道，那是他眼中滴下的血。
<br/>　　太阴太阳之眼，虽然效果逆天，但其反噬，绝非等闲。
<br/>　　窥探天地至理，等同逆天而行，他的眼睛，自是吃不消。
<br/>　　不过，这一次，这些邪修在他眼中，算是无所遁形了。
<br/>　　这比他得到邪修分部的情报还要来得直接。
<br/>　　萧逸笑笑，内视了一眼小世界。
<br/>　　那里，两个武魂异常显眼。
<br/>　　控火兽，仍旧呆萌；冰鸾剑，却即便是单纯伫立着，仍旧散发着惊天剑意。
<br/>　　这一把神秘的剑，萧逸至今无法完全掌控。
<br/>　　但，足矣。
<br/>　　“冰鸾剑，虽然许久不曾凝聚你了，但这一次，你又帮了我的大忙。”萧逸笑笑。
<br/>　　太阴太阳之眼厉害，那也是冰鸾剑赋予的能力。
<br/>　　那是冰鸾剑认可他，他作为主人的一份凭证。
<br/>　　萧逸笑笑，但数秒后，笑容又蓦地凝固。
<br/>　　他忽然发现了一件让他皱眉的事。
<br/>　　不太确定下，他忍了忍疼痛，再次凝聚出太阴太阳之眼。
<br/>　　感知，再次放出，他窥探的是控火一道的武道。
<br/>　　这一次，他确定了。
<br/>　　“这…”萧逸瞪大了眼睛。
<br/>　　在他的感知中，他看到了这片地域的所有控火武者。
<br/>　　但，在他的感知中，自己却是一片空白。
<br/>　　对，空白。
<br/>　　在天地之力形式的窥探下，他在这片天地中，竟是一片空白？
<br/>　　他当然知道，不是他自己有什么特殊。
<br/>　　而是他脸上戴着的这块幽魂面具。
<br/>　　幽魂面具，屏蔽了所有感知，连天地之力的窥探也一并无视。
<br/>　　难怪这些年来，自己以易霄的身份行走时，无论是什么强者，哪怕是传奇强者，都无法感知到他的修为、气息等等。
<br/>　　如果没有猜错的话，若他摘下面具，在他的感知中，自己会是一团火光和剑气。
<br/>　　因为他修的是剑道和控火一道。
<br/>　　想起来，当年狂血玄君是修罗战体前七层与幽魂面具一并得到的。
<br/>　　虽说不知道为何这两样东西会放在一起。
<br/>　　也不知道狂血玄君具体是从那片上古遗迹中得到的。
<br/>　　但毫无疑问，无论是修罗战体还是幽魂面具，都是逆天之物。
<br/>　　“算了。”萧逸摇了摇头。
<br/>　　这些事，他也懒得深究。
<br/>　　眼睛，再次凝聚。
<br/>　　感知中，再次出现了一团团黑气，记下了这些黑气的位置，萧逸收回了目光。
<br/>　　太阴太阳之眼，只要不是窥探天地太久，还不至于会受到反噬。
<br/>　　“这一次，我看你们还如何逃过我的追杀。”萧逸面具之下，嘴角咧过一道狞笑。
