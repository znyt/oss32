　　这一刻，元始的《神元九转》终于突破到第七转星空，识海化星空。
<br/>　　这一刻，元始感受到了他的元神完全可以堪比混元强者的元神了，也就是说，他的元神修为达到了混元境界。
<br/>　　之前《玄黄不灭》的突破，使得元始的肉身突破的混元，如今他的元神又再次突破混元。这一刻，只有他的元始之道的道基没有达到混元，否则，元始的整体道行修为亦可以完全突破混元了。
<br/>　　之所以没有突破，元始知道，这是因为他的大道是元始大道，大道太强了，注定要花费更多的时间去证道。
<br/>　　如今，若是他愿意，他可以随便以一种或者击中大道作为证道之基，直接就可以证得混元圣人，因为他有好几种大道都达到了混元的强度，如，五行，因果，四象，阴阳、吞噬等。
<br/>　　但，他若是以这几种大道证道，也就意味着他放弃了元始大道，这是元始所不允许的。
<br/>　　他的目标一直就是元始大道，为了元始大道，他甚至放弃了之前已经的达到混元十二重的因果之道，甚至可以眼睁睁看着扬眉与鸿钧一个个在他之前证道，可见他的道心何等坚固。
<br/>　　从始至终，元始一直追寻的就是元始大道。
<br/>　　如今，距离证道，有了元始印的帮助，他的积累也已经足够了，但，想要突破这最后一步，他总觉得差了一点什么，这让他不敢强行突破。
<br/>　　他一直等待着那个能够让他突破的契机，此前，最好的准备就是更多的积累。
<br/>　　“呼——”
<br/>　　元始深呼一口气，睁开双眼，眼中可见一片浩瀚无边的星空闪烁，他的眼神变得更加深邃可怕了。
<br/>　　心似海，神若星！
<br/>　　他变得更加恐怖了。
<br/>　　“还是差了一些，到底是差了哪里呢？”
<br/>　　元始细细思索道。他总感觉自己的积累已经足够了，但就是不能突破，差了一点什么。
<br/>　　“嗯？外边怎么这么吵？”
<br/>　　元始微微皱眉，神念一扫，顿时看见昆仑山外汇聚了无数修士，妖魔鬼怪，魑魅魍魉，什么都有，搞得昆仑山混乱无比，而且他们好像是在争夺什么。
<br/>　　他仔细一听，顿时就知道了原因，他们正是被元始突破时的浩大异象所吸引而来的，顿时让元始哭笑不得。
<br/>　　但此刻这些人汇聚在三清圣地之外，也不是一个事儿，吵吵闹闹，成何体统，还让不让人好好修炼了。
<br/>　　故而，元始决定将他们赶走。
<br/>　　“滚！一刻之内，再不滚出昆仑者，杀无赦！”
<br/>　　顿时，阵法之外的虚空之中，响起一声大喝，这一声大喝之中，元始掺杂了元神之力。此刻，许多修为不足之人，纷纷被震得倒地不起，这也是元始的一个震慑。
<br/>　　“谁？谁在那里说话？”
<br/>　　“前辈，饶命啊！”
<br/>　　“是哪位前辈在此清修，我等无意冒犯。”
<br/>　　“谁？给本大爷出来！缩头缩尾的，是不敢露面吗？”
<br/>　　见此情景，在场许多人顿时惊慌了起来，纷纷猜测是不是有前辈高人早已占据了此地。也有人大声痛骂，想引人出来，群起而攻之。
<br/>　　就算他修为再强，此地有这么多人，他都能杀了吗？反正他们是不信的。
<br/>　　“前辈，我们这就走。”亦有修为较低之辈，对着山谷行了一礼，就此离去。
<br/>　　但，也有一些人不愿离去，他们不甘心就此离去，都紧盯着山谷，动了各种心思，想要进山谷夺宝。
<br/>　　元始没有多管，想走的就走，想留的就留，他说一刻钟就是一刻钟的时间，在这一刻钟时间里，不管他们做什么，他都不会管，至于一刻钟之后，他要做什么，他们亦管不了。
<br/>　　“大兄出关了？”
<br/>　　太清宫中，老子睁开双眼，眼里一眼山谷之外，皱了皱眉，又看了一眼玉虚宫，轻声道。
<br/>　　“大兄已经出关，不知道他有多强，是不是去找他切磋切磋？”
<br/>　　上清宫中，通天笑道。至于外面的修士，他有心想让他们就此离去，莫要惹恼大兄，但是大兄既然已经开口，他此刻也不会多管，在他眼里，还是找大兄比试更加重要，虽然每次比试，他都会被元始打得鼻青脸肿。
<br/>　　“大兄，你成圣了吗？”
<br/>　　通天来到玉虚宫中，面对元始问道。显然，经过元始的诉说，他也是知道混元圣人这个境界的。
<br/>　　这一刻，他感觉不到元始丝毫的修为，仿佛在他眼前的不是他的大哥元始，而是一个普普通通的凡人一般，但他知道，这是不可能的。那么就只要一种情况，他大哥已经突破了。
<br/>　　“还没有，还差一点。”元始对着通天笑道，取出悟道茶，给通天满了一杯，也给自己倒了一杯。
<br/>　　“那怎么——”
<br/>　　“你是想说，你看不透我吗？哈哈！”元始轻笑一声，道：“那是因为我的元神修为小有突破，已经堪比混元元神了。”
<br/>　　“哦！原来如此。”
<br/>　　端起用混沌悟道古茶树树枝做成的茶杯，只见茶杯之上，云雾缭绕，形成九层重云，云雾之中，有神钟鸣响，凤凰飞舞，麒麟奔跑，各种异象，尽在其中。
<br/>　　“嗯。”
<br/>　　轻轻闻了一下，顿时，清香扑鼻，一股清爽缭绕心间，心中升起一股不可言喻的愉悦之感。通天夸赞道：
<br/>　　“还是大哥的悟道茶好啊，简直是人间难得几回味啊！”
<br/>　　元始轻笑不语，亦抬起一倍悟道茶，轻呡一口，顿时，一股清凉顺着他的喉咙流遍全身，使得全身无比舒坦。最后，一股清流流进了他的脑海之中，顿时，各种灵光纷至沓来，让人情不自禁闭上双眼，飘飘欲仙，神清气爽，陷入顿悟之中。
<br/>　　片刻，元始睁开双眼，看了一眼通天，只见他亦闭上双眼，好似在参悟什么。
<br/>　　此刻，一刻钟时间已到，元始向山谷外看了一眼，发现还有人在山谷之外徘徊。
<br/>　　“还真有人不怕死啊！”元始心道。
<br/>　　“既然尔等求死，那本座就成全尔等这个心愿。”
<br/>　　“嗡——”
<br/>　　元始心念一动，顿时，一股恐怖的幻境降临人间，包围了山谷之外的修士。
<br/>　　这是元始的梦幻一道的一个神通，在幻境之中，每一个生灵都会进入到他心灵深处最渴望的世界，甚至这个世界堪称‘真实不虚’！元始构造的这个幻境，在某种程度上就是真实的！
<br/>　　虚幻一道的巅峰，便是将虚幻的化为真实。
<br/>　　山谷之外的修士，凡是被幻境笼罩的，在这恐怖的幻境之下，却是拖拉这他们的灵魂，让他们直接沉沦。
<br/>　　一瞬间，所有生灵都沉沦其中，不可自拔。
<br/>　　眨眼间，他们便都死在了这幻境之中。
<br/>　　心念一动，这些人身上的宝贝灵药等便全部飞到元始手中，而这些人的尸首，却是全部化为飞灰，飘散到这山谷之间，滋养着山谷中的花草树木。
<br/>　　谈笑之中，幻杀万人！
<br/>　　一念之间，定人生死！
